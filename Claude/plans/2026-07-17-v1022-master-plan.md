## 1. 목표

v1.0.21(확장 완료판)을 재료로 **활물질 기준 3챕터 체제**로 재편하고, 각 장을 자기완결 리뷰(장별 기호표·장별 참고문헌)로 세운다:

- **Ch1 흑연 + 열특성**: 공통 이론 기초(Part 0 통계역학 사다리 — 전 장의 뿌리) + 흑연 결과 사슬(N0–N9) + **열특성 파트(현 Ch2 가역 발열·엔트로피 전량 흡수)** + 흑연 부록.
- **Ch2 LCO + 열특성**: 현 Part II(§11–17) 승격 + LCO 열특성. **서술 방식 = Ch1 식을 기반으로 "추가되는 텀만" 설명**(현 Part II 가 이미 이 방식 — "파라미터 교체 + 전자항 추가" — 이므로 자연 이관; 열특성도 전자항·게이트 시연 등 차분만).
- **Ch3 Si·혼합음극(신규 대형)**: Si 케이스별(원소 Si / SiO$_x$ / Si–C 복합 — D22-3) + **흑연 다수·Si 소수 블렌드**(f_Si = 0~30% 가변) 이론·검토 준비 + 현 Si 예비 지도(노드 매핑·GS-1) 승격.
- **코드 v1.0.22 동기**: 블렌드 합성 구현 — **f_Si 옵션 토글(끄면 흑연 단독과 bit-exact)**, Si 케이스 파라미터 셋, 신규 게이트.
- **관통 축 2**: (a) **인용 다리 수식화** — load-bearing 인용마다 "우리 식 → 그 논문 식"의 변수 매핑·중간 수식 1~3개(따라가기 쉽게), (b) **통계역학 증축** — 잔여 후보(iii CLT broadening·iv CNT 연결) + 신규(블렌드 공통-μ 대정준·Larché–Cahn 응력-화학퍼텐셜).
- **최종 형태 대비**: 챕터별 PDF → **단일 문건 병합 드라이버**(장별 참고문헌 유지 — 리뷰 모음형) 시험 빌드까지.

## 7. Phase 세부 (요지 — 각 phase 착수 시 세부 계획서)

- **R2 인용 다리 양식(신규 관통 축)**: load-bearing 인용마다 3요소 — ①우리 식(라벨)과 논문 식의 **변수 대응표 또는 등치 중간 수식 1~3개** ②논문이 그 식을 얻는 경로 한 문장(방법 요지) ③우리 전제와의 차이 1문장(가정 차 — CLAUDE.md 검수 5항 양식). 대상 선별은 D22-5(권고 핵심 ~12곳: dreyer2010/2011 히스·mckinnon1983 등온선·bazant2013 χ 분해·marianetti2004 Mott·vanderven1998 상도표·msmr_origin2017/bakerverbrugge2018·weppner_huggins1977 GITT·sethuraman 응력·larchecahn1973·verbrugge_lisi2016·baek_pilon2022).
- **R5 블렌드 이론 핵심**: 혼합 음극 = 같은 전위 손잡이($\mu$ 하나)를 공유하는 두 host 의 자리 클래스 합집합 — 대정준 반전식이 $\sum_\mathrm{host}\sum_j Q_j\xi_{\eq,j}(U_\mathrm{oc},T)=Q\bar x$, $Q=Q_\mathrm{gr}+Q_\mathrm{Si}$, $f_\mathrm{Si}=Q_\mathrm{Si}/Q$ 로 **한 줄 일반화**(sec:sm-mc 의 클래스곱이 host 곱으로 — 요동 유일근 증명 그대로 이월). 이것이 Ch3 의 통계역학 앵커이자 코드 합성 규칙.
- **R6 코드 설계**: `BlendedAnodeDQDV(f_Si, si_case)` — f_Si=0 에서 기존 흑연 경로와 bit-exact(하위호환 게이트), f_Si∈(0,0.3] 에서 용량 배분·공통 V_n 합성. Si 케이스 셋은 tier-C 시연값(문헌 기반 초기값 — R4 산출).
- **R9 병합 준비(빌드 금지)**: MERGE_READINESS.md = 전역 라벨 유일성 검사 결과(도구 출력)·매크로 통합 상태(R1 선행)·장별 bib 병합 방식(장말 유지)·병합 절차 단계별 안내·알려진 주의점. **병합 빌드는 하지 않는다** — 사용자가 별도 세션에서 버전 안정화 판단 후 직접 수행(D22-8).
