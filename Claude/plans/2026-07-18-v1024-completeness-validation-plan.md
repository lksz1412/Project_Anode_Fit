## Summary

Anode_Fit 문건·코드를 **반쪽셀(음극·양극 각각) dQ/dV 곡선 생성기**로 보고, 그 **완성도를 실측 검증으로 못박는다.** §7 정독 결과 **브로드닝 물리는 이미 성숙**(3출처·폭 예산·스코프 한계 명시·역문제 배제)하므로, 완성의 본질은 "물리 추가"가 아니라 **"우리 식의 T/I/V 조건 의존이 실제 데이터를 설명하는가"의 검증 + 그 결과 드러나는 표적 보정**이다.

**핵심 검증 설계**: 사용자 회사 표준 조건 = **GITT·0.05C·0.1C·0.2C × 15·23·35·45°C**. 이 매트릭스가 브로드닝 3출처를 자연 분리한다 — GITT=②③(①소멸), 율 시리즈 차분=①(∝|I|), 4온도=T-의존. 회사 데이터는 반출 불가이므로 **같은 조건대의 공개 데이터를 proxy**로 검증한다.

**M-통찰의 결착**: 문건은 `w_j`를 "②⊗③를 흡수하는 현상학적 자유 피팅 폭"으로 이미 규정 → **피크 높이(∝1/w)는 물리로 조절되게 설계됨**. 기존 매칭의 M(전압 신축)이 하던 일을 `w_j`가 담당한다. 검증 목표 = **w_j + L_V 만으로 실측 피크 높이를 전 조건에서 재현(= M 불요 증명)**.

## Current Ground Truth

**§7 브로드닝 — 성숙 확인 (정독 근거).**
- **3출처 분리**: ① 유한율속 비대칭 꼬리(`L_V`, ∝|I|, I→0 소멸, §8) / ② 내재 `RT/F` 폭(평형 잔여·전류 무관) / ③ 앙상블 `η` 산포(`ρ(U_app)=U_j+η`, CLT→종형, iii-a 평형층 + iii-b 브로드닝층).
- **폭 예산**(eq:widthbudget): `σ_sym² = (π²/3)(n_jRT/F)² [②] + σ_η² [③]`, 꼬리 `ℓ = L_V ∝ |I| [①]`. 대칭 폭(w_j)과 비대칭 꼬리(L_V)를 **전류 의존성으로 분리 식별**.
- **w_j 지위**: "②⊗③ 흡수 현상학적 자유 피팅 폭". `n_j=1` 출발 → 실효 폭 `RT/F≈25.7mV`.
- **스코프 한계 명시(non-goal)**: (i) 입자 크기(반경)=Gibbs-Thomson 수치로 무시 가능 증명 후 배제 / (ii) `dQ/dV→ρ` 역산=ill-posed → **forward-only** / (iii) PSD convolution 기계장치 미도입. → **이것들은 gap 아님, 의도된 경계.**

**M-factor 결착(사용자 통찰·문건 정합).** 코인셀 피팅서 피크 위치는 맞아도 높이가 안 맞아 M(전압 신축)을 넣었던 것 = dQ/dV 브로드닝(피크↓)이 dV/dQ=1/(dQ/dV)에서 값↑로 보인 것. 문건 `w_j`(브로드닝)가 이 높이를 물리로 담당 → 검증이 성공하면 M 제거.

**회사 표준 측정 조건(사용자 제공).**
- 율: **GITT · 0.05C · 0.1C · 0.2C** (전부 저율 — GITT는 준평형).
- 온도: **15 · 23 · 35 · 45 °C** (4점).
- → 검증 매트릭스 = 4온도 × (GITT + 3율) = 최대 16 조건점.

**조건 ↔ 브로드닝 출처 대응(검증 설계의 핵심).**
| 조건 | 활성 출처 | 검증 대상 |
|---|---|---|
| GITT (I→0) | ②+③ (①소멸) | 평형·앙상블 폭 + T-의존 |
| 0.05/0.1/0.2C | +① (∝\|I\|) | (율−GITT) 차분 = L_V 스케일·∝\|I\| 법칙 |
| 15/23/35/45°C | 세 출처 T-의존 | ② nRT/F · ③ σ_η(T) · ① Arrhenius L_V(T) |

**미확인(추정 금지).**
- 이 저율(≤0.2C)에서 ① 꼬리가 실제로 얼마나 큰가 — 코드 예측은 휴면(1C서 L_V~7.5e-7 → 0.2C서 ~1.5e-7). **실측 율 시리즈로 확정**(V3).
- 공개 데이터 중 GITT/저율 + 다온도 + 반쪽셀 보유 셋 존재 여부 — 조사(V1).
- 코드의 폭 T-의존(σ_η(T)) 구현 여부 — 확인(V0).
- 대상 양극 화학(NMC?) 커버리지 — 사용자 확인(D 참조).
