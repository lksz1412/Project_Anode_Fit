## 4. 핵심 기술 설계 (무엇을 어떤 식으로)

### 4-1. H1+H2+H3 동시 봉합 — A_0 정의·r 분리·Σ_j 복구
§8 꼬리식을 다음으로 재조립(순 보강, 기존 식 21은 단일전이 특수형으로 남김):
```
dΘ_tail/dq = Σ_{j=1}^{N_p} (Q_{j,tot}/Q_p) · r_j(q_a) · K_j(q)
K_j(q) = ∫_{Lmin,j}^∞ A^{prob}_{L,j}(L) (1/L) exp[-(q-q_{a,j})/L] dL
A^{prob}_{L,j}(L) = ρ_{G,j}(G(L)) (RT/L) H(L-Lmin,j),  ∫A^{prob}_{L,j}dL=1
```
- **A_0 정의(H1)**: 진폭 가중을 **전이별 용량분율 `Q_{j,tot}/Q_p`(전하보존서 도출, 무차원) × 접근성 `a_j∈[0,1]`(독립 입력)** 로 *명시 식*. 즉 `A_{0,j}=(Q_{j,tot}/Q_p)·a_j`. → 더 이상 말뿐 아님.
- **r 분리(H2)**: 동역학 잔차 `r_j(q_a)`를 A_0에서 **빼내 별도 인자**로. A_0는 정적 물리량만 → "역산 금지 독립량" 일관성 회복.
- **Σ_j 복구(H3)**: 바깥 이산합(전이 N_p) + 안쪽 연속적분(전이 내부 ρ_{G,j}) **2층 명시**.

### 4-2. H4 — Θ 일가 정의를 §8 진입부로 이동·신설
§8 첫 문단에 정의 박스: `Θ≡Σ_j(Q_{j,tot}/Q_p)ξ_j`, `Θ_eq≡Σ_j(Q_{j,tot}/Q_p)ξ_{eq,j}`, `Θ_init≡Θ(q_a)`, `Θ_tail`=post-peak 성분. (§9의 기존 정의는 참조로 정리, 중복 제거.)

### 4-3. H6 — Ch1 근사 닫힌식 (자급, KWW/stretched)
`L=L_0 exp[(G−χ_jA_j)/RT]` 라 `ρ_{G,j}=N(G_{0,j},σ_{G,j}²)` 면 `ln L` 정규 → `L` log-normal. 단일지수의 log-normal 완화시간 중첩 = **stretched-exp(KWW) 근사**(문헌 등가성 johnston2006/svare2000/lindsey1980 — 이미 참고문헌에 있음). Ch1 산출 근사 닫힌식:
```
dΘ_tail/dq ≈ Σ_j (Q_{j,tot}/Q_p) r_j(q_a) (β_j/τ_j)(Δq_j/τ_j)^{β_j-1} exp[-(Δq_j/τ_j)^{β_j}]
Δq_j=q-q_{a,j},  τ_j=L_0 e^{(G_{0,j}-χ_jA_j)/RT}(중앙 길이),  β_j=β(σ_{G,j}/RT)∈(0,1]
```
- σ_G=0 → β=1 → 단일지수(식 16) 환원 점검 포함.
- 피팅 파라미터 전부 물리량: `{G_{0,j},σ_{G,j},χ_j,k_0,a_j,U_j,w_j,Q_{j,tot},Q_bg}` → 여러 T·I 동시 fit.
- **GS-3 정직표기**: `β(σ_G/RT)` 정확한 closed map은 분포가정 의존(BOUNDED). 정규-배리어→KWW는 *근사*임을 박스로 명시. erf/log-normal 외 분포면 β 관계 달라짐.
- 이로써 식 25(dQ/dV_n)에 `dΘ/dQ=(1/Q_cell)dΘ_tail/dq` 를 **닫힌형으로 대입 가능** → "Ch1만으로 fit" 충족.

### 4-4. H5 / H7 / H8
- **H5**: §7 첫 사용에서 "`g`는 배리어 `G`의 한 값(적분 더미)" 한 줄.
- **H7**: 꼬리=post-peak 영역에서 `|V_drive−U_j|` 증가 시 선형 −χA의 유효범위를 **정직하게 한정**: (a) near-peak shoulder(소구동)와 (b) far-tail(2차 Marcus항 비무시) 구분, far-tail은 BOUNDED 표기. 무근거 "꼬리=V_drive≈U_j" 단정 삭제.
- **H8**: 식21=명시(forward, frozen-target) 꼬리, 식22=목표 변동 시 self-consistent. **§4-3 근사 닫힌식 = 식21을 frozen-target 가정서 닫은 피팅용**임을 한 문장으로 못박음.
