## 1. 목표

v1.0.20(완성 원본·동결)을 베이스로 다섯 축을 확장한다:
- **A. 통계역학 관통(GO 확정)**: ①전하 보존식의 대정준 유도(다클래스 Ξ=Π_j Ξ_{1,j}^{M_j} → ⟨N⟩=Σ M_jθ_j 반전 = eq:implicit 식별 + ∂⟨N⟩/∂μ=β·var(N)>0 유일근 증명 — 중심식의 통계역학 1급 승격) ②Eyring 꼬리의 TST 분배함수 유도(k_BT/h 의 미시 기원 + ΔS_a=R ln(q‡/q_R) — Part 0 의 q(T) 언어로 동역학 파트 접속).
- **B. 독자 인프라**: 그림 반영(v1.0.20 경쟁 산출 24본 체리픽 — 신규 vs 기존 비교 포함) + [GO 대기] 전역 항법 3종(식 의존성 지도·통합 기호 대응표·독자 경로 안내).
- **C. 일반 확장 선별분**: DIRECTION_GENERAL_REPORT 17건 중 사용자 선택(권장 top3: Ch1 end-to-end worked example·Ch1↔Ch2 로드맵 대응표·측정 원리 배경 절[GITT — weppner_huggins1977 검증 완료]).
- **D. LCO 심화**: 게이트 포함 worked example(코드 지원 확인됨) 등 — 기획 조사(Q1)로 항목 확정.
- **E. Si 계열 이론 접목(신규 요청)**: 합금화 음극(비정질화·~300% 부피 팽창·큰 히스테리시스·경사 전위 — staging 부재)의 물리를 기존 골격(격자기체·logistic·전하 보존)에 어디까지 얹을 수 있는지 → 기획 조사(Q1) 후 아키텍처 확정(신규 장 규모 예상).

## 9. 리스크
- Si 축은 물리가 이질적(비평형 비정질·기계 결합) — 기존 골격 과적용 위험 → Q1 에서 "닿는 곳/닿지 않는 곳" 경계를 명시하고 닿지 않는 곳은 정직 공백(Gn 방식)으로.
- 대형 신규 장 + 경쟁 저작 = 토큰·사고 리스크 → 조기 저장·생존 규칙·phase 별 커밋 유지.
- 문서-코드 동기 확대(Si 파라미터화) — doc-leads 유지하되 회귀 기준을 phase 내 명세(appB 방식).
