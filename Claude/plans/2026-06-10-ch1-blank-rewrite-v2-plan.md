## Summary
박사님 검수 피드백(6건, /goal) 기준 **백지 재작성**. 직전 판(`graphite_ica_ch1_Fable.tex`, 50p)은 보존하고 새 파일에 처음부터 다시 쓴다. 재구성의 축:
1. (피드백 4 ★핵심) **Eyring/Arrhenius 속도식 $k\simeq k_0\,e^{-\Delta G_a/RT}$ 를 근본식(식 1.1)으로** 문건 전체의 척추로 세운다. 온도(지수의 $T$)·전위($\Delta G_a$ 를 기울임)·C-rate(요구 속도와 $k$ 의 경쟁) 세 영향이 전부 이 식에서 가지를 치고, 평형 열역학은 정·역 속도의 균형점(정지점)으로 \emph{회수}된다 — 열역학 먼저가 아니라 속도식 먼저.
2. (피드백 1) **열역학 다리 절 신설** — 공대 물리 독자 기준: $G$ 의 의미 → $\mu=\partial G/\partial n$(입자 하나 더할 때의 $G$ 변화) → 혼합 엔트로피의 $\mu$ → 전기화학 퍼텐셜($-FV$)까지 전제 단계 전개. 직전 판의 식 (1.9)( $\mu_{Li}$ ) 뜬금 등장 해소.
3. (피드백 2) **FWHM 닫힌꼴 유도 삭제** — 꼬리가 주 토픽인 시스템에 단봉 폭 지표 유도는 과잉. 폭은 $RT/F$ 스케일 서술로 축소.
4. (피드백 3) **binodal/spinodal 재설명** — 용어 전에 물리 그림(섞인 균질 상태가 자유에너지상 불리해 두 조성으로 갈라짐 → "갈라질 수 있는 경계"와 "버틸 수 없는 경계"의 두 선)을 전제 개념(과냉각 준안정)부터 단계 전개, "왜 여기서 필요한가"(plateau·충방전 갈림의 뿌리)를 도입 시점에 명시.
5. (피드백 5) 그림 재설계 — 근본식 도식(활성화 언덕+세 가지 화살표)·정역 균형→등온선·갈라짐 도식을 신규로, 직전 판 그림 중 유효한 것(좌표 검증 완료)은 새 흐름에 맞춰 재배치·캡션 재작성.
6. (피드백 6) **연습문제 삭제**, 분량 강박 제거 — 회사 연구원 설득용 문건. 자연 분량(예상 38~45p).
작업 방식 동일: 절 단위 루프[위치→계획→계획 검수→작업→작업 검수→커밋→푸쉬], 보완 10회+, 병렬 최소(검수 Agent 1개씩), 원본·직전 판 불가침, 메모리 규범(괄호 보충 전보체 금지·보편식 먼저·다리 의무·조사-급 라운드 금지) 전면 적용.

## 새 구성 (V.1 의 절 지도 — Eyring 척추)
1. **서론** — 관측(꼬리·겹침·히스) + 접근 선언: "이 문건의 모든 식은 하나의 속도식에서 가지를 친다". 방전 메인·히스 확장 위상. 독자 두 경로(이론/실무).
2. **§1 기호와 규약** — 간결한 사전 표.
3. **§2 근본식 — 활성화 장벽과 반응 속도** ★식 1.1 = $k\simeq k_0e^{-\Delta G_a/RT}$(Eyring). 전이상태 언덕 그림(신규 도식: 언덕 + 온도/전위/율 세 화살표), $k_0=k_BT/h$, $\Delta G_a=\Delta H_a-T\Delta S_a$. 세 인자의 진입점 예고(로드맵).
4. **§3 열역학 다리 — $G$ 에서 $\mu$ 까지** (피드백 1): 등온등압 자발성 지표 $G$ → $\mu\equiv\partial G/\partial n$ 의 물리적 의미 → 격자기체 혼합 엔트로피(Stirling)로 $\mu(\theta)$ → 상호작용 $\Omega$ 항 → 전기화학 퍼텐셜(전자 일 $-FV$). "$\mu$ 가 평형을 지배하는 이유"까지.
5. **§4 정·역 속도와 평형의 회수** — $r^\pm$(같은 전이상태, 장벽의 비대칭 기울임 — BV/BEP, 합 1 강제), detailed balance, **평형 = $r^+=r^-$ 의 정지점** → logistic 등온선이 속도식에서 도출. (직전 판과 인과 방향 역전 — 사용자 의도.)
6. **§5 상호작용과 상분리** — $\Omega$ 가 등온선을 변형(좁아짐→비단조). **갈라짐의 두 경계**(쉬운 전개: 균질이 불리해지는 영역/즉시 무너지는 영역 → 그때 binodal·spinodal 명명), 준안정(과냉각 유추 아님 — 핵생성 장벽 단계 전개), plateau 와 충방전 갈림의 뿌리 예고. vdW loop 도식.
7. **§6 관측축 — 전하 보존과 내부 전위 $V_n$** — 보존식(이제 후반 배치: 속도·평형을 안 뒤 관측으로), $V_\app$/$V_\drive$, 관측 $\dd Q/\dd V$.
8. **§7 평형 peak 기준선** — logistic 미분의 위치·폭($RT/F$ 스케일, $w^{\mathrm{eff}}$ 좁아짐)·높이·면적. **FWHM 닫힌꼴 삭제**(피드백 2).
9. **§8 C-rate 가지 — 지연과 꼬리** — 요구 속도 vs $k$: 보편 ODE(원천항 포함)→일반해(기억 커널, 도식)→상승부/꼬리 두 극한.
10. **§9 전위 가지 — 유효 장벽** — $\Delta G_{\mathrm{eff}}=\Delta G_a-\chi\mathcal A$ 가 근본식 지수에 직접. 보편형 $k=r^+(1+e^{-A/RT})$ 먼저.
11. **§10 온도 가지 — Arrhenius 와 분포** — 명시적 온도 가지(기울기 $-\Delta H_a/R$) + 입자 분포($\sigma_G/RT$ 전파·stretched).
12. **§11 합성 — 방전 닫힌식과 3인자 의존** — 평형−지연 구조, 3×3 표, 식별 순서(비순환).
13. **§12 다전이 합산과 겹침** — forward-only, 융합 도식.
14. **§13 [확장] 충방전 분기** — spinodal 과주행→gap 닫힌꼴→γ.
15. **§14 [확장] 분극 분리와 관측 gap** — 절편/기울기.
16. **§15 통합식과 피팅 알고리즘** — 방전 한 줄+양방향, (1)–(8) 알고리즘(직전 판 사양 — round-trip 으로 실증된 그 사양), 참조표·진단표·스켈레톤. 부분 cycle 은 이 절의 짧은 소절로 흡수(별도 절 강등 — 덤의 덤).
17. **§16 검증과 반증** — 경쟁 원인 배제·히스 반증. (연습문제 절 없음 — 피드백 6.)
