  D. Carlier, I. Saadoune, M. Menetrier, C. Delmas,
  "Lithium Electrochemical Deintercalation from O2-LiCoO2: Structure and Physical Properties",

COLUMNS:
  col1 = x_Li_in_LixCoO2  : lithium content x in Li_xCoO2 (dimensionless stoichiometry). Range 0.170 - 0.958.
  col2 = OCV_V            : cell voltage vs Li/Li+ [V] read off the (near-equilibrium) discharge curve. Range 3.04 - 4.55 V.
  162 data rows (+1 header row). Data values are byte-identical to the repo CSV; only a header line was added
  and ", " separators normalized to ",".

POLYTYPE:  O2  (CRITICAL)
  The 2002 Carlier/Delmas paper studies the METASTABLE O2-LiCoO2 polytype (ion-exchanged from P2-Na cobaltate),
  NOT the common O3 (R-3m) LiCoO2. Abstract: "electrochemical deintercalation of Li from the metastable O2-LiCoO2 phase";
  phase sequence O2 / T#2 / T#2' / O6 during deintercalation. => This is O2, do NOT treat as O3.

MEASURED / DIGITIZED / ANALYTIC:  DIGITIZED-FROM-FIGURE (of a real experimental electrochemical curve).
  i.e. real measured O2-LCO deintercalation voltage, digitized point-by-point from published Fig 4a.
  It is NOT a raw instrument file and NOT an analytic fit.

USED-IN: diffthermo OCV-fitting paper (RK_fit_JPCL_Jan2024 = JPCL 2024). This CSV is the fit TARGET (input data).
