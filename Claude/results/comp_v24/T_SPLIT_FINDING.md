# 온도 의존 피크 분리 (45℃ 2피크 / 25℃ 병합) — 이론·코드점검 종합

> 사용자 관측: **방전 흑연 3번째 전이가 45℃서 2피크, 25℃서 1피크(병합)**. 열broadening인지 다른 이유인지 미상.
> 요청: (a) 정합성 높일 이론·논문 검토 (b) 단순 코드문제일 수 있으니 코드 점검.
> 결론 선요약: **버그 아님.** XRD 확증 열역학 현상(stage-2L 엔트로피 안정화)이고, 문건 코드가 물리 파라미터를
> 주면 정확히 재현. 현 출하 default 가 안 보여주는 건 **폭·ΔS 시드 두 파라미터**(둘 다 기존 지적 이슈) 때문.

---

## 1. 기제 — 열 broadening 아님 (반대 방향이 증거)
- **열 broadening 가설이면 고온서 더 병합**해야 함(kT↑ → 폭↑ → 겹침↑). 관측은 **정반대**(45℃ 분리·25℃ 병합).
  → 브로드닝 아님. **인접 두 전이가 고온서 벌어지는(diverge)** 현상.
- 정본: **stage-2L(“liquid-like” 면내 무질서·고배위엔트로피) 상**은 배위 엔트로피로 안정 → **~10℃ 이상서만 존재**.
  - 저온: 2L 미형성 → **3↔2 단일 전이(1피크)**.
  - 고온: 2L 출현 → **3↔2L + 2L↔2 두 전이(2피크)**. 온도↑ → 2L 창 넓어짐 → 분리↑.
- 이는 **회사서 "조건 따라 안 잡히던 피크가 잡힘"의 물리 정체**(고용체/order feature 가 아니라 **진짜 두-상 2L 출현**).

## 2. 서지 (XRD 실측 — 전부 검증가능)
| 문헌 | 관측 | 정량 |
|---|---|---|
| **Dahn, *PRB* 44, 9170 (1991)** in-situ XRD | 0–70℃ Li_xC6 상도. **2L 상 10℃ 이상 존재**·저온 소멸 | 3↔2L·2L↔2 분리 **≈0.30 mV/℃**, ~10℃서 0 |
| **Schmitt et al. 2022** (delithiation dV/dQ) | **탈리튬화(방전 음극)서** 2L 분리 재확인 | 고온 분리·상온 병합 |
| Reynier·Yazami·Fultz *JPS* 119–121, 850 (2003) | 2↔1 plateau ∂/∂T≈0 (견고 두-상) | 2L 구역만 T-민감 |

- **관측이 "방전(탈리튬화) 음극 3번째 전이"인 점이 Schmitt 2022 와 정확히 일치** — 2L 분리는 delithiation 특이적으로 보고됨.
- Δ(ΔS)=29 J/mol/K = 두 전이의 **가역 엔트로피 차** → `∂U/∂T=ΔS/F` 로 분리속도 0.30 mV/℃ 산출(29/F=0.30).

## 3. 코드 점검 (문건 v1.0.23 무수정 검사)
| 지점 | 코드 | 판정 |
|---|---|---|
| `func_U_j` (L95) | `U_j(T)=(−dH_rxn+T·dS_rxn)/F` → `dU_j/dT=dS_rxn/F` | ✅ 열역학 정확 |
| `equilibrium` (L624)·`dqdv` (L759) | 둘 다 `func_U_j(T,dH_rxn, _effective_dS_rxn(tr,T))` | ✅ 두 dQ/dV 경로 T-path 동일 |
| `_effective_dS_rxn` 흑연 base (L708) | `return tr['dS_rxn']` (항등) | ✅ 엔트로피→dU/dT 직결, 버그 없음 |
| `GRAPHITE_STAGING_LIT` (L1027) | **`stage 3→2L`·`stage 2L→2` 를 이미 별도 전이로 명명** | ✅ **2L 구조 이미 내장**(상 추가 아님) |

**→ 코드 구조 결함 없음.** 출하 default 가 T-분리를 **안** 보여주는 건 오직 두 파라미터 때문(둘 다 기존 지적):
1. **폭 default 가 shoulder-broad**(`w=0.014–0.016`=14–16mV). 3→2L(0.140)·2L→2(0.120) 은 20mV 이격인데 FWHM≈50mV
   라 **전 온도서 한 blob 으로 뭉갬**. = M-factor 분석서 지적한 **과대폭**(두-상은 ~2mV 여야). 
2. **Δ(ΔS) default 가 과소**(3→2L:0 − 2L→2:−5 = 5 J/mol/K → 0.05 mV/℃). Dahn 실측 **Δ≈29**(→0.30 mV/℃) 필요.

## 4. 재현 (물리 파라미터 주입 → 출하 `equilibrium()` 직접 호출) — `T_split.png`
두-상 폭(`w=1.6mV`) + Δ(ΔS)=29(고V 3↔2L:+14.5·저V 2L↔2:−14.5, Tm=10℃ 앵커):

| T | 분리 | 피크수 | 관측 |
|---|---|---|---|
| 5℃ | 1.5 mV | **1(병합)** | — |
| 10℃ | 0 mV | 1(병합점) | 2L 소멸(Dahn) |
| **25℃** | 4.5 mV | **1(병합)** | ✅ 사용자 관측 |
| **45℃** | 10.5 mV | **2(분리)** | ✅ 사용자 관측 |
| 65℃ | 16.5 mV | 2(분리) | — |

- **분리 기울기 0.271 mV/℃ ≈ Dahn 0.30**(정합)·**병합 ~10℃**·**25℃ 병합/45℃ 분리 정확 재현**.
- 병합/분리 판정 = prominence 기준 분해가능 피크수(sep<FWHM=3.53·w ⇒ 병합). 물리적으로 정직.

## 5. 함의 (문건·코드 반영은 사용자 최종 결정 — 여기선 확정 사실만)
1. **관측은 XRD 확증 열역학 현상**(stage-2L). curve-fitting 아님. **폐기했던 "4→6 전이"와 근본 다름** — 2L 은
   XRD 공존 확인된 **기존 4 두-상 중 2개**(3↔2L·2L↔2)이며 문건이 **이미 별도 명명**. 상 추가 없음.
2. **코드 정상.** 반영이 필요하면 **파라미터 시드 2개**만: (a) 두-상 폭 ~2mV(사용자 제안 Ω>2RT sharp 시드) (b)
   3↔2L·2L↔2 의 실 Δ(ΔS)≈29(현 0·−5 → 예: +7·−22 또는 문헌 entropic profiling 값).
3. **산업 방어력 강화 포인트**: "조건 따라 피크 개수 변함"은 사용자가 **불일치**로 우려했으나, 실은 **모델이
   언제(>10℃)·얼마나(0.30 mV/℃) 분리되는지 예측**하는 **물리적 강점**. 회사 다온도 매트릭스 설명력의 근거.
