# LCO 진단 — "흑연이 되면 LCO도 됐어야" (사용자 가설) 검증

> 사용자 가설: LCO가 흑연과 아주 다르게 진행될 리 없다. 흑연이 제대로 되면 LCO도 됐어야.
> 요청: (1)흑연과 완전 동일 방법으로 LCO, (2)LCO 추가내용 반영, (3)@3·@5도 적용 — 어디가 문제였나.
> 결론 선요약: **가설 맞음. LCO는 방법·모델상 흑연과 동일하게 작동.** 앞선 'LCO 안 맞음'은
> **실 O3 numeric 부재 + 해석 프록시(Marquis near-delta) 인공물** 때문이지 모델/추가내용 때문이 아님.
> 근거: `lco_plainfit*`·`lco_ablation*`.

## 1. 흑연과 완전 동일 plain MSMR (자유 U·w·Q 로지스틱)
| 대상 | 데이터 | plain MSMR R² |
|---|---|---|
| 흑연 | 실측 | 0.9400 |
| **LCO O2 (Carlier2002)** | **실측(digitized)** | **0.9441** ← 흑연보다 나음 |
| LCO O3 (Marquis) | 해석 프록시 | 0.8546 ← near-delta 인공물(실측 아님) |

→ **실측 LCO(O2)는 흑연과 동일 방법으로 동등 피팅.** 유일 저조 = 해석 O3 프록시(스파이크 인공물).

## 2. @3·@5 도 LCO에 적용 (흑연·블렌드와 동일)
| LCO 실측(O2) | R² | ΔR² |
|---|---|---|
| plain(4피크) | 0.9441 | — |
| @3 정칙용액커널(per-peak Ω) | 0.9565 | **+1.25%p** |
| @5 전이추가(4→6) | 0.9630 | **+1.90%p** |
| @3+@5 | 0.9654 | **+2.13%p** |

→ **LCO도 @3·@5로 개선(오히려 음극보다 큼).** @3=3.70V 두-상 sharp(Ω>2RT)+order broad(Ω<2RT)를
per-peak Ω로 포착, @5=H1/H2·x0.5·H1-3 미세구조 추가로 포착. **개선 패턴이 음극과 동형.**

## 3. LCO 추가내용(전자 엔트로피)은 상온 피팅에 무영향
- LCO 유일 확장 = MIT 전자 엔트로피 ΔS_e≈−45.7 J/mol/K (`LCOCathodeDQDV`).
- **코드 자체 명시(L956)**: "ΔH 가 ΔS_e(T_ref)를 흡수해 T_ref 평탄역이 U 에 놓이게 재보정 —
  **ΔS_e 는 ∂U/∂T(가역열)에만 작용**." → 상온(T_ref) 곡선에서 전자항은 dH 에 흡수, U_j 자유피팅이 그대로 흡수 = plain과 동일.
- 전자항의 **유일 실작용 = T1 이동률 −0.47 mV/℃**(온도 벌어질 때만). = **Codex #5·#6이 지적한 바로 그 부분**.

## 4. 종합 판정
1. **LCO = 흑연 (방법·모델).** plain 0.944·@3/@5 개선 동형. 사용자 가설 확증.
2. **'LCO 안 맞음'의 정체** = ⓐ 실 O3 numeric 공개 부재(O2만 있음·폴리타입 상이) + ⓑ 해석 O3 프록시 불일치(Marquis 스파이크 vs Ramadass broad). **데이터 문제**, 모델/추가내용 아님.
3. **Codex #5·#6과의 관계** = 그것들은 전자항의 **∂U/∂T(온도 의존)** 지적 → 상온 피팅과 **무관**. 다온도 O3 실측이 있어야 검증 가능(현재 부재).
4. **반영 함의**: @3(정칙용액 커널)·@5(전이/staging 세분)는 **화학 불문 일반**(흑연·블렌드·LCO 전부 개선) → **v1.0.24 반영에 LCO 포함 가능**(O2 실측 근거). 전자항 T-의존(#5·#6)만 다온도 O3 대기(별도).
