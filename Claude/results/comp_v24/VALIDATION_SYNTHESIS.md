## 2. 핵심 검증 결과
### 2.1 평형 브로드닝 = MSMR (확증)
- 문건 평형식 `ΣQ_j·ξ(1−ξ)/w_j` 는 **MSMR(Verbrugge/Baker 2017–18)와 정확히 동형**(`w_j=ω_j·RT/F`). 코드가
  LCO 절에 명시한 매핑을 문헌·실측이 흑연까지 확인.
- 실측 두-상 피크 = near-delta(FWHM 3.5–4 mV = 0.14–0.16·RT/F, §7 예측 정합). 출하 `equilibrium()` 피팅
  R²=0.95(흑연)·**0.998(순수 Si)**·0.94–0.98(블렌드)·**0.99(NMC 양극)**·0.965(**LCO 첫 실피팅**).
- **U_j(전위중심) 셀·조성 무관 일관**(2→1=0.102±0.002 V·4→3=0.225±0.0015 V); w_j 두-상(≈2 mV)/shoulder(≈15 mV) 이봉.

### 2.2 M-factor 제거 (실증)
- dV/dQ 골 깊이 = 1/(dQ/dV 피크높이). 올바른 폭 w_j 로 dQ/dV 피크높이를 맞추면 dV/dQ 높이가 저절로 맞음 →
  전위축 stretch(M) 불요. 과대폭(w=RT/F) default 는 dV/dQ 골 ~11× 부풀음(=M 넣던 상황). **평형 절반 실증.**

### 2.3 블렌드 합성식 §3.5 (확증)
- blend = 흑연 host + Si host 가산(R²=0.94–0.98). 흑연 피크 위치 블렌드서 불변. 고Si `f_Si=0.776≈metadata 0.755`.

### 2.4 유한율속 ① (정성 확증·정량 스코프)
- 정성: 전류↑ → 피크 broaden·lower·shift(20 mV/mA)·washout(~10 mA). 문건 L_V∝|I|·분극 두 기제 실측.
- 정량: 상용 harvested SiOx 음극은 **비옴 농도분극**(Rn_eff 율속의존) 지배 → 옴 분극(V_n=V_app−|I|Rn)만으론
  정량 peak 위치 재현 한계. (순수흑연·박막 필요; 공개 부재.)

### 2.5 온도 ∂U/∂T = ΔS_rxn/F (확증)
- 출하 `entropy_coefficient_x` 가 O'Regan 측정 흑연 dU/dT 재현 R²=0.645. 피팅 ΔS_rxn 물리 정합(dilute 양수·dense 음수).

## 3. 모델 문제 vs 데이터 문제 (판정)
20셀 R² vs 데이터 품질(밀도·단조율):
- **데이터 나쁠 때**(sparse 실측 Si-graphene, 0.2 pts/mV): **R²=0.55 붕괴** — 순수 Si(clean)는 0.998. → **데이터 문제.**
- **데이터 깨끗할 때**(SINTEF·OCP): R²=0.95–0.998, 잔차=**모델**(두-상 near-delta).
- 결론: **둘 다 참** — 실무 데이터 품질이 흔한 원인이나, 문건식은 clean 데이터를 잘 설명. 밀도·단조율 자동기록 →
  임의 데이터의 모델/데이터 비중 즉시 판정 가능.

## 4. dQ/dV 추출 = BDD 스무딩 반영
- BDD/99_Backend 방법론 포팅: 다척도 중앙차분 median slope + **왕복공간 savgol 앙상블** + 적응 단/광폭 +
  **웨이블릿 denoise**(soft-threshold 피크보존). 단일 savgol 왜곡(피크 눌림·왕복 편향) 완화. 노이즈(±5 mV) 회복 1.8× 우수.

## 5. 개선 로드맵 (실측·문헌 근거 — 전부 추가 후보, P5)
| # | 방향 | 실측 검증 | 위험 |
|---|---|---|---|
| ~~1~~ | ~~전이 4→6~~ | **★XRD 판정으로 폐기**: 흑연 두-상 전이는 **4개**(Dahn 1991). 6개=curve-fitting(물리 위반). | — |
| 1′(수정) | **XRD 상구조 정합**: 두-상 4(Ω>2RT sharp)+4↔3 고용체(Ω<2RT broad) | `GRAPHITE_STAGING_XRD.md` — 현 4전이가 이미 두-상 4에 대응. shoulder=4↔3 고용체(새 상 아님) | 낮음(물리 정합) |
| 2 | 비대칭 두-측폭(Zhu 2023) | dQ/dV +1%(일부 피크 실측 비대칭) | 낮음–중 |
| 3(폭시드) | **Ω>2RT 기준 물리 시드**(사용자 제안) | 두-상=sharp seed·고용체=broad seed. w=RT/F 일괄은 물리 위반(문건 Ω와 모순) | 낮음 ★ |
| 4-단일상 | 정칙용액 Frumkin(단일 Ω) | **실패: R²=0.79<0.95(꼬리 무거움, §6)** + XRD상 staging 은 2-sublattice 필요 | — 폐기 |
| 4-정본 | **2-sublattice 정칙용액**(면내 Ω_a>2RT + staging Ω_c) | 원리적 정본(Bazant); 문건 성격 전환·무거움 | 높음(사용자 결정) |
- **★XRD 정정(사용자 지적 반영)**: 이전 "전이 4→6" 권고는 **curve-fitting 이라 철회**. XRD(Dahn 1991)는 두-상 4개만
  확정; 0.14–0.22 V shoulder 는 **stage 4↔3 고용체**(별도 상 아님). → 개선은 "피크 추가"가 아니라 **Ω>2RT 로 두-상/
  고용체를 물리적으로 구분해 시드**하는 것. entropy 도 이 4-상 구조와 정합(전이 추가 불요).
- **일관성(산업사용)**: 자유 per-cell 피팅의 w 불일치(178%)는 아티팩트 — 소재당 **전역/제약 피팅**이면 단일 물리셋이
  전 셀 설명(R²=0.94). 절대 w 는 추출 해상 의존(동일 파이프라인 통일 필수).

## 6. 정칙용액 프로토타입 (#4) — near-delta 해법 검증 (실행: `regsol_proto.png`)
- 정칙용액(Frumkin) 단일상 peak `dQ/dV=Q·F/|RT/(θ(1−θ))−2Ω|` (Ω<2RT) 로 흑연 4피크 피팅 vs 로지스틱.
- **결과: 정칙용액이 오히려 나쁨 — R²=0.79 vs 로지스틱 0.95.** 이유: Ω<2RT 단일상 peak 은 **꼬리가 무거워**
  (peak 높이 ~45 vs 실측 ~70) near-delta 재현 실패. 진짜 sharp 는 Ω→2RT(수치 특이) 근처만 나옴.
- **함의(로드맵 정련)**: near-delta 의 원리적 해법 #4 는 **단일상 Frumkin 이 아니라 두-상(Ω≥2RT + Maxwell
  construction=진짜 plateau)** 이어야 한다. 이는 OCV 를 비해석적으로 만들고 모델 성격을 바꾸는 무거운 확장.
- **결론**: 로지스틱(MSMR) R²~0.95 는 이미 합리적 작동점이며, 마지막 5%(near-delta)는 값싼 개선(#1 전이추가
  +0.2%·#2 비대칭 +1%·#4 단일상 −16%)으로 닫히지 않는 **근본 한계**. 두-상 Maxwell(#4-full)만이 원리적
  해법이나 문건 성격 전환 결정 필요(사용자). 그 전까지 MSMR + 자유폭 w_j 가 실용 최적.

## 7. 정직한 한계
- 두-상 near-delta 는 로지스틱(MSMR) 구조상 delta 재현 불가(ω→0 근사만) → R²~0.95 천장. #4(정칙용액)가 원리적 해법.
- LCO 고유 raw 부재(계산·Dualfoil 대리)·NMC≠LCO·저Si f_Si 축퇴·율속 정량은 상용셀 confound.
- 공개데이터=상온·(대부분)C/50 — 회사 표준 매트릭스(다온도×다율속 동일전극)는 사내 스크립트.
