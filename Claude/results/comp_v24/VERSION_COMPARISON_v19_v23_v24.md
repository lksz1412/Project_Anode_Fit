# 버전 비교 재검 — v1.0.19 vs v1.0.23 vs v1.0.24 (어투·구문·내용·논리)

> 사용자 요청(2026-07-22): "v19=구문 최고·v23=논리 최고. 이 둘과 v24를 어투·구문·내용·논리로 재검. 과거 이력 대조해 의도된 변경인지 퀄리티 하락인지 판정."
> 방법: `_sections/` 파일명 버전 안정 → 같은 논리 절 버전별 직접 대조(37 공통 절). 3창 병렬(A 흑연본론 sec00-10·B LCO+Part T·C 부록/Si) + master §1.1.4 검토. 이력 근거 = `plans/`(2026-05~07 리비전 계획 ~20)·각 버전 HANDOVER/PHASE·ledger.

## 0. 결론 (선요약)

**품질 하락 = 0건.** v19의 구문 완성도와 v23의 논리 사슬은 v24에서 **모두 보존**됐고, 3창 독립 감사 + 이력 대조에서 **무기록 품질저하는 발견되지 않았다**. 모든 v23→v24 변경은 사용자 1차 피드백(F-01~F-11)으로 추적되고, 모든 v19→v23 변경은 사용자가 상찬한 논리 심화다.

**유일한 실질 변화 = 의도된 어투(voice) 평탄화.** F-04(수필체·물음형 제목·구어 은유·정직 형용사 제거)·F-10(요동/양성 등 일본식 calque 영문화)·F-01(§1.1.4 압축)이 v19/v23의 "생생하고 투명한 교육적 목소리"를 얇게 만들었다. 이는 **v24 실행 오류가 아니라 사용자 본인이 요청한 trade-off**다. 되돌리려면 F-04/F-10 결정 자체의 재판단이 필요하다(아래 §4 복원 옵션).

## 4. ★ v23 논리 핵심 무결성 (별도 정밀 판정)

## 5. 복원 옵션 (사용자 판단 — 미집행)

voice trade-off가 과하다고 판단되면 아래는 **논리/교과서 register를 해치지 않고** 되살릴 수 있는 후보:

- **(권장) §1.1.4 "엔트로피 해석지도 → 본문 4절" 교차지도 문장 1개 복원**(항목 1). 수필체 아닌 항법(navigation) 문장 → 교과서 register와 무충돌. 압축본에 1줄 추가로 복원 가능.
- **(선택) appE·sec01_map의 투명/교육 voice 부분 복원**(항목 4·5). 단 이는 F-04(자기-서사·수사 제거) 결정의 부분 되돌림 → F-04 재판단 필요. "생존 지도"·"정직 공백 GS"처럼 **defined term으로 보존**하는 절충안 있음(v24가 이미 그 둘은 보존).
- **(선택) fluctuation 혼합 register 재판단**(항목 3). sec06/sec02b가 표본. 단 F-10(요동 영문화) 사용자 직접 지목 사항.

없음(현 상태 유지)도 정당 — 모든 변경이 사용자 요청의 충실한 집행.
