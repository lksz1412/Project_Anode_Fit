## (b) ★이중계산 B — 위반 초안 조사 (CRIT 판정)

**정확 규칙(GT 35-§3, L21):** 분해식의 `ΔS^config_j` 자리에 입력하는 값 = **전이 중심 표준값 ΔS^0_j**(전이 dict 단일수). 봉우리 *내부* 조성 의존 섞임항 R ln[(1−ξ)/ξ] 은 **logistic(ξ_eq·폭 w=RT/F)이 자동으로 담당** → dict 에 다시 더하면 이중계산.

### CRIT 후보 — v9-03 (L1004–1008): **논리 모순 (HIGH, CRIT 아님)**
- boxed 식(L997): `ΔS^cat = ΔS^config + ΔS^vib + ΔS_e`, 라벨 `ΔS^config = 중심 표준값 ΔS_j^0`.
- 그러나 prose(L1007–1008): **"ΔS_j^0 은 … 전체 반응 엔트로피의 중심값이다"** (= config+vib+electronic 전체).
- **모순**: box 에선 ΔS_j^0 가 config *한 항*의 라벨인데, prose 는 ΔS_j^0 를 *세 항 합 전체*의 중심값으로 재정의. 둘 다 참일 수 없음 — ΔS_j^0 가 전체 중심값이면 `ΔS^cat = ΔS_j^0 + ΔS^vib + ΔS_e` 는 vib·electronic 을 두 번 셈.
- **심각도 = HIGH(논리 결함), CRIT(코드 B 위반) 아님**: 이 빌드는 "외부에서 config 별도 가산 금지"는 옳게 말함(코드가 config 를 두 번 넣지 않음). 결함은 **ΔS_j^0 정의의 자기모순**으로 독자가 "그럼 box 의 config 항은 뭔가" 혼란. 채택 시 **반드시 정정** 필요.
- 정정안: prose 를 v9-04/05 처럼 — "ΔS_j^0 = 전이 중심에서의 **config(+vib) 표준값**이고, 봉우리 *내부* 분포는 logistic 이 담는다. ΔS^config 자리엔 ΔS_j^0 만 입력, 봉우리 내부 R ln 재가산 금지." (전체 반응 엔트로피 중심값이라는 문장 삭제.)

### 약-결함 — v9-01 (L449): **표기 동어반복 (LOW)**
- "중심 표준값 ΔS^0_j 는 중심 표준값에만" = 동어반복(무엇이 logistic 으로 담기는지 R ln 명시 누락). B 위반은 아니나 **불완전**. boxed 도 아니고 성분 미열거 → 분해 항목 최하위.

### 나머지 7종: **B 위반 없음 + 정확**
v9-04/05/06/08/09/02 모두 "config 자리=중심 표준값, 봉우리 내부 분포=logistic, R ln 재가산 금지" 정확. **v9-04 가 μ식 로그몫→S_mix 조성미분→폭 w 사슬을 끝까지 보여 교육적으로 압도**, v9-05/06 그 다음.
