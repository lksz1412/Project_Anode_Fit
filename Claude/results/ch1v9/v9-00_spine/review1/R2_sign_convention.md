## (a) ★ ΔS_e 부호 확정 판정 — **삽입 규약 (insertion convention): `ΔS_e ≡ +∂S_e/∂x|_T < 0` (MIT 구간 음의 소량)**

### 흑연 본문이 정한 규약 (3중 독립 확인, base_v8-11.tex)
1. **반응식 방향** (line 341, eq:eqbalance): 흑연 반응이 `Li⁺ + e⁻ + [] ⇌ Li_(흑연)` 로 **삽입(lithiation)을 forward** 로 적힘. 따라서 `δG=[μ̃_Li − μ̃_Li⁺ − μ̃_e⁻]dn=0` 로 정의되는 `ΔG_j`·`ΔH_rxn`·`ΔS_rxn` 은 **삽입 방향**(생성물 = Li_in_graphite) 양이다.
2. **인자 사전 라벨** (line 212, 9종 전부 동일): `ΔH_rxn,j, ΔS_rxn,j (dH_rxn, dS_rxn) = 삽입 반응 엔탈피·엔트로피`. — 명시적으로 **"삽입 반응(insertion reaction)"** 이라고 코드 인자 사전에 박혀 있음. 이것이 결정적 단서.
3. **수치 검산** (line 1050·1056): stage `2→1` (LiC₆, 완전 리튬화·최정렬) 이 `ΔS_rxn = −16 J/(mol·K) < 0`. Li 삽입으로 stage-1 정렬상 형성 → config 엔트로피 **감소** ⇒ 삽입 방향 ΔS < 0. ✓ `U=(−ΔH+T·ΔS)/F=(13000−298.15×16)/96485=0.0853 V` 가 표의 U=0.085 와 일치 → **표의 ΔS=−16 이 곧 func_U_j 가 직접 먹는 삽입-방향 값**임이 round-trip 으로 증명됨.

### ∴ 일관 부호
`∂U_j/∂T = ΔS_rxn,j/F` (eq:Uj 미분, line 375). `dS_rxn` slot 이 **삽입-방향** 엔트로피를 먹으므로, 같은 slot 에 들어갈 ΔS_e 도 **삽입-방향 부분몰** `ΔS_e ≡ +∂S_e/∂x|_T` 여야 1:1 정합.
- MIT 에서 Li 삽입(x↑) ⇒ metal→insulator ⇒ `g(E_F)↓` ⇒ `∂g/∂x<0` ⇒ **`ΔS_e = +∂S_e/∂x < 0`** (작음, MIT 국소 종).
- 결과: `∂U^cat/∂T|_e = ΔS_e/F < 0` (음의 소량 기여). **이것이 v9-02·v9-03 의 선택.**

### v9-01/04/06 의 "탈리튬화 양수(`−∂S_e/∂x>0`)" 가 틀린 이유 (refute)
이들은 "forward 전이가 탈리튬화(ξ:0→1) 방향이므로 ΔS_rxn 도 'Li 제거당'" 이라는 **전제**를 깔았다. 그러나 진행률 `ξ`(탈리튬화 0→1)와 **반응 엔트로피 `ΔS_rxn`(삽입 반응 기준, line 212·341)은 별개 객체**다 — 흑연 표의 `ΔS=−16` 은 ξ-step 당이 아니라 **삽입 반응 자체**의 값이고 그대로 `func_U_j` 에 들어간다. ξ 방향과 ΔS_rxn 규약을 **혼동**한 것이 결함의 뿌리. (θ=1−ξ 부호 한 번 차이는 logistic 인자에서 흡수되지, ΔS_rxn 의 *정의 방향*을 바꾸지 않는다.)

### +0.83 mV/K 은 ΔS_e 부호를 가르지 못한다 (v9-01 의 오용 반박)
단전극 `dφ/dT≈+0.83 mV/K ⇒ ΔS_rxn^cat≈+80 J/(mol·K)` 는 **config+vib 가 지배**(LCO ΔS≫NMC/LFP). 전자항 `ΔS_e≈0.18 k_B/atom≈1.5 J/(mol·K)` 은 +80 의 미소 성분 → **전체 +0.83 은 ΔS_e 가 음(소량)이든 양이든 양수**다. 따라서 +0.83 은 ΔS_e 의 sub-부호에 **agnostic**. v9-01 이 "+0.83 정합"을 ΔS_e>0 근거로 든 것은 **non-sequitur**. (설계문서 35 의 "탈리튬화당 +기여" 도 *물리 서술*일 뿐 — 시스템 엔트로피가 탈리튬화로 늘어 `−∂S_e/∂x>0` 인 건 참이나, 코드 `dS_rxn` 에 넣을 *규약 부호*는 삽입-방향이어야 한다. v9-03 이 이 둘을 명시 분리해 정확.)

## (d) v9-05 "ξ 고정 + 셀-라벨↔σ_d 매핑" 채택 권고 — **채택 (틀은 최선, 단 ΔS_e 부호는 v9-03 으로 교체)**

- **메커니즘**(v9-05 L276-286): `ξ`·`σ_d` 의미를 *전극*이 아니라 *ξ 진행 방향*(탈리튬화 진행률, "전위 올리면 느는 양")에 고정 → forward logistic 규칙 `σ_d=+1 ⇒ V↑⇒ξ_eq↑` 가 양극에서도 **글자 그대로** 성립. 양극에서 셀-방전=리튬화로 의미가 뒤집히는 것은 **N0 의 `_direction_to_sigma` 셀-라벨↔σ_d 매핑 한 곳**에서만 흡수. eq:eqcond·eq:Uj·eq:xieq·히스·꼬리 **전 사슬 byte-불변**.
- **왜 가장 견고한가**: (i) forward 부호식을 한 글자도 안 건드림 → 흑연 부호 사슬 self-test(S1-S8) 그대로 유효, 회귀 위험 0. (ii) 전극 차이를 *값*(U_j~4V)과 *라벨 매핑* 두 곳으로 국소화 → 검증 표면 최소. (iii) v9-01/04 식의 "ΔS 부호 플립으로 전극 차 흡수"보다 **부작용(역부호) 없음**.
