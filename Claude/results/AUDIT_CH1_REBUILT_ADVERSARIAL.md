## F. §7~9 집중 재감사 (2차, "피팅식을 실제로 만들려는 독자" 관점 — 정본 기준)

> 1차 감사가 놓친 핵심: **닫힌 식을 실제 계산하려고 하면 정의 안 된 객체와 사라진 합이 막는다.** 이것이 사용자가 "Ch1만으로 피팅 근사식을 못 만든다"고 한 실체.

| ID | 위치 | 결함 (봐주지 않고) | 심각도 |
|----|------|---------------------|--------|
| **H1** | §7 line 467-480 | **$A_0(L)$ 가 식이 없다.** 모든 꼬리 mode의 진폭을 싣는 핵심 객체인데 "입자 크기 분포·활성 site 밀도 같은 물리량"이라는 **말뿐**, 함수형도·구하는 법도·정규화도 없음. $A_L=A_0A_L^{prob}$ 의 $A_0$ 가 미정이면 kernel 적분(식 21)이 **수치적으로 비어 있다** → 피팅 불가의 직접 원인. | **CRIT** |
| **H2** | §8 line 493 | $A_0$ 에 **동역학량 $r(q_a)$(=$\xi_\eq-\xi_j$ 잔차)를 흡수**시킴. $r(q_a)$ 는 정적 물리량이 아니라 경로 의존 동역학량. 이를 "물리적 진폭 가중"에 섞으면 "$A_0$=독립 물리량, 역산 금지"(line 478,485) 라는 주장과 **자가당착**. | **HIGH** |
| **H3** | §7→§8 (432·495·517) | **전이 합 $\sum_{j=1}^{N_p}$ 이 증발.** §7은 전부 전이 $j$ 첨자($\xi_j,k_j,\mathcal A_j,\rho_G,A_j$)인데, 식 21·22의 $\Theta_\mathrm{tail},A_L,\mathcal K$ 엔 $j$ 도 $\sum_j$ 도 없음. 게다가 **두 종류의 다중성**(이산 전이 $N_p$ 개 vs 전이 내부 연속 배리어 $\rho_G$)을 한 $\int dL$ 로 **무근거 합침**. $\Theta=\sum_j(Q_{j,\tot}/Q_p)\xi_j$ 로 가는 조립 규칙 부재. | **HIGH** |
| **H4** | §8 (495·517) vs §9 (542) | **$\Theta$ 일가가 정의 전에 등장.** $\Theta_\mathrm{tail}$ 은 §8(495)에서 먼저 쓰이고 $\Theta$ 정의는 §9(542). $\Theta_\eq,\Theta_\mathrm{init}$(식 22)은 **끝까지 정의 없음**. 사용자의 "신규 변수가 정의 없이 등장"이 정확히 여기. | **HIGH** |
| **H5** | §7 line 429-434 | 소문자 $g$ 가 "배리어를 가진 집단"으로 등장하나 $G$ 와의 관계(값/적분 더미)를 첫 사용에서 명시 안 함. 기호표엔 $G$ 만 있음. | MED |
| **H6** | 제목·§9 제목 vs line 528-530 | **"피팅 가능한 닫힌 논리식" 과대표방.** 식 25는 $\dd\Theta/\dd Q$ 를 요구하고, 그건 자기참조 Volterra(식 22) 풀이가 있어야 하는데 **Ch6 로 연기**. 게다가 $\rho_G,A_0$ 는 미지 입력. 즉 Ch1 산출물은 **닫힌 식이 아니라 구조**다. (단, "closure·수치는 Ch6 이관"은 P1 프로젝트 방침이기도 함 → **설계 의도 vs 사용자 현 기대 충돌**, 결정 필요.) | **HIGH(결정)** |
| **H7** | §5 boundbox vs §6·§10 | (1차 G5 유지) 선형 $-\chi\mathcal A$ 는 $\mathcal A$ 작을 때 유효라면서 꼬리=post-peak는 $\mathcal A$ 커지는 영역 → 유효범위 자기모순. | MED-HIGH |
| **H8** | §8 식 21 vs 식 22 | 꼬리에 대한 **두 표현**(식 21=도함수형, 식 22=목표형 Volterra)의 관계가 안 묶임. 독자가 "피팅엔 어느 식을 쓰나" 판단 불가. | MED |

**2차 결론:** 사용자 지적은 정당. 핵심 병목은 **H1(=$A_0$ 미정의) + H3(=$\sum_j$ 증발) + H4(=$\Theta$ 미정의)** 의 결합 — 이 셋 때문에 식 21/25가 **기호적으론 닫혀 보여도 수치적으론 비어** 피팅 불가. H6은 "Ch1이 어디까지 책임지나"라는 **설계 결정** 사안.
