### I-01 [HIGH] Ch2 §1.2 — 폭 w=RT/F를 이상 단상 전이에만 유효한 값으로 도입하나, 문서 흐름에서 독자가 두-상 전이(흑연 staging)에도 동일하게 적용되는 것으로 오해할 수 있는 도입부 불명확성

- **위치**: Ch2 graphite_ica_ch2_v1.0.10.tex, §1.2 ssec:logistic, 줄 155 `w≡RT/F`; keybox 줄 162-165
- **물리적으로 틀린 것**: `w≡RT/F`가 단일 자리 2-상태 격자기체(이상 단상)에서 나오는 열적 폭임에도, Ch2 keybox(줄 162)에 "logistic 폭 w=RT/F는 임의 모수가 아니라 단일 자리 2-상태 분배함수가 정하는 분포의 열적 폭이다"라고 서술하고, 그 직후 문서 흐름이 흑연 staging 전이 분석으로 이어진다. 독자는 흑연 두-상 전이의 dQ/dV 종 폭도 RT/F로 예측 가능하다고 오해할 수 있다.
- **왜 문제**: §1.3 BW·§4(파생 C)에서 두-상 전이의 w는 현상학적 자유 피팅 폭이라고 뒤늦게 정정하는 구조다. 그러나 §1.2 keybox에서 이미 "임의 모수가 아니라 분배함수가 정하는 폭"이라고 단정해 버리면, 뒤에서 "두-상에서는 피팅 파라미터"라는 정정이 논리 역전처럼 읽힌다(G-follow 단절).
- **근거**: keybox 줄 162 `w=RT/F`는 이상 단상(Ω=0) 극한이다. Ω>2RT(흑연 staging)에서 평형 단일입자 등온선은 plateau = near-delta이고 w≈0이며, 실측 종 폭은 broadening에서 온다(§4 파생 C 본문). 이 두 서술이 keybox ↔ §4 사이에 선언 없이 전환되고 있다.
- **렌즈**: 물리적 논리의 연결 끊김(G-follow), 극한조건 과도 일반화

### I-02 [HIGH] Ch2 §2 eq:dVdT_config — 편미분 ∂V/∂T|_ξ 계산에서 두 번째 항(w(T) 의존) 유도가 생략됨

- **위치**: Ch2 §2, 줄 235-243, eq:dVdT_config
- **물리적으로 틀린 것**: 식 (2.6)에서
  ```
  ∂V/∂T|_ξ = ΔS_rxn,j/F + (R/F)·ln[ξ/(1-ξ)]
  ```
  라고 결론으로 제시하는데, "w=RT/F의 명시적 T 의존이 둘째 항을 낳음"이라는 설명(줄 240-243) 한 줄로 유도를 끝낸다. 구체적으로 V(ξ) = Uj + (RT/F)·ln[ξ/(1-ξ)]를 T로 편미분할 때:
  ```
  ∂V/∂T|_ξ = ∂Uj/∂T + (R/F)·ln[ξ/(1-ξ)] + (RT/F)·∂ln[ξ/(1-ξ)]/∂T|_ξ
  ```
  세 번째 항 `(RT/F)·∂ln[ξ/(1-ξ)]/∂T|_ξ`가 0이 되는 이유가 제시되지 않는다.
- **왜 문제**: T를 고정할 때 ξ도 고정이므로 ln[ξ/(1-ξ)]의 T 의존이 없다는 것을 명시해야 한다. 그 설명이 없으면 독자가 "왜 세 번째 항이 없는가"를 의문으로 남기게 된다(G-follow 단절, 유도 비약).
- **근거**: 식 (2.5) V(ξ)=Uj+(RT/F)·ln[ξ/(1-ξ)]를 T로 편미분 시 체인률 적용. 고정 ξ에서 ln[ξ/(1-ξ)]는 T 무관이므로 세 번째 항=0이지만, 이 사실이 본문에 명시되지 않는다.
- **렌즈**: 논리 점핑(유도 비약)

### I-03 [HIGH] Ch2 §4(파생 B) — 이중계산 금지 warnbox의 ΔS^0_j 정의가 Ch1 코드(func_U_j) 입력 dS_rxn과 일치하는지 독자가 추적하기 어려움

- **위치**: Ch2 §4(파생 B) warnbox, 줄 294-309, eq:weighted 종합식(줄 672-684)
- **물리적으로 틀린 것**: Ch2 warnbox(줄 295)에서 "ΔS^0_j ≡ ΔS_rxn,j ≡ [ΔS(x)]_{ξ=1/2}로 한 기호로 정의한다(봉우리 중심 ξ=½에서 측정되는 값, config=0)"라고 선언한다. 그러나 코드(줄 78, func_U_j)에서 dS_rxn은 logistic 평형 중심 U_j(T) = (-ΔH_rxn + T·dS_rxn)/F를 결정하는 입력이며, entropy_coefficient 함수(줄 548-578)에서 `dS_eff/F + config`로 쓰인다. 즉 코드의 `dS_rxn`이 정확히 Ch2의 ΔS^0_j (ξ=½에서 config=0인 봉우리 중심값)임을 명시적으로 연결하는 bridge가 없다.
- **왜 문제**: 독자가 코드를 보고 "dS_rxn이 ΔS^0_j 와 같은가 아니면 다른 양인가"를 스스로 추론해야 한다. Ch2 파생 B의 핵심 주장(이중계산 금지)은 ΔS^0_j의 정체에 달려 있으므로, 코드 연결 없이 문건만으로는 G-usable이 낮다.
- **근거**: 코드 entropy_coefficient 줄 562-563: `dS_eff = self._effective_dS_rxn(tr, T)`, 줄 575: `num += Qg*(dS_eff/F + config)`. Ch2 eq:weighted (줄 672): `ΔS^0_j/F + (R/F)·ln[ξ/(1-ξ)]`. 두 식이 구조상 동일하나 Ch2 본문에 "코드 dS_rxn = Ch2 ΔS^0_j"라는 명시 연결이 없다.
- **렌즈**: G-usable, G-follow

### I-04 [CRIT] 코드 entropy_coefficient — func_ksi_eq 호출 시 σ_d=+1 고정으로 히스 분기 중심이 아닌 equilibrium 중심 U_j를 쓰는 것이 Ch2 파생 D(분기별 ∂U/∂T)와 불일치

- **위치**: Anode_Fit_v1.0.10.py, 줄 556-578, entropy_coefficient; Ch2 §4 파생 D(줄 570-593) eq:hys_branch, eq:hys_rev
- **물리적으로 틀린 것**: 코드 entropy_coefficient(줄 560-576)에서 `func_ksi_eq(T, V, U_j, n_j)`를 호출할 때 사용하는 U_j는 히스테리시스 분기가 없는 equilibrium 중심이다(σ_d 인자가 없고, func_U_branch 호출이 없다). 반면 Ch2 파생 D(eq:hys_branch, eq:hys_rev)는 "가역 발열에 들어가는 것은 두 분기의 평균 ½(∂U^ch/∂T + ∂U^dis/∂T)"라고 명시하며, 각 분기의 봉우리 모양 g_j^(d)(x)로 가중된 별도 식이 필요하다.
- **왜 문제**: 흑연 staging 전이는 Ω>2RT로 히스테리시스가 실재한다(코드 GRAPHITE_STAGING_LIT: Ω=6000~13000 J/mol, 298K에서 2RT≈4958 J/mol이므로 전이 모두 Ω>2RT). 히스 분기 중심이 방전/충전에서 다를 경우, 두 분기 평균이 아닌 단일 equilibrium 중심 U_j로 ξ를 평가하면 g_j가 분기 평균 가중과 다르다. 코드 주석(줄 552-554)에서 "γ 대칭 전제이며, 비대칭 분기별 ∂U/∂T는 미구현"이라고 라벨은 있으나, 이것이 Ch2 파생 D가 제시하는 식과 얼마나 벗어나는지 정량화가 없다.
- **그래프 관찰**: P5 V3 패널(graphite q_rev): V≈0.05~0.15V 구간에서 q_rev가 선형적으로 변하는데, 히스 분기 비대칭이 있다면 방전/충전 평균이 현재 코드 결과와 다를 것이다. γ>0인 모든 전이에서 이 오차가 누적된다.
- **렌즈**: 물리 오표현(왜곡), G-usable, 코드-문건 불일치

### I-05 [HIGH] Ch2 §3.2(electronic 엔트로피) — Sommerfeld 적분 유도의 단위 불일치: S_e = (π²/3)·k_B²·T·g(E_F)에서 kB² 차원이 불명확

- **위치**: Ch2 §3.2, 줄 391-398, eq:Se
- **물리적으로 틀린 것**: 식 (2.14): `S_e = (π²/3)·k_B²·T·g(E_F)`에서 "k_B² 중 한 몫은 T·g(E_F)(g가 1/에너지 차원)가 상쇄해 알짜로 k_B가 남는다"고 설명한다(줄 397). 그러나 여기서 g(E_F)의 단위가 무엇인지(자리당 에너지 역수인가, 몰당 에너지 역수인가)를 명시하지 않는다. Sommerfeld 표준 교과서(Ashcroft & Mermin 기준): g(E_F)는 단위 부피당, 또는 원자당, 또는 몰당으로 정의가 갈린다.
- **왜 문제**: "자리당 k_B 차원이라 몰당 R·k_B" (줄 397-398)라는 서술이 단위 논증 없이 결론만 제시한다. 독자가 S_e의 단위를 직접 확인할 수 없어 G-follow가 떨어진다. 실제로 `N_A·k_B = R`이므로 "k_B → R"은 자리당→몰당 환산이고, 이 변환에서 g(E_F)가 어떤 기준으로 정의됐는지가 중요하다.
- **근거**: 줄 394: `S_e = (π²/3)·k_B²·T·g(E_F)` — 이 식이 맞으려면 g(E_F)가 [1/에너지] 단위여야 하고, S_e의 단위가 [k_B²·K·에너지⁻¹] = [k_B²·K·(k_B·K)⁻¹] = [k_B]로 맞아야 한다. 이 논증이 문건 내에 없다. 코드 func_dSe_molar(줄 176): `ΔS_e = -(π²/3)·R·(k_B·T/e_V)·(g_max_eV/dx_MIT)·σ(1-σ)`에서 단위 주석(줄 177-179)은 "÷e_V(eV⁻¹→J⁻¹)"를 명시하나, Ch2 §3.2 본문은 이 단위 다리를 생략한다.
- **렌즈**: 논리 점핑, 차원 불명확(포괄)

### I-06 [HIGH] Ch2 §6(가역 발열) 부호 규약 srcbox — ΔS = +F·∂U_oc/∂T로 정의하나 전셀 조립 시 음극 몫 부호 전환 내용이 범위 외로만 처리됨

- **위치**: Ch2 §6, 줄 650-654, srcbox
- **물리적으로 틀린 것**: srcbox에서 "전셀 조립 시 음극 몫은 부호 반전 +IT·∂U_an/∂T로 들어가나 본 장 범위 밖"이라고 짧게 선언만 한다. 그러나 Ch2의 주 사례가 흑연 음극 하프셀이고, 가역 발열 식 q_rev = -I·T·∂U_oc/∂T에서 U_oc가 흑연 하프셀 전위인지 전셀 전위인지가 사용자에게 혼동을 줄 수 있다.
- **왜 문제**: 코드 reversible_heat(줄 580-588)는 `return -float(I) * T * self.entropy_coefficient(V_n, T)`로 단순히 I·T·∂U/∂T를 쓰는데, 이것이 하프셀 기준임을 알아야만 부호가 맞다. 전셀 BMS 적용 시 음극 q_rev는 +I·T·∂U_an/∂T(반대 부호)임을 Ch2 본문에서 명시적으로 경고하지 않으면 오적용 위험이 있다.
- **근거**: 식 (2.21): `q_rev = -I·T·∂U_oc/∂T = -(I·T/F)·ΔS(x)`. ΔS>0(저-x, config 우세) + 방전 I>0 → q_rev<0(흡열). 전셀에서 음극 q_rev = +I·T·∂U_an/∂T → q_rev>0(발열)로 부호 역전. 이 전환이 srcbox 한 줄로만 처리된다.
- **렌즈**: G-usable, 물리 범위 오표현 위험

- **그래프 실측**: P4 패널(a)에서 V≈0.085V 이하의 낮은 전위 구간에 2→1 전이 꼬리는 보이나 별도 spike로 분리되지 않는다. 전위 범위 0.03~0.34V를 포함함에도 4→3 전이(U=0.210V) 구역에서 별도 봉우리 없음.

## §4 정독 후 무결 영역

### 무결 확인: Ch2 §1.1 — 단일 자리 분배함수 Z_1 유도

eq:Z1(Z_1 = 1 + e^{-β(ε₀-μ)})과 eq:occ(⟨n⟩ = 1/(1+e^{β(ε₀-μ)})) 유도는 정확하다. 2-상태 합의 Gibbs 인자 적용이 교과서 표준이고 오류 없음.

### 무결 확인: Ch2 §2(configurational 엔트로피) eq:Sconfig 유도

`S_config = -R[θlnθ + (1-θ)ln(1-θ)]`가 Bernoulli 분포의 Shannon 엔트로피 `-R Σ p_i ln p_i`에서 나옴은 정확하다. 부분몰 미분 eq:dSconfig도 수학적으로 올바르다.

### 무결 확인: Ch2 파생 D(히스테리시스) — 가역/비가역 분리 원칙

eq:hys_branch와 eq:hys_rev의 "가역 발열 = 분기 평균" 원칙은 물리적으로 타당하다. 히스 gap Δ U^hys가 비가역 소산이고 분기 평균이 가역 엔트로피에 대응한다는 서술은 정확하다.

### 무결 확인: Ch2 가역 발열 부호규약 srcbox(줄 651-653)

`ΔS = +F·∂U_oc/∂T` 규약과 `q_rev = -I·T·∂U_oc/∂T`의 Bernardi 출처 연결은 정확하다. 방전(I>0)에서 ΔS>0→흡열 부호가 맞다.

### 무결 확인: 코드 reversible_heat(줄 580-588) — T²가 아닌 T 한 번

`return -float(I) * T * self.entropy_coefficient(V_n, T)`에서 entropy_coefficient가 이미 [V/K] = [∂U/∂T]이므로 T를 한 번만 곱한다. T² 오류 없음.

### 무결 확인: 코드 func_dSe_molar — 단위 검산

줄 176-179: `ΔS_e = -(π²/3)·R·(kB·T/eV)·(g_max_eV/dx_MIT)·σ(1-σ)`. 검산: g_max_eV=13·dx_MIT=0.05·T=298 → 골 깊이≈−45.7 J/mol/K로 코드 실행 시 −45.7이 확인되고 Ch1 검증값 −46과 정합(demo_lco_heat.py 줄 21-22 print 결과에서 "기대 ~−45.7" 명시). 단위 3중 가드(÷e_V, R·kB, 부호)가 구현되어 있음.

1. **초점(Ch2 발열·엔트로피·이중계산)**: CRIT 1건(I-07 그래프 개형), HIGH 5건(I-01·I-02·I-03·I-04·I-06), MED 4건, LOW 1건. 총 11건의 물리·코드 문제 발견.

2. **문제 수**: 11건(CRIT 1, HIGH 5, MED 4, LOW 1). cross-version 회귀 4건 중 정당 컷 3건, 물리 입장 전환 1건(CV-03).

3. **최중대 문제**: I-07 — 실측 그래프에서 4 staging spike가 분리되지 않고 단일 뭉갠 bell로 나타난다. GRAPHITE_STAGING_LIT의 n=1(w=RT/F)이 초기값으로 이상 단상 폭을 쓰고 있으나, Ch2 파생 C의 "두-상 전이에서 w는 현상학적 피팅 폭"이 코드에 미반영됨. I-04(코드 entropy_coefficient에서 히스 분기 비대칭 미구현)도 CRIT급.

4. **발열 그래프 개형 판정**: 흑연 q_rev (P5 V3): V≈0.05~0.15V 구간에서 선형 하강(ΔS<0) 후 저전압에서 ΔS>0으로 부호 전환하는 형태는 물리적으로 합리적이나, 발열 프로파일의 세부 모양은 4 staging 분리가 안 된 dQ/dV 개형에 종속되어 있으므로 신뢰도 낮음.

5. **오적발 가능성**: I-10(부분몰 발산 서술)은 정성적으로 맞고 정밀도 문제이므로 오적발 가능성 있음(정확성보다 교육적 명확성 문제). I-13(수치 검증 재현 불가)은 결과 자체의 물리 오류가 아닌 문서 불완전성이라 CRIT로 오판하지 않음.
