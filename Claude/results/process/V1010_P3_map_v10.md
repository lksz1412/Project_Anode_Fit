## §2. 발열 이론 갈고닦기 (중점) (채택: O2 §2-1 4단계 박스 + O1 §2-4 비가역 3분해)

> 작업지시 ②. q_rev 유도를 식→식으로 닫고 ΔS↔q_rev·비가역·부호·차원·극한을 자기완결로. **발열 코드는 P4** — 본 §은 이론 정련.

### 2.1 q_rev 유도 — Bernardi 수지 → −I·T·∂U/∂T (채택: O2 §2-1 4단계, 식→식)

**(a) 출발 — 셀 에너지 수지.** 등압 전지의 발열률은 전기 일과 엔탈피 변화의 차로 닫힌다. 단일 활물질 하프셀에서 Bernardi–Pawlikowski–Newman 일반 수지(bernardi1985, Ch2 eq:qrev L643)는

```
Q̇ = I(U_oc − V) − I·T·(∂U_oc/∂T)          (Ch2 L643)
     └─ Q̇_irr ≥ 0 ─┘   └──── Q̇_rev ────┘
```

여기서 `I(U_oc−V)`는 과전압 소산(2법칙상 항상 발열; Ch1 동역학 꼬리·분극이 만드는 entropy production), 둘째 항이 가역열이다.

**(b) 연산 — 가역항을 반응 엔트로피로.** 평형 전위와 반응 Gibbs 에너지가 `ΔG=−FU_oc`(Ch1 eq:eqcond)로 묶이므로, Gibbs–Helmholtz `ΔS=−∂(ΔG)/∂T`에 대입하면

```
ΔS = −∂(ΔG)/∂T = −∂(−FU_oc)/∂T = +F·(∂U_oc/∂T)        (★부호 규약 통일, Ch2 srcbox L651)
```

★부호 확정 다리: ΔG=−FU 의 음부호와 ΔS=−∂ΔG/∂T 의 음부호가 곱해져 **+**가 된다(Newman·msmr_partI 정합).

**(c) 중간식 — Q̇_rev에 역대입.** `∂U_oc/∂T = ΔS/F`를 가역항에 넣으면(치환이지 곱셈 아님)

```
Q̇_rev = −I·T·(∂U_oc/∂T) = −I·T·(ΔS/F) = −(I·T/F)·ΔS(x)    (T 정확히 한 번)
```

**(d) 박스 — 닫힌 발열식.**

```
┌─────────────────────────────────────────────────────────┐
│  Q̇_rev = −I·T·(∂U_oc/∂T) = −(I·T/F)·ΔS(x)               │   (Ch2 eq:qrev, L645)
└─────────────────────────────────────────────────────────┘
```

여기서 ΔS(x) = §sec:vibel 세 분포의 합(config+vib+electronic) = +F·∂U_oc/∂T. [확정]

> **★q_rev 무결 확정 [확정 채택 사실]**: 화살표 `Bernardi 수지 → ΔG=−FU → Gibbs–Helmholtz → ΔS=+F∂U/∂T → Q̇_rev=−(IT/F)ΔS` 4단계 모두 점프 없음. **T는 정확히 한 번** 곱해진다. (c)에서 ∂U/∂T를 ΔS/F로 *치환*하는 것이지 *곱하는* 게 아니다 — 이중곱 오류 없음(§2.3 #5 차원 검산이 확정).

### 2.2 ΔS ↔ q_rev 부호 사슬 (흡열/발열 교대)

방전(I>0)에서 식 (c)가 부호를 곧장 정한다:

### 2.5 비가역 분해 (채택: O1 §2-4, IR+η_ct+η_diff)

가역(Q̇_rev)과 대비되는 비가역 소산열. 과전압 `η ≡ U_oc − V`를 세 몫으로 분해(Ch1 분극 N1·sec:pol 정합):

```
η = I·R_n        (ohmic, IR; 코드 V_n=V_app−σ_d|I|R_n, L408)
  + η_ct          (전하전달, Butler–Volmer; η_ct≈(RT/F)·|I|/i₀ 소전류극한)
  + η_diff        (확산/완화; Ch1 꼬리 L_q·_causal_lowpass)

∴ Q̇_irr = I·η = I²·R_n + I·η_ct + I·η_diff ≥ 0        (2법칙)
```

- **I²R 항**: 순 Joule 열, 항상 양, T-미분 무관(가역열에 안 섞임).
- **η_ct·η_diff 항**: 동역학 소산. Ch1 활성화 엔트로피 `ΔS_a`(코드 L100 `dS_a`)는 **이 비가역 동역학**에 들어가지 가역열엔 안 들어간다 — Ch2 §elec "활성화 엔트로피는 비가역 동역학을 지배하고 가역 발열에 안 들어감"의 식적 근거.

★핵심 분리 명제: `∂U/∂T` 미분은 **가역열만**(§2.1), η 의 소산은 **비가역열**. 하나의 ∂U/∂T로 뭉치면 안 된다 — Ch2 파생 D(히스 gap 비가역 분리)와 같은 분리 원리. **히스 비가역**: 한 사이클당 ∝ I·ΔU^hys의 entropy production, ∂/∂T와 별개(가역열 불참, Ch2 L586-587).
