## 2. ★B-2 — 방향 규약 원자 클러스터 (F-2 + R-1/F-7 + F-5④-2): **수정 채택 — 옵션 (i) f=+σ_d 정정 + 진행률 pairing 명시**

세 건은 하나의 방향 규약이므로 원자 판정한다. 결론: **(i) 채택 / (ii) 기각**. 아래는 master 자체 제1원리 재유도 전문이다.

### 2.1 외부 실측 — MSMR 원계열 부호 규약 (확정)

원계열(Baker–Verbrugge 계열, PyBaMM MSMR 공식 문서 실측):
$$U_j=U_j^0+\frac{\omega_j}{f}\ln\frac{X_j-x_j}{x_j},\qquad f=\frac{F}{RT}\ (>0\ \text{항상})\ \Longleftrightarrow\ x_j=\frac{X_j}{1+\exp[f(U-U_j^0)/\omega_j]}.$$
$x_j$ = **점유(리튬화) 분율**로 $U$ 에 단조 **감소**, $\omega_j$ = 무차원 폭(Nernst 이탈 모수, $\omega_j=1$ 에서 Nernst 회복). msmr2024(L1954 bibitem, DOI 10.1149/2754-2734/ad7d1c)는 이 계열의 **흑연** 파라미터화 논문(Part I 본문 식은 이미지 렌더라 직접 인용 불가 — 그러나 $x_j$·$\omega_j$ 정의·quasi-Nernst 서술을 같은 논문 페이지에서 확인, 계열 규약과 완전 일치. 잔여 리스크 극소).
출처: [PyBaMM MSMR 문서](https://docs.pybamm.org/en/stable/source/examples/notebooks/models/MSMR.html) · [ECS Advances Part I](https://iopscience.iop.org/article/10.1149/2754-2734/ad7d1c).
⇒ **F-5④-2 는 확정**: Ch1 L1777 "MSMR 의 $f$ 는 종별 부호 인자 $\pm1$" 은 원계열과 불일치. $f=F/RT$ 를 폭에 흡수한($\omega_j\mapsto\omega_jRT/F$, 전압 차원) **재모수화**로 서술해야 정확하고, 그 재모수화가 흑연 $w_j=n_jRT/F$($n_j\leftrightarrow$무차원 $\omega_j$)와의 구조 일치를 드러낸다(F1 정정문안 인준).

### 2.2 제1원리 재유도 — pairing 과 부호 (master 독립)

원문 규약 실측: `eq:xieq`(L787) $\xi_{\eq,j}=1/(1+\exp[-\sigma_d(V-U_j^d)/w_j])$ · $\theta_j=1-\xi_j$(L211–213: $\xi$=탈리튬화 진행률, $\theta$=점유) · $\sigma_d=+1$ 방전/$-1$ 충전(L203) · 흑연 방전 = 탈리튬화 = 문서 주 전개(L216–217).

1. **정합 pairing 은 두 가지뿐이고 둘 다 $f=+\sigma_d$ 를 준다**:
   - 점유↔점유: $\theta^\mathrm{MSMR}=1/(1+e^{+f(\cdot)/\omega})$ ↔ $\theta_\eq=1/(1+e^{+\sigma_d(\cdot)/w})$ ⇒ $f=+\sigma_d$. (Ch2 `eq:logistic` L152–153 의 $\theta_\eq$ 형과도 동일.)
   - 진행률↔진행률: $\xi^\mathrm{MSMR}=1-x_j/X_j=1/(1+e^{-f(\cdot)/\omega})$ ↔ $\xi_\eq=1/(1+e^{-\sigma_d(\cdot)/w})$ ⇒ $f=+\sigma_d$.
2. **$f=-\sigma_d$ 는 점유=진행률 등치**: $f=-\sigma_d$ ⟺ $\theta^\mathrm{MSMR}=\xi_\eq^{(\sigma_d)}$ — 리튬화 분율을 탈리튬화 진행률에 직접 등치한 여집합 뒤바꿈. map §5(c)의 괄호 해설("MSMR 점유형 = 반대 방향 인자로 평가한 ξ-서식", 즉 $\theta^\mathrm{MSMR}=\xi^{(-f)}$)을 그대로 풀면 $f=+\sigma_d$ 가 나와 **해설과 결과가 문장 안에서 모순**(F1r·O1r 합치 — master 재확인).
3. **검산 1(흑연 = msmr2024 실대상)**: 주 전개 $\sigma_d=+1$ 에서 $f=-\sigma_d=-1<0$ — 원계열 $f=F/RT>0$ 와 정면 모순이고, $x_j$ 가 $U$ 에 증가(점유가 전위 상승에 증가 — 비물리).
4. **검산 2(LCO 셀-라벨 상쇄)**: 충전 $\sigma_d=-1$ 을 먹이면 $f=+1$ 로 겉보기 정합 — 그러나 그때 $\xi_\eq^{(-1)}$ 는 $V$ 에 감소해 L314("충전=탈리튬화=$\xi:0\to1$·전위 상승")와 모순 = **정확히 F-2 의 결함**. 곧 $f=-\sigma_d$ 는 두 오류(뒤바뀐 pairing × 셀-라벨 σ_d)가 상쇄될 때만 살아남는 대응이다(F1r 논지 — master 독립 재확인).
5. **관측 불변**: $\theta(1-\theta)=\xi(1-\xi)$ — peak 박스(`eq:lco-msmrpeak`)·dQ/dV·"같은 코드가 LCO 를 그린다" 결론 전부 무손상(잠복형). 판정 기준은 서술의 물리 일관성.
6. **X_j↔Q_j 구조 대응 반론(S1r LOW) 처분**: $\sum X_j\theta_j$ = 리튬 함량, $Q_\bg+\sum Q_j\xi_j$ = 추출 전하 — 두 합 자체가 **서로 여집합 관계의 양**이므로, 합 구조를 근거로 피가수 $\theta\leftrightarrow\xi$ 를 슬롯 등치하는 것이 바로 "함수형 동형 ≠ 물리량 동일"(O1 가드) 위반이다. 반론 기각 — 다만 (b) 문장에 이 구분을 명시한다(아래 실행문).

**④** map §5 (c) — "이제 식~\eqref{eq:lco-msmrnorm} 의 점유형 지수 …" 부터 "…기본형이다)." 까지 통째 교체(R-1 핵심):
```latex
이제 \emph{같은 물리량끼리} 맞댄다 — MSMR 점유율 $\theta_j^{\mathrm{MSMR}}=x_j/X_j$(리튬화 분율)는 Ch1 의
점유 $\theta_{\eq,j}=1-\xi_{\eq,j}$ 와, 그 여집합 $\xi_j^{\mathrm{MSMR}}$(탈리튬화 진행률)는 Ch1 의 진행률
$\xi_{\eq,j}$(식~\eqref{eq:xieq})와 같은 종류의 양이다. 진행률끼리 지수를 맞대면($-f\leftrightarrow-\sigma_d$),
점유끼리 맞대도($+f\leftrightarrow+\sigma_d$) 동일하게, 슬롯별로
\begin{equation}
U\leftrightarrow V,\qquad U_j^0\leftrightarrow U_j^{\,d},\qquad
\omega_j\leftrightarrow w_j,\qquad X_j\leftrightarrow Q_j,\qquad
f=+\sigma_d
\label{eq:lco-msmrmap}
\end{equation}
가 1:1 로 읽힌다 — 방향 인자 대응 $f=+\sigma_d$ 는 두 지수가 \emph{모든} $U$ 에서 일치할 것을 요구하는
항등식의, 같은 물리량끼리 짝짓는(점유↔점유$\cdot$진행률↔진행률) pairing 하에서의 유일해다. 원계열의
$f=F/RT>0$ 는 이 규약에서 탈리튬화 진행($\sigma_d=+1$ 슬롯)에 대응하며, MSMR 평형 등온선 자체는 방향이
없는 양이라 $\sigma_d$ 는 훑는 방향의 해석만 정하고 평형 종은 여집합 교환(식~\eqref{eq:lco-comp})에
불변이다. ★같은 logistic 이라는 것은 \emph{함수형 동형이지 물리량 동일이 아니다} — $x_j/X_j$ 는 리튬
함량 몫(점유)이고 $\xi$ 는 탈리튬화 진행률이라, 둘을 직접 등치하면 부호가 뒤집힌 $f=-\sigma_d$ 가 나온다
(여집합 뒤바꿈 — 종 $\xi(1-\xi)$ 는 이 교환에 불변이라 관측 peak 은 어느 쪽으로도 같으나, 방향 서술은
그렇지 않다). 이로써 탈리튬화/리튬화 진행 방향이 두 모델에서 일관되게 맞춰진다($U_j^0\leftrightarrow
U_j^{\,d}$ 대응은 방향분해 파라미터화로 읽으며, 순정 MSMR 평형 등온선은 $\gamma_j{=}0$ 의 $U_j$ 대응이
기본형이다).
```

## 6. 판정의 한계·자기표시

- F-5④-2 의 "원 논문 Part I 본문 식" 은 이미지 렌더라 자구 직접 인용은 미확보 — 그러나 같은 페이지의 $x_j$·$\omega_j$ 정의 서술 + 계열 공식 문서(PyBaMM: $f=F/RT$ 명문) 이중 실측으로 **확정** 처리했다(뒤집힐 여지: 원 논문이 계열 규약과 다른 자체 재정의를 썼을 가능성 — 극소, 그 경우에도 ④-2 정정문의 "재모수화" 서술과 $f=+\sigma_d$ 결론은 문서 내부 정합성만으로 성립).
- R-1 의 유일한 반박 독법(σ_d 를 "슬롯값"으로 읽기)은 문서 전역의 σ_d 정의(L200–208·코드 `s`)가 배제함을 재확인 — F1r 자기표시와 합치.
- 관측 불변 확인: 본 판정의 모든 채택은 결과식 수치·peak·코드 동작을 바꾸지 않는다(F-1·F-4 = 프로즈 부호/방향 자구, B-2 = 방향 서술·주석, 나머지 = 한정어·표기).
