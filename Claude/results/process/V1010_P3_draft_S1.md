### 1.2 코드 엔트로피 항 정합 — dS_rxn vs dS_a 구분

**P1 audit 확정**: 코드는 엔트로피를 두 채널로만 다룬다.

- `dS_rxn` (L78, `func_U_j`): **반응(평형) 엔트로피**. 평형 중심 U_j(T) = (−ΔH_rxn + T·ΔS_rxn)/F에 기여. Ch2 §2.2-§2.4의 ΔS(x) 이론이 최종적으로 이 항을 통해 전위에 들어온다. 코드는 전이당 스칼라 상수값을 쓴다.
- `dS_a` (L100, `func_L_q`): **활성화(동역학) 엔트로피**. ΔG_a = ΔH_a − T·ΔS_a 로 동역학 꼬리 길이 L_q에만 기여. Ch2가 다루는 평형 발열과 **물리가 다른 양** — Ch2 §2.3.4에서 이 구분을 명시.
- **가역 발열 q_rev**: P1 audit §2-E 확정 — 코드에 `q_rev`, `dT/dt`, 열생성 항 일체 없음. P4 구현 예고.

## 2. 발열 이론 갈고닦기 (★ 중점)

### 2.1 가역열 q_rev의 두 유도 경로

Ch2 §2.6의 eq:qrev는 결과만 있다. 아래에서 두 경로를 식→식으로 전개한다.

#### 경로 I — Bernardi 에너지 수지 경로 (Ch2 §2.6 현행)

**전제**: 단일 활물질, 등온 셀, 방전 전류 I > 0 (방전 부호 규약: I > 0 → 방전).

열역학 1법칙 + 전기 일 + 엔트로피 생성 분리:

```
총 열방출률  dQ_total/dt = I·(U_oc − V)   ← Bernardi eq.(2) 단전극 한계
```

여기서 V = 단자 전압, U_oc = 평형 OCV. 이를 두 항으로 분리:

```
dQ_total/dt = I·(U_oc − V)
            = I·(U_oc − V_n)       ← 과전압 항 (비가역 소산)
            + I·(V_n − V)          ← 이 항은 IR drop 항, 방향에 따라 부호 갈림
```

보다 표준적인 Bernardi 분리:

1. **비가역 소산**: `dQ_irr/dt = I·(U_oc − V) ≥ 0` (2법칙, 항상 발열)
2. **가역(엔트로피) 발열**: Gibbs-Helmholtz → `ΔG = ΔH − TΔS` 에서 `ΔG = −FU_oc` 이므로

```
∂(ΔG)/∂T = −ΔS   (Gibbs-Helmholtz)
         = −F · ∂U_oc/∂T          ← U_oc = −ΔG/F 미분
```

따라서 `ΔS = F · ∂U_oc/∂T` (부호 규약: 삽입 기준 ΔS = +F · ∂U/∂T).

방전 전류 I에 의해 시스템이 단위 시간당 처리하는 전하량 I에 대응하는 엔트로피 흐름:

```
dQ_rev/dt = T · (엔트로피 흐름)
          = T · I · (∂U_oc/∂T) / F × F      ← 몰당 환산 복원
          = I · T · ∂U_oc/∂T
```

부호 주의: 방전(I > 0)에서 ΔS > 0이면 `dQ_rev/dt < 0` (흡열). 따라서:

```
╔══════════════════════════════════════════════════════╗
║  dQ_rev/dt = −I · T · ∂U_oc/∂T                     ║
║            = −(I·T/F) · ΔS(x)                       ║
║                                                       ║
║  부호: I > 0(방전), ΔS > 0 → dQ_rev < 0 (흡열)      ║
║        I > 0(방전), ΔS < 0 → dQ_rev > 0 (발열)      ║
╚══════════════════════════════════════════════════════╝
```

이것이 Ch2 eq:qrev 이다. 차원 검산: [A·K·(J/mol/K)/(C/mol)] = [A·J/C] = [W]. 무결.

#### 경로 II — 통계열역학 ΔS↔q_rev 연결 경로 (Ch2의 핵심 기여)

Ch2가 추가로 세우는 것: ΔS(x)의 **분포 구조** → q_rev의 x 의존 모양.

**단계 1**: 격자기체 분배함수 Z₁ = 1 + exp(−β(ε₀−μ)) → 점유 θ = ⟨n⟩ = 1/(1+exp(β(ε₀−μ))).

**단계 2**: μ를 전위로 환산. eq:muV: μ = μ⁰ − F(V − U_j) → θ(V,T) = 1/(1+exp(+(V−U_j)/w)), ξ = 1−θ.

**단계 3**: 1몰 자리의 S_config = −R[θ ln θ + (1−θ)ln(1−θ)] (eq:Sconfig).

**단계 4**: ξ에 대한 부분몰 미분. θ = 1−ξ 이므로:

```
∂S_config/∂θ = −R ln[θ/(1−θ)] = −R ln[(1−ξ)/ξ]
```

따라서 ξ 기준 부분몰:

```
∂S_config/∂ξ = +R ln[(1−ξ)/ξ]     ← Li 탈리튬화(ξ 증가)당 config 엔트로피 변화
```

**단계 5**: 전체 ΔS(x)는 세 분포의 합 (Ch2 §2.3 핵심 박스):

```
ΔS(x) = R ln[ξ/(1−ξ)]            (config 분포, 봉우리 내부 x-의존)
       + ΔS⁰_j                    (vib+elec 중심 흡수, 전이당 상수)
       + δS_vib/e(x)              (전이 폭 내 급변 시만 — 흑연에선 ≈0)
```

**단계 6**: 이를 eq:qrev에 대입:

```
dQ_rev/dt = −(I·T/F) · [R ln(ξ/(1−ξ)) + ΔS⁰_j + δS_vib/e]
```

겹침 구간에서는 eq:weighted의 dQ/dV 가중 블렌드를 쓴다:

```
∂U_oc/∂T = [Σ_j Q_j g_j(x) (ΔS⁰_j/F + (R/F)ln(ξ_j/(1−ξ_j)))] / [Σ_j Q_j g_j(x)]

→  dQ_rev/dt = −I·T · ∂U_oc/∂T    (위 완전식 대입)
```

이것이 "분포를 재배열하는 열"이라는 Ch2 §2.6의 물리적 해석을 식으로 닫는 경로다.

### 2.3 비가역(Joule·과전압) 항과의 분리

Ch2 §2.6이 이미 두 항을 분리하지만, 코드 관점에서 명시:

```
총 발열 = 비가역 소산항 + 가역항
        = I·(U_oc−V) + (−I·T·∂U_oc/∂T)
         ↑                 ↑
   항상 ≥ 0 (2법칙)    부호 ±(ΔS에 따라)
   ← 코드 Ch1의 분극·꼬리가 여기 기여 →   ← Ch2 발열, 코드 미구현 →
```

- **비가역 항**: Ch1 코드의 분극 V_n = V_app − σ_d|I|R_n(L408), 꼬리 L_V(L303-347)가 U_oc−V 기여. 항상 발열.
- **가역 항**: ∂U_oc/∂T × T × I. 코드 미구현(P4). 부호 ±.
- **히스 비가역 소산**: 한 사이클당 `∝ I · ΔU_hys` — Ch2 §2.4.4 언급. 코드에선 gap 크기(`func_dU_hys`)는 있으나 소산 정량식 없음.

#### 현행 Ch2(v5) 보강 필요 사항

**B2. ΔS(x) 완전식과 q_rev의 명시적 연결** (§2.6 후반):
- 현행: q_rev = −IT·∂U_oc/∂T로 닫고, ∂U_oc/∂T의 완전식이 §2.4.1 keybox에 따로 있음.
- 보강: q_rev 절 안에서 완전식을 직접 대입한 "계산 가능 형태" 1줄 제시 (§2.2 위 경로 II 최종식).
- 근거: 코드 구현자가 §2.4와 §2.6을 별도로 읽어야 하는 독자 부담.

**B3. 코드 dS_rxn 키와 Ch2 ΔS⁰_j의 명시적 연결** (§2.2, §2.6):
- 현행: ΔS⁰_j ≡ ΔS_rxn,j 라고 §2.2의 파생 B에서 정의하지만, 코드 키 `'dS_rxn'`과의 1:1 연결이 명문화 안 됨.
- 보강: §2.6 종합식 keybox에 "입력 ΔS⁰_j = 코드 키 `'dS_rxn'`" 한 줄 추가.
- 근거: P4 발열 구현 시 독자 혼란 방지.

**B4. dS_a(활성화 엔트로피)와 ΔS_rxn(반응 엔트로피) 구분 — §2.3.4 보강**:

- 근거: 두 ΔS가 같은 차원([J/mol/K])이라 코드 구현 시 혼동 실제로 발생 가능.

**B5. 비가역 소산 I²R·과전압 항의 Ch2 위치 명확화** (§2.6):
- 현행: "첫 항은 과전압 소산" 서술만.
- 보강: Ch1의 분극(Rn, L408)·꼬리(L_V, N7-N8)이 U_oc−V 기여자임을 명시 (1문장).

## 6. 핵심 정리

**Ch2↔코드 정합 상태**: 통계열역학 이론 구조(Z₁→logistic, BW, S_config, S_vib, Se)는 코드와 직접 대응하지 않지만 의도적 배경이다. 코드가 직접 구현하는 항은 `dS_rxn`(ΔS⁰_j 통로)·`dS_a`(활성화 동역학)·Ω(히스)·`func_dU_hys`이며, q_rev 자체는 P4 예정이다.

**발열 이론 갈고닦기 핵심**: q_rev = −IT·∂U_oc/∂T 유도를 Gibbs-Helmholtz 경로로 명문화하고, Ch2 §2.4의 ∂U_oc/∂T 완전식을 §2.6 q_rev에 직접 연결하는 것이 가장 중요한 보강이다. 차원·부호·극한은 현행에서 무결이다.

**v5 타당성**: v4의 w_eff 오류 제거는 옳다. 현행 Ch2(v5)의 분배함수→엔트로피→발열 사슬은 물리적으로 자기완결이며, 수치 검증(파생 A 175점)이 실증한다. 권고 R1-R6는 식 유도 밀도·코드 연결·명확화 보강이며 구조 변경이 아니다.
