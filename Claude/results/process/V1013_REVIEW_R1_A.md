- **무엇이**: eq:n0map 와 기호표가 $|I|$ [A] $=$ c\_rate$\cdot Q_\cell$, $Q_\cell$ [C] 로 선언. c\_rate 의 단위는 표에 미선언.

- **자기표시**: c\_rate 단위가 문건 표에 없어 "1/h 관례 + 코드 docstring" 을 근거로 한 판정 — c\_rate 를 s$^{-1}$ 로 재선언하면 식은 살지만 통용 관례와 어긋난다.
- **수정안**: 표에 c\_rate 행(단위 1/h) 추가 + eq:n0map 에 "$|I|=\text{c\_rate}\cdot Q_\cell/3600$ (Q_cell [C] 기준)" 또는 $Q_\cell$ 을 A·h 로 통일 선언 중 택일 — 코드와 동시 정정 필요.

### [MED] M-6. L526–532 (eq:gxi) — $g_j^0$ 이 ξ-선형항을 은닉: 식이 자구대로는 항등식이 아님 ★가장 약한 1곳
- **무엇이**: "선형·상수 몫만 $g_j^0$ 으로 재편된다: $g_j(\xi)=g_j^0+RT[\xi\ln\xi+(1-\xi)\ln(1-\xi)]+\Omega_j\,\xi(1-\xi)$".
- **근거(재유도)**: eq:sm-gtheta 의 $g(\theta)=\mu^0\theta+RT[\text{mix}]+\Omega_j\theta(1-\theta)$ 에 $\theta=1-\xi$ 대입 → $g=\mu^0(1-\xi)+RT[\text{mix}(\xi)]+\Omega_j\xi(1-\xi)$. 즉 $g_j^0$ 자리는 상수가 아니라 affine $\mu^0(1-\xi)$ 다. 위첨자 0 상수 표기 관례로 읽으면 등식이 거짓이고, prose("재편")를 affine 으로 읽어야만 성립 — 전제 미명시(렌즈 ②) + G-follow 걸림돌. 하류 안전성은 확인함: eq:sm-thresh 는 $g''$(affine 소멸), L533 의 1계 미분 논거는 우함수 조성부 $f$ 만 사용, fig:sm-gxi·fig:doublewell(L963, "shifted" 명기) 모두 조성부만 플롯 — 물리 결과 훼손은 없다.
- **수정안**: eq:gxi 를 $g_j(\xi)=g_j^{\mathrm{lin}}(\xi)+RT[\cdots]+\Omega_j\xi(1-\xi)$, $g_j^{\mathrm{lin}}(\xi)\equiv\mu^0(1-\xi)$ 로 명시하거나, 최소한 prose 를 "선형·상수 몫은 (ξ 의 affine 함수) $g_j^0(\xi)$ 로 재편되며 이후 미분 논거에는 기여하지 않는다" 로.
