## §2. ★ 발열 이론 갈고닦기 (중점) — 식→식 유도와 적대검산

> 이 절은 Ch2 §2.6(sec:revheat, L636-716)을 **유도 깊이 1급**으로 정련하는 본체다. 현 본문은 Bernardi 식을 srcbox(L650-654) 한 칸에 압축 — 타 전공(열·기계) 독자가 비약 없이 따라가도록 **식→식**으로 전개하고, 부호·차원·극한을 적대검산한다.

### 2-1. 출발 — 1법칙에서 전지 에너지 수지로 (Bernardi)

전지를 한 열역학계로 보면, 1법칙은 내부에너지 변화 = 들어온 전기일 + 계가 환경과 주고받은 열이다. 단자에 흐르는 전류 `I`(>0 방전), 단자 전압 `V`, 평형(개회로) 전위 `U_oc` 에 대해, 단위시간 발열률 `Q̇` 는 [정합: bernardi1985 추출본·Ch2 식 2.27]

```
(식 P3-1)   Q̇ = I·(U_oc − V)  −  I·T·(∂U_oc/∂T)
                └─ Q̇_irr ─┘     └─── Q̇_rev ───┘
```

유도 다리(식→식, 본문 srcbox 가 생략한 중간 단계):

**(a) 전기일과 화학퍼텐셜.** 계가 외부에 한 전기일률은 `Ẇ_elec = I·V`. 한편 반응이 방출하는 화학에너지률은 평형 전위로 `I·U_oc`(평형에서 한 몰 반응당 ΔG=−FU_oc 를 전류로 환산). 둘의 차가 비가역 소산이다:
```
(식 P3-2)   Q̇_irr = I·U_oc − I·V = I·(U_oc − V) ≥ 0.
```
부등호는 2법칙(과전압 η=U_oc−V 가 방전에서 양, 충전에서 부호반전하되 I 와 곱이 항상 양)이 강제 — Ch2 §2.6(L648) "2법칙상 항상 발열".

**(b) 가역(엔트로피) 항.** 가역 성분은 등온 반응의 흡·방열 `T·ΔS` 를 전류로 환산한 것이다. 반응 진행률 `dξ_rxn/dt = I/F`(Faraday, n=1) 이므로
```
(식 P3-3)   Q̇_rev = −T·(dS/dt) = −T·ΔS·(dξ_rxn/dt) = −(I·T/F)·ΔS.
```
음부호: 흡열(ΔS>0)이면 계가 열을 **흡수**하므로 발열률에 음으로 기여. 이제 ΔS 를 측정 가능한 전위 기울기로 바꾸는 다리가 핵심이다(2-2).

### 2-2. ★ 가역열의 두 등가형 — T·ΔS 경로와 ∂U/∂T 경로의 식→식 연결

이 소절이 발열 유도의 심장이다. `Q̇_rev` 를 **(경로 I) 통계열역학 ΔS** 와 **(경로 II) 측정 전위 기울기 ∂U/∂T** 두 길로 적고, 둘이 같음을 Gibbs–Helmholtz 로 못박는다.

**경로 II — Gibbs–Helmholtz 다리 (식→식).**
반응 Gibbs 에너지와 평형 전위는 `ΔG = −F·U_oc` (n=1). 양변을 정압 T 미분:
```
(식 P3-4)   ∂(ΔG)/∂T = −F·(∂U_oc/∂T).
```
열역학 정의 `ΔS = −∂(ΔG)/∂T` 를 (식 P3-4)에 대입:
```
(식 P3-5)   ΔS = −∂(ΔG)/∂T = +F·(∂U_oc/∂T).          ← 부호 +
```
이것이 ★**부호 확정 다리**다. (CH2_v3 survey 의 Decision-queue 가 표준법 논문 추출본의 `∂U/∂T=−ΔS/nF` 를 "추출 부호 오류 의심"으로 남겼는데 — bernardi 카드 #1·standardised 카드 — **여기서 식 P3-4·P3-5 가 +부호를 닫는다**: ΔG=−FU 의 음부호와 ΔS=−∂ΔG/∂T 의 음부호가 곱해져 +가 된다. 본 supplement 는 `ΔS=+F·∂U_oc/∂T` 를 확정으로 채택 — Newman·Reynier 2003·Ch2 srcbox L651-653 와 일치 [정합·확정].)

**두 경로의 합류.** (식 P3-5)를 (식 P3-3)에 넣으면
```
(식 P3-6)   Q̇_rev = −(I·T/F)·ΔS = −(I·T/F)·(F·∂U_oc/∂T) = −I·T·(∂U_oc/∂T).
```
곧 **T·ΔS 경로(통계열역학)** 와 **∂U/∂T 경로(측정)** 가 정확히 같은 `Q̇_rev` 를 준다 — 이것이 Ch2 가 "분포(ΔS)" 와 "측정(∂U/∂T)" 를 한 식으로 잇는 결속점이다. Ch2 식 2.27(L645) 의 박스 `Q̇_rev=−I·T·∂U_oc/∂T=−(IT/F)ΔS(x)` 가 바로 (식 P3-6).

### 2-3. ★ 통계열역학 ΔS ↔ q_rev 연결 — 분포를 재배열하는 열

(식 P3-6)의 `ΔS(x)` 는 Ch2 가 세운 **세 분포의 합**이다(§2.2 핵심박스 L414-418):
```
(식 P3-7)   ΔS(x) = R·ln[ξ/(1−ξ)]      (config 분포; 봉우리 내부 x-의존)
                  + ΔS⁰_j                (vib+elec 중심 흡수; 전이 상수)
                  + δS_{vib/e}(x)        (전이 폭 내 급변 시만; 흑연=0, LCO MIT=유지)
```
이를 (식 P3-6)에 대입하면 가역 발열이 **분포 재배열 열**로 분해된다:
```
(식 P3-8)   Q̇_rev = −(I·T/F)·{ R·ln[ξ/(1−ξ)] + ΔS⁰_j + δS_{vib/e}(x) }.
```
물리 읽기(식→의미, 비약 0):
- **config 몫** `−(IT/F)·R·ln[ξ/(1−ξ)]`: Li 가 격자 자리 사이로 재배열되며 주고받는 열. 희박(ξ→1)서 ln→+∞ 이라 흡열 발산, 만충(ξ→0)서 ln→−∞ 이라 발열 발산 — 봉우리 내부의 비선형 q_rev 모양이 **점유 분포에서 직접** 온다.
- **중심 몫** `−(IT/F)·ΔS⁰_j`: 전이 중심의 표준 반응열. 흑연 4→3 의 `ΔS⁰=+29`(흡열)·2→1 의 `−16`(발열).
- **잔차 몫** `δS_{vib/e}`: 흑연은 0(중심에 흡수), LCO MIT 만 전자분포 급변으로 살아남(Ch1 sec:lco-electronic 의 `ΔS_e<0` 골).

곧 가역 발열 = "Li 배열 분포(config) + 포논 분포(vib) + 전자 분포(elec)를 한 SOC 에서 다음 SOC 로 재배열하며 주고받는 열". Ch2 §2.6 L656-666 이 산문으로 말한 것을 (식 P3-8)이 **식으로** 닫는다.

### 2-4. 비가역 발열 — I²R 과 과전압 η 의 분리 (혼동 금지)

Ch2 §2.6 식 2.27 의 1항 `Q̇_irr=I(U_oc−V)≥0` 을 코드·Ch1 동역학과 잇는다. 과전압 `η ≡ U_oc − V` 를 세 몫으로 분해(Ch1 분극 N1·sec:pol 정합):
```
(식 P3-9)   η = I·R_n            (ohmic, IR; 코드 V_n=V_app−σ_d|I|R_n, L408)
                + η_ct            (전하전달, Butler–Volmer; η_ct≈(RT/F)·|I|/i₀ 소전류극한)
                + η_diff          (확산/완화; Ch1 꼬리 L_q·_causal_lowpass)
```
따라서
```
(식 P3-10)  Q̇_irr = I·η = I²·R_n + I·η_ct + I·η_diff ≥ 0.
```
- **I²R 항** `I²·R_n`: 순 Joule 열, 항상 양, T-미분과 무관(가역열에 안 섞임).
- **η_ct·η_diff 항**: 동역학 소산. Ch1 의 활성화 엔트로피 `ΔS_a`(코드 L100 `dS_a`)는 **이 비가역 동역학**에 들어가지 가역열에 안 들어간다 — Ch2 §2.3(L423-427) "활성화 엔트로피는 비가역 동역학을 지배하고 가역 발열에 들어가지 않는다"의 식적 근거.

★**핵심 분리 명제**(Ch2 가 강조하나 식으로 덜 박은 것): `∂U/∂T` 미분은 **가역열만** 뽑고(식 P3-6), η 의 소산은 **비가역열**(식 P3-10)이다. 둘을 한 `∂U/∂T` 로 뭉뚱그리면 안 된다 — Ch2 파생 D(히스 L586-593)가 히스 gap 을 비가역으로 분리한 것과 **같은 분리 원리**.

### 2-5. ★ 적대검산 — 부호·차원·극한 (11행)

| # | 검산 항목 | 식·근거 | 검산 | 판정 |
|---|---|---|---|---|

### 2-6. 발열 절 정련 권고 (Ch2 §2.6 본문 반영 후보)
