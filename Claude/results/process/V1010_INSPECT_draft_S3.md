### [CRIT-S3-01] 두-상 전이에 w=nRT/F 적용 — 이중지위 선언만 있고 정합 조건 도달 경로 누락

**위치**: Ch1 `graphite_ica_ch1_v1.0.10.tex` §sec:width (N4), 식 eq:wbase `w_j = n_j RT/F`, §sec:broadening (N6 보강), §sec:sum Tab.staging  
**무엇이 물리적으로 틀렸나**:  
문건은 두-상 전이의 평형 단일 입자 dQ/dV가 "near-delta"임을 인정하면서, 동시에 logistic 평형 점유 ξ_eq의 폭도 `w=nRT/F`로 계산한다(eq:wbase). 그러나 §sec:broadening에서 "두-상 봉우리의 실측 폭은 현상학적 자유 피팅 폭이며, broadening 세 출처가 만든다"고 선언할 뿐이다.  

핵심 인과 연결이 끊긴 지점: `w_j = n_j RT/F`이라는 식이 N4에서 **단상 equilibrium prediction으로 유도됨** → N5에서 `ξ_eq = logistic[σ_d(V-U^d)/w_j]`에 그대로 삽입됨 → N6에서 `dQ/dV = Q_j ξ_eq(1-ξ_eq)/w_j`의 분모로도 쓰임. 즉 두-상에서 w가 "현상학적 피팅 폭"이라면, N4에서 w를 nRT/F로 **물리적으로 유도하는 과정이 두-상에는 적용되지 않아야 함**에도 같은 식이 N4→N5→N6으로 흐른다.  

**왜 문제인가**:  
- G-follow 렌즈: 독자가 N4의 `w = nRT/F` 유도를 따라가면 단상 격자기체 분포에서 나온 식으로 이해함 → N6에서 두-상에도 같은 식을 쓴다는 것을 보며 혼란 발생. 문건은 "이중지위"를 선언하나 **N4 유도에서 두-상 예외를 명시적으로 차단하는 절이 없음**.  
- Ch2 §ssec:weff(파생 C)에서 같은 내용을 상세히 다루지만, Ch1 N4 절 자체에는 "두-상에서 w는 검증 가능 예측이 아님"이라는 국소 경고가 없다.  
- 실질적 오해 위험: 독자가 초기값 표(Tab.staging)의 폭 폴백 0.012~0.020 V를 `w=nRT/F`(≈25.7 mV@298K)와 대조하면 일관성 없는 수치를 발견함. n=1이면 폴백 w는 이미 nRT/F와 다른 값인데, 그 이유가 N4 절에서 설명되지 않음.

**근거**: N4 본문 "폭의 이중지위 — 단상의 평형 예측, 두-상의 현상학적 폭"에서 개념 도입은 있으나, 이 절이 N4에 있고 N4 자체는 `w=nRT/F`를 유도하는 절이어서 구조적으로 단-한-식이 두 역할을 하는 이유를 인과 없이 선언만 함.  
**렌즈**: 물리적 논리 연결(N4→N5→N6 흐름에서 두-상 예외 인과 누락)·G-follow

### [HIGH-S3-02] 인과 꼬리 peak_shape = (ξ_eq − ξ_lag)/L_V 의 차원 정합 경로 비약

**위치**: Ch1 §sec:tail (N8), 식 eq:peakshape  
**무엇이 물리적으로 틀렸나**:  
식 eq:peakshape는 `peak_shape = (ξ_eq − ξ_lag)/L_V`를 정의하며, 이것이 `dQ/dV`에 `Q_j`를 곱해 합산된다(eq:sum). 즉 peak_shape의 단위는 1/V여야 한다.

- `ξ_eq`, `ξ_lag`는 무차원(점유율 0~1)  
- `L_{V,j}`는 전압 단위 [V]  
- 따라서 `(ξ_eq − ξ_lag)/L_V`의 단위 = [1/V] ✓ — 차원은 맞음

그러나 문건의 유도 경로에서 이 peak_shape가 `dξ_eq/dV`로 수렴한다는 서술(eq:peakshape 바로 뒤 본문: "이 차분이 매끈한 미분 dξ_eq/dV로 수렴하는 극한은 L_V≫Δ_grid 쪽")과, 평형 분기(eq:branch)의 `ξ_eq(1-ξ_eq)/w`가 같은 의미라는 연결이 명시적 동등성 증명 없이 기술됨.

**구체적 비약**: eq:lowpass의 점화식 `ξ_{lag,i} = ρ ξ_{lag,i-1} + (1-ρ) ξ_{eq,i}`에서 `(ξ_eq − ξ_lag)/L_V`가 왜 `dξ_eq/dV`에 가까운지는 "ρ→1이면 커널이 미분 커널이 된다"는 정성적 서술만 있음. ρ→1일 때 `(ξ_eq,i − ξ_lag,i)/L_V ≈ dξ_eq/dV`임을 Taylor 전개 또는 convolution 커널의 극한으로 수학적으로 닫는 단계가 없음.

**왜 문제인가**:  
G-follow 렌즈로 읽는 독자가 "꼬리가 평형 종에 어떻게 합류하는지"를 추적할 때, 식의 극한이 수식으로 닫히지 않아 신뢰 불가.  
**근거**: eq:peakshape 이후 본문 및 eq:branch.  
**렌즈**: 논리 점핑(중간 수학 단계 비약)·G-follow

### [HIGH-S3-03] 히스테리시스 gap 유도에서 Maxwell 작도와 spinodal 결과의 인과 연결 누락

**위치**: Ch1 §sec:hys (N3), 식 eq:spinodal·eq:dUhys  
**무엇이 물리적으로 틀렸나**:  
N3는 정규용액 모형의 spinodal 조건 `ξ_s± = ½(1±u), u=√(1-2RT/Ω)`에서 히스 gap `ΔU_hys = (2/F)[Ω·u - 2RT·artanh(u)]`를 제시한다. 이 식의 출발점은 Maxwell 작도(등면적 원칙)이나, 유도 연결이 다음과 같이 비약된다:

1. "spinodal 극값 ξ_s± 를 V(ξ)에 대입해 두 극값 전위의 차이가 ΔU_hys"라고 해야 하는데, 문건은 "Maxwell 작도와 Dreyer 순차 전환이 열역학 극한을 정한다"만 언급하고 eq:dUhys가 어디서 오는지를 닫는 대수 단계가 없음.  
2. eq:dUhys의 artanh(u) 항은 `∫V(ξ)dξ` 형태의 등면적 조건에서 나오는 것인데, 이 적분 과정이 문건에 없음.

**왜 문제인가**:  
독자가 eq:dUhys를 독립 검증하려면 Maxwell 작도 적분을 스스로 해야 하나, 문건은 결과만 제시. "결과를 받는다"는 접근은 Ch1이 "단계별로 닫힌다"는 설계 원칙(브리프 §7 a-b-c-d 출발-연산-중간식-박스 구조)과 충돌.  
**근거**: Ch1 N3 §sec:hys, eq:dUhys 전후. 비교: v10 동일 위치도 같은 비약 존재 → cross-version에서 해결 안 됨.  
**렌즈**: 논리 점핑(Maxwell 등면적 적분 단계 누락)

### [HIGH-S3-04] 충전 격자 역전 방향 서술 — 꼬리 방향이 물리와 반대로 기술된 가능성

**위치**: Ch1 §sec:tail (N8), 식 eq:reversal, 그림 fig:reversal 캡션  
**무엇이 물리적으로 틀렸나**:  
충전 캡션(b)은 "꼬리가 낮은 V 쪽으로 — 방전의 거울"이라 기술하고, 코드 본문은 "충전에서는 V가 큰 쪽으로 늘어진다"(충전 분기 본문 §sec:tail)고 반대로 적고 있다.

구체적:
- N8 §★충전 격자 역전 마지막 문장: "peak_shape = (ξ_eq−ξ_lag)/L_V 는 양수를 유지한다(꼬리가 진행 방향 뒤쪽, **곧 충전에서는 V가 큰 쪽으로 늘어진다**)."  
- fig:reversal 캡션(b): "충전은 진행이 V↓라 …꼬리가 **낮은 V 쪽으로** — 방전의 거울."

충전의 진행 방향은 V 감소이므로 꼬리(지연 방향의 과거)는 진행 방향의 반대, 즉 **높은 V 쪽** 으로 늘어져야 한다. 방전은 V↑ 진행이므로 꼬리가 높은 V 쪽, 충전은 V↓ 진행이므로 꼬리가 낮은 V 쪽 — 이것이 캡션(b)의 "낮은 V 쪽"이다.

그런데 N8 본문은 "충전에서는 V가 큰 쪽으로 늘어진다"고 쓰고 있어 캡션과 모순.

**왜 문제인가**:  
부호 사슬 S7(충전 거울) 검산에서 "peak\_shape 양수 유지"를 확인하지만, 구체적 꼬리 방향(V 크거나 작거나)에 대한 본문-캡션 불일치는 독자 혼란 유발. 충전 ICA 물리 해석(C-rate 증가 시 peak 이동 방향)과 직결.  
**근거**: Ch1 §sec:tail N8 본문 vs fig:reversal 캡션(b). v10 동일 절 동일 위치 확인.  
**렌즈**: 왜곡(본문 vs 캡션 불일치)·부호

### [HIGH-S3-05] LCO 전자항 _effective_dS_rxn의 T-동결 근사 — 물리 전제 미명시

**위치**: `Anode_Fit_v1.0.10.py` L657-673 `LCOCathodeDQDV._effective_dS_rxn`, Ch1 §sec:lco-decomp  
**무엇이 물리적으로 틀렸나**:  
`ΔS_e(x,T)`은 Ch1 eq:dSegate에서 `∝ T`임을 명시한다(Sommerfeld 항). 그러나 코드 `LCOCathodeDQDV._effective_dS_rxn`은 이 항을 **T_ref=298.15 K에서 동결**한 상수 오프셋으로 더한다:

```python
dS = dS + func_dSe_molar(tr['x_center'], T_ref, ...)  # T_ref 고정
```

그 결과 U_j(T) = (-ΔH + T·dS_eff)/F에서 dS_eff가 T-무관 상수가 되어 ∂U_j/∂T = dS_eff/F도 T-무관해진다. 그러나 물리적으로 ΔS_e ∝ T이므로 ∂U_j/∂T의 전자 기여는 온도 의존이 있어야 한다.

코드 주석(L663-666)에 "단일-기준 근사"라고 라벨링했지만, Ch1 문건은 이 근사 조건이나 오차 범위를 서술하지 않는다.

**왜 문제인가**:  
G-usable 렌즈: 다온도 피팅(T≠298 K) 시 전자항의 T-의존이 소거되어 MIT 영역에서 ΔS_e의 T-스케일 정보를 잃음. 코드-문건 사슬에서 "ΔS_e ∝ T (Ch1 eq:dSegate)"가 코드에서 "∝ T_ref(상수)"로 꺾이는 지점이 Ch1 본문에 명시되지 않아 사용자가 ch1을 보고 코드를 짤 때 오류 가능성.  
**근거**: Ch1 eq:dSegate·eq:dSemolar 대 code L663-673.  
**렌즈**: 물리적 논리 연결(ΔS_e∝T → 코드에서 상수 처리, 인과 단절)·G-usable

### [MED-S3-06] Ch2 파생 A 완전식 — 봉우리 내부 config 항과 dqdv 경로의 이중계산 경계 서술 모호

**위치**: Ch2 §ssec:overlap, Ch2 §sec:revheat 종합식, 코드 L544-578 `entropy_coefficient`  
**무엇이 물리적으로 틀렸나**:  
Ch2 §sec:revheat 종합식 keybox는 `∂U_oc/∂T`에 config 항 `(R/F)ln[ξ_j/(1-ξ_j)]`을 **명시적으로** 더한다. 코드 `entropy_coefficient`도 L573-575에서 이를 계산한다.

그런데 코드 L550-552 주석: "★dqdv 곡선은 이 config 항을 넣지 않는다(폭 w가 이미 담음, Ch2 파생 B) — q_rev 경로만 명시 가산한다. 두 경로는 같은 물리의 다른 산출(직교; 이중계산 아님)."

**왜 문제인가**:  
G-usable 렌즈: 사용자가 dqdv와 reversible_heat를 동시에 쓸 때 이중계산 위험 여부를 확인하기 어려움.  
**근거**: Ch2 §ssec:overlap 파생 B, code L550-578.  
**렌즈**: 물리적 논리 연결(두 경로의 이중계산 미발생 근거 누락)·G-usable

### [MED-S3-07] detailed balance → logistic 유도에서 "전기화학 퍼텐셜 μ 정의" 삽입 점프

**위치**: Ch1 §sec:width (N4-N5), 식 eq:db·eq:xieq  
**무엇이 물리적으로 틀렸나**:  
N4 유도는 `r+/r- = exp(A/RT)`(detailed balance)에서 `A = σ_d F(V-U)/(nRT)`를 넣어 `ξ_eq = logistic[σ_d(V-U)/w]`를 도출한다. 이 과정에서 **affinity A의 물리적 정의** — "전자 이동 반응의 전기화학 퍼텐셜 차이가 A = σ_d F(V-U)"임 — 에 대한 출발 근거가 선언 없이 삽입된다.

구체적으로:
- eq:db에서 A를 "반응 친화도"로 소개하지만, 이 A가 왜 `σ_d F(V_n - U_j)`인지의 연결이 없음.  
- Ch2 §ssec:logistic에서 μ = μ^0 - σ_d F(V-U_j) 형태로 전기화학 퍼텐셜을 정의해 ξ_eq를 유도하는데, 이것이 Ch1 N4의 detailed balance 접근과 동등함은 어디에도 명시되지 않음.

**왜 문제인가**:  
독자 입장에서 Ch1은 동역학(rate 비율)으로, Ch2는 평형 통계역학(분배함수)으로 같은 logistic을 도출하는 두 경로가 있는데, 이 둘이 동등함을 보이는 연결이 없어 redundancy처럼 보이거나 서로 다른 모델처럼 오해 가능.  
**근거**: Ch1 N4 eq:db, Ch2 §ssec:logistic eq:logistic.  
**렌즈**: 논리 점핑(동역학 affinity A = 통계역학 전기화학 퍼텐셜 차이의 동등성 미연결)

### [MED-S3-08] 컷 affinity A의 z_cut=4.357 선택 근거 수치 검산 가능성

**위치**: Ch1 §sec:lag (N7), 식 eq:Acut, 코드 L350-352  
**무엇이 물리적으로 틀렸나**:  
문건은 `z_cut=4.357`이 "logistic 미분 종의 5% 폭이 중심에서 ±z_cut RT/F 규모"라고 서술한다. 그런데 logistic 함수의 미분 `f'(z) = e^z/(1+e^z)^2`가 최댓값의 5%가 되는 z값을 실제로 계산하면:

`f'(0) = 1/4`(최대, n=1, w=1 스케일)  
`f'(z_cut) = 0.05 × f'(0) = 0.0125` 조건 →  
`exp(-z)/(1+exp(-z))^2 = 0.0125` → z ≈ 4.394 (수치 계산)

그러나 문건은 4.357을 제시. 5% 기준이 "미분"이 아니라 "점유 ξ_eq 자체의 5%"라면 다른 z값이 나옴:  
`ξ_eq(z_cut) = 0.05` → `1/(1+exp(-z_cut)) = 0.05` → `z_cut = ln(19) ≈ 2.944`

또한 문건은 "점유 ξ_eq 자체가 아니라 미분 기준"임을 강조하나, 4.357이 실제 logistic 미분의 5% 컷인지 독립 계산값과 0.037 차이가 나는 이유가 불명. (n_j=1 기준이 아닌 다른 기준을 쓰거나 일방향 기준일 가능성)

**왜 문제인가**:  
코드·문건을 독립 재현하려는 사용자가 z_cut=4.357을 도출할 수 없음. 수치 일관성 의심.  
**근거**: Ch1 N7 eq:Acut, 상기 독립 계산.

### [MED-S3-09] Ch2 가역 발열 부호 규약 — 발열의 방향 판단에 혼란 가능

**위치**: Ch2 §sec:revheat, 식 eq:qrev, 부호 규약 srcbox  
**무엇이 물리적으로 틀렸나**:  
Ch2 eq:qrev: `Q̇_rev = -I·T·∂U_oc/∂T = -(I·T/F)·ΔS(x)`

서술: "방전(I>0)에서 ΔS>0이면 Q̇_rev<0으로 흡열(endothermic, 셀이 열을 흡수)"

그런데 배터리 열 연구의 관례에서 I>0이 방전을 의미하고 Q̇>0이 셀에서 **나가는** 열(발열, exothermic)임을 가정하면, `Q̇_rev = -I·T·∂U_oc/∂T`에서 ΔS>0이면 `Q̇_rev<0`은 "셀에서 나가는 열이 음수 = 흡수"로 읽힘. 이 방향은 Bernardi 원문 부호규약과 일치하나 문건은 "Bernardi 일반 에너지 수지"를 인용하며 I·(U_oc-V)≥0을 비가역 발열로 사용.

문제는 `I>0` 규약이 문건 안에서 두 가지로 쓰임:
- N0에서 `σ_d = +1`이 방전(탈리튬화)
- Ch2에서 `I>0`이 방전

이 규약이 동일한지, 그리고 코드에서 `reversible_heat(I=1.0)` 기본값이 방전인지 아닌지가 Ch2 본문에서 확인 불가.

**왜 문제인가**:  
G-usable: 사용자가 `reversible_heat(V_n, T, I=1.0)`를 호출할 때 양수/음수 해석이 모호함.  
**근거**: Ch2 eq:qrev 부호 srcbox, code L580-588.  
**렌즈**: 왜곡(규약 불명확)·G-usable

### [MED-S3-10] N8 branch 스위치의 불연속 정당화 — "실용적 선택" 선언만으로 물리 완결성 부족

**위치**: Ch1 §sec:tail (N8), 식 eq:branch  
**무엇이 물리적으로 틀렸나**:

**왜 문제인가**:

**근거**: Ch1 N8 eq:branch 이후 본문.

## 2. 정독 후 무결 판정 영역

(빈 통과 금지 — 문제를 발견하지 못한 영역도 정독 근거 명시)

- **부호 사슬 전수 검산(§sec:signcheck, S1-S8)**: 정독. S1~S8 각 항의 체크마크가 근거와 함께 제시되어 있으며, 특히 S6의 "∂lnL_q/∂V < 0은 동기(motivation)이며 운영상 실현 미분은 0"이라는 자기모순 해소 논증은 물리적으로 정합하다. **무결**.  
- **분극 V_n = V_app - σ_d|I|R_n (N1, eq:vn)**: 방전에서 측정전압 > 내부전압, 충전 반대. 단방향 부호 정합 확인. **무결**.  
- **detailed balance r+/r- = exp(A/RT) (N4, eq:db)**: Bazant(2013) 비평형 열역학에서 표준 결과. 유도 방향 적합. **무결**.  
- **지수 기억 ODE 일반해 (N8, eq:memory)**: 적분인자 exp(q/L_q) 방법 적용, 일반해 정합. 이산 저역통과(eq:lowpass) 도출 논리 흐름 확인. **무결**.  
- **합산식 (N9, eq:sum)**: 보존식 Q_cell q = Q_bg + ΣQ_j ξ_j의 직접 미분임. 가정이 아닌 항등식. **무결**.  
- **Ch2 격자기체 분배함수 → 점유 분포 → logistic (§sec:partition)**: Z_1 → ⟨n⟩ → ξ_eq 유도 전 단계 수학 정합. Fermi-Dirac 동형 설명 적절. **무결**.  
- **Ch2 configurational 엔트로피 S_config = -R[θlnθ + (1-θ)ln(1-θ)] (§sec:config)**: Boltzmann-Shannon 정보 엔트로피 직접 계산 일치. 부분몰 ∂S/∂θ = -R·ln[θ/(1-θ)] 유도 정합. **무결**.  
- **Ch2 Sommerfeld 적분 → S_e = (π²/3)k_B²T·g(E_F) (§ssec:elec)**: C_e/T가 T'-무관 상수임을 이용한 적분 ∫(C_e/T')dT' 단계 정합. **무결**.  
- **Ch2 Bragg-Williams 평형 전위 (§ssec:BW)**: ∂g(θ)/∂θ = μ/N_A 조건에서 V_eq 유도 정합. 상분리 임계 Ω=2RT 도출 정합. **무결**.  
- **Ch2 히스 분기 평균 → 가역열 (§ssec:hys, eq:hys_rev)**: 한 사이클 평균에서 히스 중심 상쇄 논증 물리 정합. **무결**.  
- **코드 func_ksi_eq 수치 안정성**: `np.where(z>=0, 1/(1+exp(-z)), exp(z)/(1+exp(z)))` — 양/음 z에 대한 overflow 방지 분기 적절. **무결**.  
- **코드 _causal_lowpass lfilter 계수 [1-ρ]/[1,-ρ]**: 점화식 `y_i = ρ·y_{i-1} + (1-ρ)·x_i`와 일치. lfilter 분자/분모 배치 정합. **무결**.  
- **코드 entropy_coefficient config 항 클리핑 eps=1e-12**: ξ→0 또는 →1에서 log 발산 방지. 물리적 발산 방향 보존하며 수치 안전. **무결**.
