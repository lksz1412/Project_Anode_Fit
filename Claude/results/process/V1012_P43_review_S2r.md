### A4. §4 decomp — 원문 자체 결함(F-1) 독립 재확인, map의 처리는 적절
map은 이 결함(부호 반전)을 §8 F-1로 이미 식별·격리했다. 본인이 **독립적으로 재유도**한 결과도 동일 결론에 도달했다:
$S_\mathrm{config}=-R[\theta\ln\theta+(1-\theta)\ln(1-\theta)]$, $\partial S/\partial\theta=-R\ln[\theta/(1-\theta)]$, $\theta=1-\xi$ 대입 ⇒ $-R\ln[(1-\xi)/\xi]=+R\ln[\xi/(1-\xi)]$. 극한 확인: $\xi\to1$(Li-희박)에서 $+\infty$ 이어야 물리적으로 옳다(빈자리 폭증 → 엔트로피 발산) — 정정형만 이 극한을 만족, 원문형(부호 반전)은 $-\infty$로 향해 물리 위반. ch1.tex L1728 실측 = "$R\ln[(1-\xi)/\xi]$"(반전형) 확인, 동일 파일 내 `(1-ξ)/ξ` 패턴은 이 1곳뿐(grep 1건). ch2.tex 3곳(`eq:dVdT_config` L242–243, revheat keybox L678, 파생 A L482) 전부 정부호 사용 — 결함이 Ch1 L1728 국소 1곳임을 재확인. map의 처리(§4(c) `eq:lco-configsplit`을 정정형으로 작성 + F-1 채택과 원자적 결합 명시)는 물리적으로 올바르고 편집상 안전하다. **추가 의견 없음 — map의 판단 지지.**

### A5. §5 code — 수식은 PASS, F-2와의 연결고리 미기재 (신규 보완)

**재유도/실측 근거**: `Anode_Fit_v1.0.12.py` L644–656 `LCOCathodeDQDV` 클래스 docstring 직접 확인 — "흑연 음극 모델을 그대로 상속한다... σ_d 를 뒤집지 않는다" 명문. 이는 F-2가 제기하는 "충전=탈리튬화이므로 셀 라벨 그대로 σ_d 를 먹이면 분기·꼬리가 반대로 걸린다"는 우려에 대해 코드 저자가 이미 내린 (반대 방향의) 판단이다. 두 판단이 상충하는지, 아니면 서로 다른 하위 메커니즘(엔트로피 기울기 부호 vs 히스 분기/꼬리 방향)을 가리켜 실제로는 상충하지 않는지는 본 검수에서 확정하지 못했다(추정 금지 — 근거 미발견으로 분류).

**수정안**: §8 F-2 "P4 코드 함의" 절에 `LCOCathodeDQDV`(L644–656) docstring 인용을 추가하고, 10차 판정 시 이 기존 코드 결정과 F-2 제안(σ_d 를 탈리튬화 여부로 재정의)이 정말 상충하는지 — 상충한다면 코드도 함께 수정 대상에 넣을지 — 를 명시적 안건으로 상정 권고. (단, 현재 §2/§3의 히스/꼬리 관련 LCO 코드 경로는 A2에서 확인했듯 Ω=0 폴백으로 비활성 상태이므로, 이 상충이 지금 당장 실행 결과에 영향을 주지는 않는다 — 실무 긴급도는 낮으나 문서 정합성 문제로는 남는다.)

### C2. [강한 후보] §5 code의 round-trip 승격 gate가 map엔 없고 S2엔 있다

S2 §5 (draft, "round-trip 가드")는 config 표준값(Motohashi 0.47/1.49 J/(mol·K))을 초기값→신뢰값으로 승격시키는 **확인 가능한 조건**을 제시한다: 대략 "$|U_1^\mathrm{cat}(T_0{=}298.15\mathrm K)_\mathrm{model}-3.90\,\mathrm V|<\varepsilon_U$ 이고 $\mathrm{sign}[\partial U_1^\mathrm{cat}/\partial T]_\mathrm{model}=\mathrm{sign}[\Delta S_{\rxn,1}^\mathrm{cat}(T_0)]_\mathrm{fit}$이면 승격"이라는 형태다. map §5는 "다온도 자기일관은 round-trip 위임"이라고만 적어 정성적 선언에 그친다 — 이 프로젝트 자체의 "Gate 설계 원칙"(정성 판단이 아니라 확인 가능한 조건만 gate로 인정)에 비춰 볼 때 S2의 구체 gate 조건이 더 그 원칙에 부합하며 병기 가치가 있다. **수정안**: §5 또는 §8 F-1 인접에 이 gate 조건을 병기 후보로 추가 권고(신규 물리 도입 아님 — 이미 있는 round-trip 서술의 구체화).

### C3. [사실관계 정정] §5의 "자기참조(고정점) 처리 = O2 판단 계승" 귀속 오류

map §5 서두는 "자기참조(고정점) 처리 = O2 판단 계승"이라 명시하나, O1·O2·O3 세 파일을 대조한 결과 "자기참조"·"고정점" 논의는 **O1**에만 있다(O1 §5 논리감사: "$\xi_{\eq,1}(V)$ 는 $U_1^{d,\mathrm{cat}}$ 에 의존하고 $U_1$ 은 다시 $\Delta S_{e,1}(x(\xi_{\eq,1}))$ 에 의존 → 암시적 자기참조 … self-consistency 는 피팅 단계로 위임"). O2 원문 어디에도 이 어휘가 없다. 내용 자체는 map 최종본에 이미 올바르게 반영돼 있어 **물리적 손실은 없다** — 순수 출처 표기(attribution) 오류. **수정안**: "O2 판단 계승" → "O1 판단 계승"으로 정정.

### C4. [애매/병기 후보] 그 외 소소한 발견 (물리 상충 없음, 병기 가치만)
- S1 §3: T2(≈0.55)·T3(≈0.48) 조성창이 서로 겹치지 않는다는 구체적 수치 근거(표 `tab:lco-staging` 대조로 확인됨)를 대며 "합산 시 전이간 교차항 없음"을 지지 — map 최종 §3엔 이 구체 수치가 없음(정성적으로만 성립 주장). 필수는 아니나 보강 가치.
- O1: "MSMR 함수형 동형 ≠ 물리량 동일(부호 인자가 진행 방향을 흡수)" caveat — map §5(c)의 "항등식의 유일해" 서술이 자칫 과확신으로 읽힐 여지를 줄이는 병기 후보(§A5/F-2 논의와 연결됨).
- S3 §6: bottom-up(`eq:Z1` 논거)의 더 상세한 버전(`eq:BW` L183-186·`eq:Veq_BW` L191 구체 지목) — map은 이 통찰을 S2 것으로 credit하는데 S3가 더 완전한 버전을 갖고 있음. 결론 차이 없음(심화판).
- O3 원안에 대한 map의 "완화 채택"(§10)이 다소 박함 — O3 원문은 "함수형=분배함수 결과(전 전이)"라고 쓴 직후 스스로 "값(진폭)의 지위는 전이 종류로 갈린다"고 층위를 재분리해 두고 있어, map이 지적하는 "두-상 과대주장" 정도가 O3 원문 그대로 읽었을 때만큼 심각하지 않을 수 있음(경미, 채택 여부에 영향 없음).
- C1_s1·C2_s1: 둘 다 "방전(σ_d=+1)은 분기 중심을 위로, 충전(σ_d=-1)은 아래로"라는 문장을 헤지 없이 그대로 §2(d)에 남겨(§8 F-2가 지적하는 LCO 방향 함정을 그대로 안고 있음) — map 최종 §2(d)는 이 문장 자체를 넣지 않아(원문에도 방향 문장 없음) 이 함정을 회피했다. **map의 편집 선택 자체는 정확했다** — 다만 §10의 "9개 사슬은... 물리 상충 0" 총평이 C1_s1·C2_s1의 이 특정 문장에는 해당하지 않는다는 점에서 C1과 함께 재확인.
