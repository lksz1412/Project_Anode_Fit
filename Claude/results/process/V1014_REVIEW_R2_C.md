## 3. ch1 R1 정정분 수식 2건 재검증

**(1) ΔU_hys Taylor 괄호(L1105–1110)** — **정확**. Ω=2RT/(1−u²) 동시 전개: Ωu=2RT·u/(1−u²)=2RT(u+u³+⋯) 과 artanh u=u+u³/3+⋯ 을 함께 넣으면 (2/F)·2RT[(u+u³)−(u+u³/3)]=**+8RT/3F·u³**. 수치 검증: exact/점근 비율 u=0.1→1.0121, u=0.01→1.0001 (수렴 확인). 소박 대입(Ω≈2RT 상수) 함정: (2/F)·2RT[u−artanh u]=**−4RT/3F·u³** — 수치 비율 u=0.01→1.0001 로 함정 서술도 정량 정확. u³∝(T_c−T)^{3/2} 도 u²=1−T/T_c 에서 자명.

**(2) 영점 규약 문장(L514–518, 보조 L479–482)** — **정확**. [2sinh(βħω/2)]⁻¹=e^{−βħω/2}·[1−e^{−βħω}]⁻¹ 항등 확인 — 두 규약의 q 는 정확히 인자 e^{−βħω/2} 차이. 그 인자의 T ln q 기여 = T·(−ħω/2k_BT)=**−ħω/(2k_B)** (온도 무관 상수, 차원 [K] 정합) ⇒ `eq:sm-sint` 온도 미분에서 소멸 — 엔트로피 동일 결론 성립. 보조 문장(L481)의 T→0 극한 ε₀+(3/2)ħω₀ 도 3모드×ħω₀/2 로 정확.

## 4. 생존 지적 (refute 시도 후 생존만)

**F-1 [LOW] ★가장 약한 1곳 — 그림 A.2(fig:app-phasediag) binodal 좌표 4점의 마지막 자리 슬립.** 캡션은 "좌표는 식~(binodal)·(spinodal) 그대로의 수치 평가"를 명시하는데, 표기 ξ 그대로 재평가하면 (0.01, 0.4266)→정확값 0.42654(=0.4265), (0.35, 0.9693)→0.96924(=0.9692), (0.40, 0.9864)→0.98652(=0.9865), (0.45, 0.9966)→0.99666(=0.9967). 최대 편차 1.2×10⁻⁴ = y축 4.4 cm/unit 에서 **0.53 µm** — 시각적으로 완전 무해하나, 그림 A.1(전점 정확)과 ch1 hysloop 그림(4자리 전점 정확 — 같은 기준이 달성 가능함을 자증)이 지키는 자기 기준을 이 4점만 못 지킨다. 부록의 반증가능성 앵커(캡션 주장)가 문자 그대로는 깨지는 유일한 곳. 수정 시 4개 y 값만 0.4265/0.9692/0.9865/0.9967 로 교체.

**F-2 [LOW] 미사용 정의 — `derivbox` 환경(L29).** `\newtheorem*{derivbox}{유도}` 가 선언만 되고 본문 사용 0회. 자족 초안의 orphan 정의(keybox 는 2회 사용). tikz 라이브러리 `calc` 도 미사용.

**F-3 [LOW] S6/R4 의 국소-구동력 counterfactual 단조성의 암묵 전제.** "V↑⇒𝒜↑⇒유효 장벽↓⇒L_q↓" 사슬은 dlnL_q/d(𝒜/RT)=−χ_d+[1+e^{𝒜/RT}]⁻¹ 이므로 χ_d>e^{−𝒜/RT}/(1+e^{−𝒜/RT})≈0.0127(𝒜=4.357RT)–0.018(𝒜=4RT 캡) 에서만 성립 — 허용 경계값 χ=0(방전 χ_d=0)에서는 detailed-balance 분모 몫이 남아 방향이 뒤집힌다. 실현 미분은 𝒜 동결로 어차피 0 이라 표의 PASS 판정 자체는 유지되며, 동기 서술이 "χ_d 가 미소하지 않다"는 전제를 무언 중 깔고 있다는 관찰. 판정 변경 불요 — 후속 개정 시 한 구 보강 후보.
