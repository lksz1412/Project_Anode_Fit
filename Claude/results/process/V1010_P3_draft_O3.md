## §3. ★ 발열 이론 갈고닦기 (중점)

> 작업지시 ②. q_rev 유도를 식→식으로 닫고, ΔS↔q_rev·비가역 옵션·부호·차원·극한을 자기완결로.

### 3.1 Bernardi–Pawlikowski–Newman 에너지 수지 → q_rev (식→식 유도)

**(a) 출발 — 셀 에너지 수지.** 단일 활물질 한계에서 전지의 총 발열률은 가역·비가역 두 성분으로 갈린다(Bernardi 등 1985, Ch2 eq:qrev L643):
```
Q̇ = I·(U_oc − V) − I·T·∂U_oc/∂T
     └─ Q̇_irr ≥ 0 ─┘   └─── Q̇_rev ───┘
```
- 첫 항 Q̇_irr = I(U_oc−V): 단자 전압 V가 평형 U_oc에서 벗어난 만큼(과전압 η=U_oc−V)의 소산. 2법칙상 방전·충전 무관 **항상 ≥0**(발열). Ch1의 동역학 꼬리·분극이 만드는 entropy production. [확정·Ch2 L648]
- 둘째 항 Q̇_rev = −I·T·∂U_oc/∂T: 가역(엔트로피) 발열, **부호 양방향**.

**(b) 연산 — ∂U_oc/∂T를 ΔS로 환산.** 평형 전위와 반응 Gibbs 에너지가 ΔG=−F·U_oc로 묶이므로(Nernst), Gibbs–Helmholtz:
```
ΔS = −∂(ΔG)/∂T = −∂(−F·U_oc)/∂T = +F·∂U_oc/∂T          (식 3.1)
```
∴ ∂U_oc/∂T = ΔS/F. [확정·Ch2 srcbox L651-653]

**(c) 중간식 — 대입.** 식 3.1을 Q̇_rev에 넣으면:
```
Q̇_rev = −I·T·∂U_oc/∂T = −I·T·(ΔS/F) = −(I·T/F)·ΔS(x)    (식 3.2)
```
**(d) 박스 — 가역 발열 닫힌식.**
```
┌──────────────────────────────────────────────┐
│  Q̇_rev = −I·T·∂U_oc/∂T = −(I·T/F)·ΔS(x)        │   (eq:qrev, Ch2 L645)
└──────────────────────────────────────────────┘
```
여기서 ΔS(x) = §sec:vibel 세 분포의 합(config+vib+electronic) = +F·∂U_oc/∂T. [확정]

### 3.2 ΔS ↔ q_rev 부호 사슬 (흡열/발열 교대)

방전(I>0)에서 식 3.2가 부호를 곧장 정한다:

| ΔS(x) 부호 | Q̇_rev 부호 | 열 거동 | 흑연 SOC 영역 (Ch2 L661-664) |
|---|---|---|---|
| ΔS > 0 | Q̇_rev < 0 | **흡열**(endothermic, 셀이 열 흡수) | 저-x: config 양의 발산 지배(+29 J/mol/K @x=0.08) |
| ΔS < 0 | Q̇_rev > 0 | **발열**(exothermic) | 고-x: vib 음 baseline(−16 @x>0.5); stage 2→1 큰 음 ΔS |
| ΔS = 0 | Q̇_rev = 0 | 중립 | 봉우리 중심 ξ=½ (config=0, 중심 표준값만) |

→ SOC에 따라 ΔS(x) 부호가 바뀌므로 **흡·발열이 교대**. stage 2→1의 큰 음 ΔS가 가역 발열의 두드러진 신호 → calorimetry 직접 관측과 정합(Ch2 standardised2024 인용). [확정]

**물리 해석(자기완결)**: q_rev는 "한 충전상태→다른 충전상태로 넘어갈 때 Li 배열 분포·포논 분포·전자 분포를 **재배열하며 주고받는 열**". 모양이 곧 분포다. [확정·Ch2 L656-657]

### 3.3 가역 / 비가역 분리 (옵션·히스테리시스)

**가역(2종)**: 위 q_rev=−(IT/F)ΔS. 충·방전 대칭(부호 양방향).

**비가역(2종, 항상 발열)**:
1. **과전압 소산** Q̇_irr=I(U_oc−V)≥0: Ch1 동역학 꼬리·분극(R_n) 기원.
2. **히스테리시스 gap** ΔU^hys: 한 사이클당 ∝ I·ΔU^hys의 entropy production(소산열). **∂/∂T와 별개**, 가역열에 안 섞임. [확정·Ch2 L586-587]

**★히스 분기 가역/비가역 분리 (파생 D)**:
```
∂U_oc^(d)/∂T = (1/F)·ΣQ_j·g_j^(d)·ΔS_rxn,j / ΣQ_j·g_j^(d)    (분기별, eq:hys_branch L576)

가역 = 두 분기 평균:
┌─────────────────────────────────────────────┐
│ ∂U_oc^rev/∂T = ½(∂U_oc^ch/∂T + ∂U_oc^dis/∂T)  │   (eq:hys_rev, L583)
└─────────────────────────────────────────────┘
```
근거: 한 사이클 돌면 히스 중심(±½ΔU^hys)이 상쇄되고 열역학적 ΔS만 남음. [확정]

**혼동 금지(Ch2 keybox L588-593)**: ∂U^rev/∂T = 분포 재배열의 가역 엔트로피; ΔU^hys = 경로 비가역성의 소산. 하나의 ∂U/∂T로 뭉치면 가역/비가역을 섞음. (경로의존 측정 불확실도 정량 = Ch2 범위 밖, 갭 명시.)

### 3.4 부호·차원·극한 적대 검산 (자기완결)

**차원 검산:**
| 양 | 식 | 차원 | 판정 |
|---|---|---|---|
| ΔS | +F·∂U_oc/∂T | [C/mol]·[V/K]=[J/(mol·K)] | ✅ |

**부호 검산:**
- ΔS_rxn>0 → ∂U/∂T>0 (U_j=(−ΔH+TΔS)/F의 T계수 양): 열역학 정합. [확정·Ch1 eq 일관]

**극한 검산 (6코너 — Ch2 tab:limits L606-617 + 발열 확장):**

### 3.5 계산용 종합식 (use-this, Ch2 keybox L668-685 정리)
```
                ΣQ_j·g_j(x)·[ΔS⁰_j/F + (R/F)·ln(ξ_j/(1−ξ_j))]
∂U_oc/∂T(x) = ──────────────────────────────────────────────── ,   g_j = ξ_j(1−ξ_j)/w_j
                          ΣQ_j·g_j(x)

Q̇_rev = −I·T·∂U_oc/∂T
```
- **입력** = {ΔS⁰_j, Q_j, U_j, w_j} + Ch1의 ξ_j(V,T) (eq:logistic).
- ΔS⁰_j = 봉우리 중심 표준값(파생 B 정의), 다온도 dQ/dV 동시 피팅으로 획득.
- w_j: 단상(Ω<2RT)=평형 예측 RT/F / 두-상(Ω>2RT, 흑연 staging)=현상학적 피팅 폭(파생 C).
- ξ_j 평가: 전하보존 음함수 ΣQ_j·ξ_eq,j(U_oc,T)=Qx 풀어 U_oc(x,T) 얻고 되먹임. [확정]
