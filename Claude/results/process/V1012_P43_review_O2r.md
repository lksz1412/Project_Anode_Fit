**가장 약한 1곳**: **D-1** — map 자신의 체크리스트를 그대로 따르면 `sec:lco-decomp` 한 절 안에 정부호(c)와 역부호 bullet L1728 이 동시 존재하는 **실재 내부 모순**이 별도 커밋 전까지 산다. (다른 적발은 전부 LOW/cosmetic 또는 확정-정합.) 본 검수는 F-1 이 3곳 교차로 진짜 부호 결함임을 독립 확정했으므로, F-1 정정을 §4 편입과 **동시**로 올려 원자성을 강제할 것을 권한다.

**5줄 요약**
1. 삽입 정합(줄·라벨·경계)·물리 소당성(절별 독립 재유도 + Python 실측 12항)·물리 불변 계약 **전량 PASS** — map 은 편입 가능 상태.
2. map §8 **F-1(Ch1 L1728 config 내부항 역부호)은 진짜 결함**임을 독립 3중 교차(Ch2 dVdT_config·파생A·revheat) + 극한(ξ→1 +∞) 으로 확정 — map 의 적발·정부호 configsplit 작성이 옳다.
