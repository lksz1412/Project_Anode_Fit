## 3. 척추(spine) — 절 구성 (플로우차트 순서대로)
각 절 = 한 노드의 물리식을 유도. 권장 절 골격(번호·제목은 자유, *순서는 고정*):
1. 도입·기호·실험조건 매핑 (N0): direction→σ_d, C-rate→|I|, T, Q_cell, q=Q/Q_cell. 본 장이 뽑는 양 = dQ/dV.
2. 분극 (N1): V_n = V_app − σ_d|I|R_n 유도(관측↔열역학 전위).
3. 평형 중심 (N2): U_j(T)=(−ΔH_rxn+TΔS_rxn)/F. ΔG=−FU·열역학에서.
4. 폭·평형 점유 (N4,N5): w=nRT/F (w^eff 옵션), ξ_eq=logistic[σ_d(V_n−U)/w]. 통계역학(정규용액 μ)에서 logistic 유도.
5. 평형 peak (N6): Q_jξ_eq(1−ξ_eq)/w = Q_j|dξ/dV|. |I|→0 한계 = equilibrium().
6. 히스테리시스 분기 (N3): 정규용액 Ω, spinodal gap ΔU_hys, 분기중심 U_j^d=U_j+½σ_d h_η γ ΔU_hys. **방전 메인·히스는 분기 위상**을 구조 선언으로.
7. 동역학 꼬리 (N7): Eyring/BV → affinity A, 방향 χ_d(충전 1−χ), ΔH_a^eff=ΔH_a−χ_dΩ, L_q, L_V=|dV/dq|·L_q.
8. 인과 기억 꼬리 (N8): 지수기억 합성곱, ★충전 격자 역전, peak=(ξ_eq−ξ_lagged)/L_V. L_V 작으면 평형 peak 환원.
9. 합산 (N9): dQ/dV = C_bg + Σ_j Q_j[평형−지연]. staging 4전이(GRAPHITE_STAGING_LIT) 적용.
- 절마다 **도입(앞 절에서 받은 것)·마무리(다음 절로 다리)** 의무. 본론(방전)/확장(히스) 위상 선언.

## 4. 형식 규범 (v5 따름 + Anode 교과서 규범)
- **수식 주도**: 산문 최소, 논리 다수를 식 사슬이 운반. 산문-only 유도단계는 명시 중간식으로 승격.
- **괄호 전보체 금지**: 주술 갖춘 보충 명제·3어절+ 단서를 괄호에 넣지 말고 완결 문장으로. 허용 괄호 = 식·절 참조, 짧은 동격(기호 병기)뿐.
- **보편식 먼저**: 일반해/보편형 먼저 유도 → 극한(Ω≤2RT, |I|→0, post-peak)은 그 코너 케이스로. 극한 가정을 먼저 깔아 전역에 쓰지 말 것.
- **피팅 실행력 (G-usable)**: "이 문건만 보고 v11 곡선 재현 코드를 짤 수 있을 만큼" — 컷(z_cut)·창·회귀 입출력·초기값·진행이 닫혀야. GRAPHITE_STAGING_LIT 초기값·각 인자 의미 표 포함.
- **G-follow**: 학부생이 앞 식+본문만으로 따라갈 수 있게(전제 없이 등장하는 개념 → 전제부터 단계 전개, 비유 의존 X).
- 본문 = 한글 prose + 영어 학술/기술 용어 원어. 식별자·기호·부호는 v11_final 과 1:1.
- 분량은 콘텐츠의 자연 결과(패딩 금지). 코드 식별자 1:1 박스/표·worked example 권장.

## 7. 작성 방식 — 절별/수식별 루프 + 자체 10회 검수 (★토큰 각오)
- **단위 구성 루프**: 절(또는 식 블록)을 **하나씩** 닫는다 — 정독→작성→자체검수→앞 절·기반식 정합→(주기적) 빌드→사유. **통째 한 번에 쓰지 말 것**. 복붙·무수정도 정독 근거 없으면 미완.
- **자체 10회 가변-청크 검수**(하한 10, 연속 2R 0결함까지): 매 라운드 **청크 스킴 전환**(통독/절/식/라인) + **렌즈 로테이션**: ①구조·spine 정합 ②물리·**부호 사슬** 적대검산(재유도·극한·반례) ③G-follow ④G-usable(피팅 코드 시뮬) ⑤완결성 orphan ⑥그림 효과·시각(PDF) ⑦직전 수정이 들인 새 결함. "조사만 바꾸는" 표층 라운드 금지.
- 매 라운드 발견 결함을 직접 수정하고 빌드 유지.

## 8. ★부호 사슬 체크리스트 (라운드마다 전건 확인 — 코드 v11 과 대조)
1. U_j(T)=(−ΔH+TΔS)/F (부호: ΔH<0 흡열 아님, ΔG=−FU).
2. ξ_eq = logistic[σ_d(V_n−U)/w] — 방전 σ_d=+1 에서 V↑→ξ↑.
3. dξ/dV = σ_d·ξ(1−ξ)/w → peak 는 양수(방향 불변).
4. ΔU_hys ≥ 0, Ω≤2RT→0. 분기중심 ±½σ_d…(방·충 반대).
5. χ_d: 방전 χ / 충전 1−χ. ΔH_a^eff=ΔH_a−χ_dΩ.
6. ∂lnL_q/∂V<0 (V↑→A↑→장벽↓→꼬리 짧아짐).
7. 꼬리: 충전 격자 역전(ξ[::-1]…[::-1]); 충전 dQ/dV = 방전 거울(양수).
8. 분극: V_n=V_app−σ_d|I|R_n.
*방전 시 V↑→탈리튬화↑, 충전 dQ/dV=방전 거울* 와 전건 정합. 한 곳이라도 어긋나면 결함.
