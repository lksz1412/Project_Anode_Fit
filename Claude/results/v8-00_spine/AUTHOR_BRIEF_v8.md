## 3. ★유도 사슬 의무 (이번 작업의 핵심 — G-derive)
각 결과 박스 식 위에 **(a) 출발 식/전제 명시 → (b) 적용 연산(미분·대입·극한·근의공식·적분인자 등) 명시 → (c) 중간식 ≥1 보임 → (d) 박스**. **"대입하면/넣으면/정리하면 [박스]" 한 줄 점프 금지.** (단 자명한 1줄 이항은 과주 금지 — 균형.)

복원할 11 식의 유도 단계(이 단계들이 *보이게*):
1. **U_j(T)**: G=H−TS → μ≡∂G/∂n → 전기화학 평형 μ=μ⁰−sF(V−U) → ΔG=−sFU → ΔG=ΔH−TΔS 대입 → U=(−ΔH+TΔS)/F.
2. **w_j**: 이상 격자기체 μ 로그 몫 기울기 → w=RT/F(단일) → 다중도 n_j → w_j=n_jRT/F.
3. **ξ_eq logistic**: 격자기체 μ(섞임 엔트로피 로그 몫) → 정·역 속도 r⁺=r⁻ 정지점 → Boltzmann/detailed balance 비 ξ/(1−ξ)=e^{A/RT} → ξ 풀이 → logistic[σ_d(V−U)/w].
4. **평형 peak**: 보존식 Q q=Q_bg+ΣQ_jξ_j → V 미분 → dξ/dz=ξ(1−ξ)·연쇄율 dz/dV=σ_d/w → Q_jξ(1−ξ)/w.
5. **ΔU_hys**: g(ξ)=g⁰+RT[ξlnξ+(1−ξ)ln(1−ξ)]+Ωξ(1−ξ) → g″=RT/[ξ(1−ξ)]−2Ω → g″=0 근의공식 → spinodal ξ_s±=½(1±u), u=√(1−2RT/Ω) → V_eq(ξ)=U+RT/(sF)ln[ξ/(1−ξ)]+Ω/(sF)(1−2ξ) 비단조 → 두 극값 대입((1−2ξ_s∓)=±u·로그 몫 역수) → artanh 정리 → ΔU_hys=(2/F)[Ωu−2RT·artanh u].
6. **분기 중심**: 두 spinodal 평균=U_j(대칭) → 방향별 한 자유도 → U_j^d=U_j+½σ_d h_η γ ΔU_hys.
7. **L_q**: 지연 r=ξ_eq−ξ ODE dr/dq=dξ_eq/dq−r/L_q (정전류 dq/dt=|I|/Q_cell 환산) → 적분인자 일반해 → 완화 속도 k_j=k⁺+k⁻=k⁺(1+e^{−A/RT})(평형 근방 선형 완화=정·역 합·detailed balance k⁻/k⁺=e^{−A/RT}) → Eyring k⁺=(k_BT/h)e^{−(ΔG_a^eff−χ_dA)/RT} → L_q=|I|/(Q_cell k_j) 로그 4항(T_*/T·−ln(1+e^{−A/RT})·ΔG_a/RT·−χ_dA/RT).
8. **컷 affinity 𝒜**: 원천 dξ_eq/dq 가 정점의 5%로 떨어지는 컷 → 𝒜=z_cut·n_jRT(상한 A_cap·RT) — *전이당 상수*(코드 L335). ∂lnL_q/∂V<0 은 컷 규칙의 *동기*이지 곡선 따라 미분하는 양 아님(운영상 실현 미분 0). ★이 구분을 흐리지 말 것(v7 系統 결함).
9. **ΔH_a^eff**: 꼬리 구동력서 상호작용 몫 −Ω(1−2ξ) 분리 → 깊은 꼬리 ξ→1(충전 거울 ξ→0) 상수 +Ω 흡수 → ΔH_a−χ_dΩ. χ_d=방전 χ/충전 1−χ(합-1).
10. **L_V·인과 꼬리**: L_V=|dV/dq|·L_q → ODE 일반해의 지수 커널 합성곱 → 이산 저역통과 ξ_lag=ρξ_lag+(1−ρ)ξ_eq, ρ=e^{−Δgrid/L_V} → peak=(ξ_eq−ξ_lag)/L_V → ★충전 격자 역전(ξ[::-1]…[::-1], 인과 방향).
11. **합산**: 보존식 선형 합 → dQ/dV=C_bg+ΣQ_j[peak] → V_n 역보간.
