## ★A. 누락된 핵심 물리 — v3/v4/v5엔 있고 v8·v9엔 **0건** (확정)

### A-1. 종모양 broadening 설명 = "왜 Ω>2RT 평형-델타가 실측 유한폭 종인가" ★최대 누락

> "분기 영역(Ω>2RT)의 평형 기여는 plateau — 이상적으론 무한히 날카로운 선 — 인데 실측 peak가 유한폭 종인 까닭은 **두 가지 broadening**: (1) **입자마다 분기 전위가 달라 선이 입자 분포 폭만큼 번지고**(다입자 그림) (2) **동역학 지연이 비대칭 꼬리**를 더한다. 따라서 분기 영역 폭 w_j는 평형 예측이 아니라 broadening이 정하는 **현상학적 피팅 폭**이다."

### A-2. ρ_G 배리어 분포 절 ("입자 분포 — 같은 온도가 산포를 키운다")
v4/v5 verbatim: "입자마다 크기·결정성·국소 환경이 달라 활성화 장벽이 분포 ρ_G(G)를 이룬다 → 꼬리의 용량분율 가중 합 → stretched/KWW 꼬리. ρ_G→δ면 단일지수 환원."

### A-3. 다입자(Dreyer) one-at-a-time 전환 그림
v3 verbatim: "입자마다 분기 전위가 조금씩 달라 입자들이 한 번에 하나씩 전환하며…" — 평형 plateau의 다입자 기원.

### A-4. w의 이중지위 명시
- 단상(Ω<2RT): w=nRT/F = *검증 가능한 평형 예측*.
- 두-상(Ω>2RT, ★우리 staging 4개 전부): w = **현상학적 자유 피팅 폭**(평형 예측 아님).

## ★B. 내가 *적극적으로 잘못 만든 것* (오류)

### B-1. ★Ch2 v4의 w_eff(Ω)=w(1−Ω/2RT) "델타로 좁힘" = 잘못된 그림 전파 + 종 설명 누락 [최대 오류]
- 내가 이번에 Ch2 v4 파생 C로 "Ω↑ → w_eff→0 → dQ/dV가 델타로 뾰족"이라고 유도·박았다. 그런데:
  1. **실측은 종모양**(핸드오버·코드실측). 내 식은 *평형 델타 극한*만 주고 **종모양 설명(broadening)은 전무** — 사용자 분노의 정확한 지점.
  2. **코드는 그렇게 구현 안 함**: `use_w_eff=False`가 기본, w는 명시 자유 피팅(12–20 mV). w_eff 경로는 staging에 'w'가 있어 적용조차 안 됨.

- ★결론: Ch2 v4 §C(w_eff)는 (a) 종모양의 진짜 원인(두 broadening) 누락 (b) 두-상서 무의미한 평형-narrowing 그림 전파. **재작업 필요**: w_eff narrowing 대신 "평형=델타 / 실측 종=입자분기분포+동역학꼬리, w=현상학적 피팅 폭" 으로.

### B-2. ★코드 `Anode_Fit_v11_final.py`의 use_w_eff = 면적 보존 깨진 버그 (내가 인지 못 함)

## 개선-1. 종모양 broadening 절 — 전이별 구분 + 동역학 정초 (★최우선)
- **전이별 출발 구분(누락이던 핵심)**: dilute→stage4·**4L↔3L = solid-solution → 단일입자가 이미 종**(broadening 불요). **2L→2(LiC₁₂)·2→1(LiC₆) = two-phase plateau(강1차) → 평형은 델타에 가까움 → 이 둘만 broadening 필요**. [확정, Levi&Aurbach 1999·RSC D0TA12607A]
- **두-상 델타→종 3 기작**(전부 평형 U_j 분포 아님): ① **단일입자 유한율속 비대칭 꼬리**(=모델 L_V, 치우침 주범) ② ★**입자 크기 *kinetic* 분산** τ=r²/D, 무차원수 (r²/D)·C-rate — C-rate↑ 심화·저율 소멸(=사용자 신가설, **옳음**; Yang 2023 흑연 operando 직접) ③ **내재 RT/F 폭**(floor ~26 mV).
- ★**반경은 *동역학 채널로만***: apparent-U = U_j + η, 종을 만드는 건 **η(과전압) 분포**. 평형 U_j 는 GITT상 입자 무관 상수(반경→U_j 열역학·밴드 둘 다 마이크론서 ~0.01 mV·무효). **반경→평형U 가설은 끌어오지 말 것**(Round 1 부정).
- **ρ_G 배리어 분포 절 복원**(입자 크기·결정성·국소환경 → ρ_G → KWW 꼬리) + ★**forward-only·역산 금지(ill-posed) 경고** 동반. "입자크기→G 맵"은 동역학(τ∝r²)으로.

## 개선-2. w 의 이중지위 명시 + Ch2 v4 §C(w_eff) 정정
- **단상(Ω<2RT)**: w=nRT/F = 검증 가능한 평형 예측. **두-상(Ω>2RT, 우리 staging 4개 전부)**: w = **현상학적 자유 피팅 폭**(평형 예측 아님). → v8/v9·Ch2 v4 본문에 이 구분 명시.
- **Ch2 v4 §C(w_eff=w(1−Ω/2RT)) 정정/제거**: 두-상서 무의미(평형-narrowing은 실측 종과 반대). 종모양은 §개선-1 broadening 이 설명. w_eff 절을 (Ω<2RT 단상 한정 각주 + "두-상 종폭은 broadening·자유 피팅" 으로) 바로잡거나 제거.

## 개선-3. 코드 `Anode_Fit_v11_final.py` 결정 (원본불가침 → 승인 후)
- **use_w_eff**: ① **제거 권장**(물리 거꾸로 + 면적보존 깨진 버그·staging엔 'w' 있어 적용도 안 됨) / ② ξ_eq 폭·분모 w 일치 수정(면적 보존). → 사용자 결정.
