# 문건이 종모양/분포를 어떻게 설명하는가 — v3·v4·v5 vs v8 (verbatim 대조)

> 사용자 질문: "저 분포는 v8 혹은 이전 v3·v4에서 뭐라고 설명하더냐?" → 문건 직접 grep·정독.

## v3(Fable_v3)·v4(Opus_v4)·v5(Opus_v5) — ★분포를 *명시적으로* 설명 (셋 다 동일 골격)
**핵심 문장(v4 line 2084–2089 / v3 line 2080 / v5 line 882, verbatim):**
> "분기가 존재하는 영역($\Omega_j>2RT$)의 평형 기여는 plateau — 이상적으로는 무한히 날카로운 선 — 인데, 실측 peak 가 유한 폭의 종으로 보이는 까닭은 **두 가지 broadening** 이다. **입자마다 분기 전위가 조금씩 달라 선이 입자 분포의 폭만큼 번지고(다입자 그림), 동역학 지연이 비대칭 꼬리를 더한다.** 따라서 분기 영역의 폭 $w_j$ 는 평형 예측이 아니라 broadening 이 정하는 **현상학적 피팅 폭**이다 — 단상 영역에서 $w_j$ 가 …검증 가능한 평형 예측인 것과 지위가 다르다."

→ 즉 v3/v4/v5는 사용자 질문(Ω>2RT 델타가 왜 종?)에 **이미 답했다**: 종 = (1) **입자간 분기(전이) 전위 분포** [방향 공통] + (2) **동역학 지연 꼬리** [방향별]. 강1차 구간의 $w_j$ 는 평형값이 아니라 현상학적 피팅 폭.

**+ 배리어 분포 절(v4/v3 §"입자 분포 — 같은 온도가 산포를 키운다", line ~1611/1605):**
- "실제 전극은 입자 집단이고, 입자마다 **크기·결정성·국소 환경**이 달라 활성화 장벽이 한 값이 아니라 분포 $\rho_G(G)$ 를 이룬다." 각 집단 $L_q(G)\propto e^{(G-\chi\mathcal A)/RT}$ → 꼬리의 용량분율 가중 합(식 superpose) → stretched/KWW 꼬리. $\rho_G\to\delta$ 면 단일지수로 환원.
- ★**역문제 경고(v4 line 1650 / v3 1644, verbatim)**: "$\rho_G$ 는 꼬리에서 **역산하지 않는다. stretched 꼬리에서 분포로의 역문제는 악조건(ill-posed)** … 입자 크기·구조 분포에서 **forward 로 모형**해 넣고 예측을 데이터와 대조한다(구체 분포족과 **입자 크기→G 맵은 이후 장의 소관**)."
- line 2028(v3): "입자마다 분기 전위가 조금씩 달라, **입자들이 한 번에 하나씩 전환**하며…"(Dreyer 다입자 그림).

## v8(v8-11) — 분포를 *단순화로 제거*
grep 결과 v8-11.tex 엔 `\rho_G`·"입자 분포" 절·"두 가지 broadening"·"현상학적 피팅 폭" **없음**(Boltzmann 분포·broadened 꼬리 라벨 등 일반 언급만). v8은 **단일 유효입자**로: 평형 폭 $w=nRT/F$(또는 $w_{eff}$) + **단일 kinetic lag $L_V$**(causal lowpass 꼬리). 분포는 유효 $w$·$L_V$ 로 흡수.

## 연구(R1–R3) 결론과의 대응 — 문건은 "구조"는 맞았고, 연구는 "원인"을 채운다
| 문건의 설명(v3/v4/v5) | 연구가 밝힌 물리적 원인 |
|---|---|
| (1) "입자마다 분기 전위 분포" (다입자, 방향공통) | apparent-U=U_j+η 분포. **평형 U_j 분포는 작음/반경독립**(R1·R3): 반경→U_j 열역학·밴드 둘 다 마이크론서 무시; 진짜 평형 이질성=구조무질서(흑연화도)+미평형/히스 |
| (2) "동역학 지연 꼬리" (방향별) | R2: kinetic lag $L_V$; **입자 크기 kinetic 분산**(τ∝r²)이 여기 들어감(C-rate 비례·저율 소멸) |
| $\rho_G(G)$ 배리어 분포 ← 입자 크기·결정성·국소환경 | = R2 동역학 분산의 정식화. 크기가 들어오는 **올바른(동역학) 채널** |
| "$\rho_G$ 역산 금지·forward-only·ill-posed" | = R1 결론과 **정확히 일치**(1종 Fredholm·DRT 동형) |
| "입자 크기→G 맵은 이후 장 소관" | 미완 — R2가 그 맵(τ∝r²·Yang2023)을 채움 |

→ **사용자 본인 v3/v4/v5가 이미 옳은 골격(델타+두 broadening, forward-only, ill-posed)을 담았고, v8이 단순화로 덜어냈다.** 연구는 "입자간 분기전위 분포"의 *물리적 정체*(대부분 kinetic+구조무질서, 평형 반경분포 아님)를 규명해 그 빈칸을 채운다.
