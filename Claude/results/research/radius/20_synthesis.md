# 20 — 종합·삼각검증·경쟁가설 분리·역문제 사슬 커버맵

> master 직접 작성(CORE 판단). 4개 축 카드(`10`·`11`·`12`·`13`) 전부 master 정독 후 삼각검증.
> 4-tier: 확정 / 근거미발견 / 추정 / 미검증.

## A. 사용자 가설의 논리 사슬 — 고리별 판정
사용자 추정 = 5-고리 사슬:
**[L1] 단일입자 Ω>2RT → dQ/dV ≈ 델타** → **[L2] 실측 종모양 = 단일입자 델타 ⊛ U_j 분포(통계역학적 평균)** → **[L3] U_j 분포 = 입자 반경 분포(반경만 의존, 구형)** → **[L4] peak → 분포 역변환(deconvolution)** → **[L5] 반경 분포 추산**.

| 고리 | 판정 | 근거(삼각검증) | tier |
|---|---|---|---|
| L1 단일입자 델타 | **조건부 성립** — 평형/저전류 극한에서만 | Maxwell 작도(00_scope master 논리) + Bai-Cogswell-Bazant Nano Lett. 2011(10.1021/nl202764f): 단 임계전류↑면 spinodal 소멸→균질충전(델타 아님). 흑연 전이 중 4L–3L 은 solid-solution(1차 아님) | 확정(평형)·확정(고전류 예외) |
| L2 델타⊛분포=종모양 | **조건부·약함** — 분포가 *유일* 원인 아님 | Dreyer Nat.Mater.2010(10.1038/nmat2730): *동일* 입자도 순차충전만으로 plateau → 종폭이 반드시 분포 때문 아님. 흑연 dQ/dV 를 "델타⊛분포"로 직접 분해·검증한 1차문헌 **근거미발견** | 추정 |
| L3 U_j분포=반경분포 | **★불성립(마이크론 흑연)** | 축B: Gibbs–Thomson ΔV=2γV_m/zFr → 마이크론(r≈5µm) ≈ **0.01–0.05 mV**, peak폭(수십 mV)의 **~1/1000**. 큰 크기효과는 나노(<~50nm; LFP 임계 22nm, Cogswell-Bazant ACS Nano 2012, 10.1021/nn204177u)에서만. 흑연 staging 전위의 *평형* 입경 의존 직접 문헌 **근거미발견**(흑연 입경 문헌은 전부 kinetic) | 추정(값)·확정(1/r 스케일)·근거미발견(흑연 평형) |
| L4 역변환 | **ill-posed** | 축C: 1종 Fredholm = DRT 동형, "intrinsically ill-posed, Tikhonov 필수". dQ/dV→PSD 직접 선례 근거미발견(EIS 선례 Song-Bazant 2012 arXiv:1205.6539 은 형태가정+fit). 비유일(identifiability arXiv:1702.02471) | 확정(ill-posed) |
| L5 반경분포 산출 | **현 조건에선 불가** | L3 평탄·L4 ill-posed·L2 다원인 중첩의 합성 결과 | 확정(부정) |

## C. 삼각검증·갭
- **합의(≥2 독립)**: (i) 단일입자 1차전이=평형 plateau→델타[축A·00_scope]. (ii) 크기→전위 결합은 나노 현상, 마이크론서 1/r 소멸[축B·C5 lattice-gas 틀과 정합]. (iii) 역변환 ill-posed[축C, DRT 동형]. (iv) 유한전류 kinetic 이 broadening 주도·C-rate 가 분리신호[축D·축A 카드1-3].
- **논쟁/nuance**: 비대칭의 재료내재 성분(Weng/Siegel) — kinetic도 반경도 아님. 순차충전(Dreyer)이 분포 없이 plateau 생성 → L2 약화.
- **공백(근거미발견)**: ① 흑연 staging 전위의 *평형* 입경 의존 직접 측정 ② "델타⊛U_j분포=흑연 종모양" 직접 피팅 ③ dQ/dV→PSD 직접 역변환 선례 ④ 반경→U_j mV 환산 정량. → 가설은 *반증*된 게 아니라 마이크론서 *정량적으로 무효+미입증*.
- **abstract-only 강등**: Nature/Springer 본문·ScienceDirect 403 다수(축A·D). 핵심 정량(축B 0.01–0.05 mV 계산, 축D 70 mV Weng 본문, Park GITT)은 접근·검증됨.

## D. 역문제 성립조건(L4) — 충족도
1. 커널 확정(단일입자 본질폭) — 모델서 미확정(w_eff floor 1.3 mV vs 기본 25.7 mV): **미충족**.
2. 커널폭 ≪ 분포폭 — 마이크론선 분포(반경 기여)가 ~0.01 mV 라 오히려 분포가 커널보다 좁음: **역전·미충족**.
3. 반경→U_j 단조·일대일 — 마이크론서 사실상 평탄(무감): **미충족**.
4. broadening 원 분리(kinetic·불균일·조성 공제) — 가능하나 선행 필수: **조건부**.
5. 정규화/형태가정 — 필수(Tikhonov/lognormal): **요구사항**.
6. 독립 PSD(SEM·레이저회절) 대조 — 필수: **요구사항**.
