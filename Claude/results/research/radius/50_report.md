# 50_report — 교수님 1–2p 요약: dQ/dV peak 형상으로 입자 반경 분포를 추산할 수 있는가

## 질문
흑연 음극 ICA(dQ/dV)에서 Ω≥2RT 인 staging 전이는 단일 입자라면 (유사) 델타로 나와야 하는데, 실측 코인 하프셀은 *치우친 종모양*이다. 이 종모양을 "단일입자 델타가 전이 전위 U_j 의 입자간 분포로 퍼진 통계역학적 형상"으로 보고, 흑연 입자를 구형·반경만이 U_j 를 결정한다고 가정하면, **peak 형상 = U_j 분포 = 반경 분포**로 역변환해 입자 반경 분포를 추산할 수 있는가?

## 답 (한 줄)
**부분 타당·조건부 — 직관은 옳으나 핵심 고리(반경→U_j)가 마이크론 흑연에서 정량적으로 무효이고, 역변환은 ill-posed이며, 실측 치우침은 반경보다 동역학적 분극이 지배한다.** 현 형태(마이크론·반경 단독·직접 deconvolution)로는 성립하지 않는다.

## 근거 (4단계)
1. **단일입자 델타는 맞다(평형 극한).** Ω>2RT 1차 상전이 → Maxwell 작도 → 전압 plateau → dQ/dV 델타. 단 고전류에선 단일입자도 균질충전(델타 아님, Bai-Cogswell-Bazant 2011), 흑연 4L–3L 은 연속 전이.
2. **ensemble 퍼짐 개념도 실재하나 유일 원인 아니다.** 다입자 순차충전은 *분포가 없어도* plateau 를 만들고(Dreyer 2010, Nat. Mater.), 입자 상호작용(autocatalysis·avalanche)이 있으면 분포 정보가 왜곡된다 → "ensemble = 정적 U_j 분포의 1:1 반영"은 입자 독립·근평형에서만.
3. **★반경→U_j 결합이 너무 약하다(치명적).** Gibbs–Thomson·strain 으로 전위가 1/r 로 이동하나, 마이크론 흑연(r≈5 µm)에선 **ΔV ≈ 0.01–0.05 mV** — 실측 peak 폭(수십 mV)의 **약 1/1000**. 큰 크기 효과는 **나노(<~50 nm; LiFePO₄ 임계 22 nm, Cogswell-Bazant 2012)**에서만. 흑연 staging 전위의 *평형* 입경 의존 직접 문헌은 없고(전부 kinetic), GITT 는 staging 전위를 입자 무관 상수(0.22/0.12/0.08 V)로 측정.
4. **역변환은 ill-posed + 치우침은 kinetic 지배.** peak→분포 = 1종 Fredholm(DRT 동형) → 비유일·불안정, 정규화/형태가정 필수, dQ/dV→PSD 직접 선례 없음. 실측 치우침은 유한 전류 분극(0.05→0.5C 에서 ~70 mV, staging 간격과 동급)이 주도하며 **C-rate 의존성이 분리 신호**.

## 성립 조건 (모두 충족 시 부분 의미)
근평형 측정(GITT·C/20–C/100) · **나노 활물질**(마이크론 흑연 부적격) · 입자 독립(상호작용 부재) · 반경이 U_j 이질성의 지배 원인 · 단일 격리 전이·커널 확정 · 정규화/형태가정 + 독립 PSD(SEM·레이저회절) 대조.

## 권고 (가설을 시험하려면)
C-rate 사다리+GITT 로 kinetic 공제 → 남는 잔여 폭만이 평형 이질성 상한 → 독립 PSD 와 상관 검정 → 단일입자 dQ/dV 폭으로 ensemble 기여 상한 실측. (모델에 분포를 넣는다면 반경 단독이 아니라 일반 U_j 이질성 분포 + C-rate 식별구속으로 — 구현은 본 조사 범위 외, 검토 후.)
