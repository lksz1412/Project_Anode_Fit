# 추출카드 — Ménétrier/Delmas 1999 (insulator–metal 전이·전자 물성)

- **저자·연도·DOI**: M. Ménétrier, I. Saadoune, S. Levasseur, C. Delmas, *J. Mater. Chem.* **9**, 1135–1140 (1999). DOI 10.1039/a900016j.
- **축**: C2(★상전이·전자 엔트로피).
- **핵심 방법**: LixCoO2 (0.5≤x≤1) 전기화학 탈리튬화 시료. XRD + 전기전도도(electrical conductivity) + thermoelectric power(Seebeck) + **7Li MAS NMR**(Knight shift)로 전자 국재(localized)↔비국재(delocalized) 전이 추적.
- **지배식/정성 결과**:
  - Li 탈리 시 **국재 전자 → 비국재(itinerant) 전자**로 점진 변화 = **insulator → metal 전이**.
  - **MIT 가 biphasic(2상) 도메인의 driving force** — 7Li MAS NMR 이 직접 시사. (절연 Li-rich phase + metallic Li-poor phase 공존)
- **정량값(값·단위·조건·불확도)**:
  - MIT 발현 x: 고화학량 Li1CoO2 는 **x=1−δ (δ≪1)** 에서 이미 metal 출현(즉 거의 x=1 직하). "x≈0.94" 의 통상 인용보다 빠름 — **O3-R1 하한 경계가 결정 결함 농도에 민감**(Motohashi 카드 line 416 재확인). tier A(다중 cross).
  - 2상역: x≈0.75–0.94 (Reimers·Motohashi 와 정합).
  - 전도도·Seebeck 절대값: 원논문 Fig. 유료 → 정성("localized→delocalized")만 확보. 정량 = tier C(미확보).
- **타당·한계**: MIT 의 **전자적 본질**과 2상역과의 인과를 NMR 로 직접 입증한 1차 원천. 한계: g(E_F)·전자 비열계수 직접 산출 X(→ Motohashi D(E_F)=13 e/eV, Reynier S_e≈0.18 kB/atom 으로 보완). 정량 전도도·Seebeck 수치는 본문 유료로 미확보.
- **우리 의도 관련성 (★전자 엔트로피 직접 anchor)**:
  - LCO 의 **MIT = 전자 자유도(국재↔비국재) 전이** → 전자 엔트로피 항이 x≈0.75–0.95 에서 *active* 임을 1차로 근거화. 우리 forward 의 전자 엔트로피 plug-in 발현구간 = 이 2상역.
  - "MIT 가 2상역 driving force" = 이 plateau(3.9 V, Xia 카드)는 **순수 configurational 로 설명 불가**(ML 카드: stat-mech 가 이 2상역을 못 만듦) → 전자 항 필수.
- **정독범위**: abstract(RSC·RG) full read + 2차 인용(Motohashi 본문 정독에서 [38] 로 재확인). 원논문 full-text = snippet.
- **tier**: MIT 본질·2상 인과 **A**, MIT x=1−δ **A**, 전도도·Seebeck 정량 **C**.
