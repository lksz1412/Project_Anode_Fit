## 제안 2 — 조성 의존 상호작용 Ω(ξ): 비정규용액 확장 (차기 후보 1순위)
- **현상태**: 정규용액 Ω_j 가 전이당 **상수**. g(ξ)=RT[ξ ln ξ+(1−ξ)ln(1−ξ)]+Ωξ(1−ξ).
- **문제**: (i) 흑연 4단 staging 비등간격이 Ω 상수로 미포착. (ii) LCO T2/T3 order-disorder 가 ξ=½ 비국소(hex↔monoclinic 대칭 파괴) — 대칭 이중웰로 비대칭 못 담음.
- **개선 방향**: Ω→Ω(ξ)=Ω₀+Ω₁(2ξ−1)(준화학 1차 보정). Ω₁≠0 → 이중웰 비대칭 → spinodal 끝점 ξ≠½ 이동. sublattice(Bragg-Williams) 확장 → T2/T3 대칭파괴 자연 예측. Ω₁ 물리기원(흑연: 층간, LCO: Li-빈자리 정렬 에너지) 1차원리 산출 권장.
- **난이도 중·침습도 높음**: g(ξ)·spinodal·히스(N2/N3)·Part 0 정규용액 코어 다수 수정 → 검증 부담 큼(회귀 재정초 위험). **선행 데이터**: Ω₁ 물리기원(문헌/DFT) 또는 비등간격 실측.
- **외부 위임 ★★★**: 1차원리 Ω₁ 산출은 독립 연구로 적합.

## 제안 3 — Cahn-Hilliard 동역학 → 히스테리시스 γ_j 물리 유도
- **현상태**: N3(§hys) 히스 gap 은 spinodal 유도이나 분기 축소 인자 γ_j 는 **현상학적**(유도 아님). 부록 `appendix_phase_separation.tex` 에 Cahn-Hilliard·spinodal 분해 선형이론 있으나 본문 미연결(★v1.0.17 에서 부록 κ/M 차원 정합 보강 = 이 승격의 토대).
- **문제**: spinodal 끝점(평형)까진 엄밀하나 과주행 궤적이 모델 가정. γ_j 피팅 파라미터라 예측력 0. 핵생성 vs spinodal 분해 구분의 gap 영향 미정량.
- **개선 방향**: 부록 CH(∂ξ/∂t=M∇²μ, μ=f'(ξ)−2κ∇²ξ)를 본문 N3–N7 로 승격. spinodal 성장률 R(k)=−Mk²(f''+2κk²)로 꼬리 길이 L_V 물리 근거. 핵생성 장벽 ΔG*=(16πγ³)/(3(Δg)²)에서 γ_j 유도.
- **난이도 고·침습도 높음**: 부록→본문 승격 + γ_j 유도 재구조. **선행**: 없음(v1.0.17 κ/M 기반). **외부 위임 ★★☆**(부록 토대 有).

## 제안 4 — 농도 분극 분리: Butler-Volmer + Nernst-Planck
- **현상태**: N1 분극 = lumped 저항 R_n(V_n=V_app−σ_d|I|R_n). 전해질 전도도·확산층이 R_n 흡수.
- **문제**: 고율속 농도 분극이 활성화 분극과 미분리. Li⁺ 농도 변화의 Nernst shift 누락. 고율속 dQ/dV 꼬리 비대칭의 동역학(L_V) vs 열역학(Nernst) 몫 미구분.
- **개선 방향**: Butler-Volmer i₀ 에 활동도 γ± + Nernst-Planck 확산층. R_n → R_ct(활성화)+Warburg(농도) 분리. 확산층 c_s/c_∞=1−|I|/i_L → Nernst shift ΔU=(RT/F)ln(c_s/c_∞).
- **난이도 중·침습도 높음**(N1 신규 layer). **선행 데이터**: 임피던스(EIS)·율속 series. **외부 위임 ★★☆**(표준 전기화학).

## 제안 5 — 다입자 PSD 컨볼루션: 나노 사이즈 확장
- **현상태**: broadening ③ Dreyer 순차전환 서술, 입자크기 효과는 Gibbs-Thomson 수치 배제(δU_PSD≈0.20 mV, 마이크론 흑연 한정). PSD 컨볼루션 항등 처리. ρ(U_app) 역산 ill-posed 금지.
- **문제**: 배제는 마이크론 한정 — 나노(Si·LCO 나노입자) 1/r 효과 무시 불가. Dreyer 유한 N 보정 없음. ρ(U_app) 미정량 → broadening ③ 크기가 폭 w_j 에 전부 흡수.
- **개선 방향**: PSD 컨볼루션을 유효반경 r_eff 분포로 정량. Gibbs-Thomson ΔU(r)=2γV_m/(Fr)를 PSD 위 적분 → broadening ③ 형상 예측. Dreyer 유한 N 보정(N 작으면 plateau 계단).
- **난이도 고·데이터 의존**. **선행 데이터**: PSD 실측(대상 전극). **외부 위임 ★☆☆**(데이터 의존 큼).
