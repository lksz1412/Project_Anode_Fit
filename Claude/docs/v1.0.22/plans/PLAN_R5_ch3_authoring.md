## 과제 패키지 (3창 공통 — 전 절 저작)
1. **§3.1 생존 지도 본문화** — 현행 `ch1_appD_si`(tab:simap·N0~N9 생존/재해석/새 물리·GS-1)를 장 본문 격으로 확장(부록체→본문체·Part 0/I 결과 xr live 인용).
2. **§3.2 케이스별 활물질** — 원소 Si·SiOₓ·Si–C 복합 3소절: 곡선 개형·평균 전위·1차 비가역(ICE)·히스테리시스·검토용 파라미터 셋 표(tier 명기). 문헌 = `comp_R4/SI_CASES.md`+`comp_R4/upgraded/` 4본(검증 완료 후보 — 수치는 표에 원문 확인분만).
3. **§3.3 혼합 음극 — 공통-μ 대정준** — 마스터플랜 §7 앵커의 수식화: $\sum_\mathrm{host}\sum_j Q_j\,\xi_{\eq,j}(U_\mathrm{oc},T)=Q\,\bar x$, $Q=Q_\mathrm{gr}+Q_\mathrm{Si}$, $f_\mathrm{Si}=Q_\mathrm{Si}/Q\in[0,0.3]$ — sec:sm-mc(eq:sm-mc-balance) 클래스곱→host 곱 한 줄 일반화·요동 유일근 논증 이월(xr). **+정직 공백 GS-2 신설**: 블렌드 실측(Ai2022·Chatzogiannakis2025 — comp_R4/upgraded/BLEND_UP)의 SoC 구간별 host 전환·비가산 거동 → 공통-μ 완전 동시반응은 1차 근사임을 공백 4분류로 명시.
4. **§3.4 기계 히스테리시스(GS-1)** — Larché–Cahn(larchecahn1973, 원장 V1) 응력-화학퍼텐셜 결합의 유도 **시도**: 닫히는 데까지만 적고, 못 닫으면 공백 4분류(논리 공백/물리 가정 충돌)로 정직 선언(마스터플랜 Non-goals: 못 닫으면 공백 유지).
5. **§3.5 코드 확장 요구명세(예고)** — R6 `BlendedAnodeDQDV(f_Si, si_case)` doc-leads 요구명세 초안(f_Si=0 bit-exact 게이트·용량 배분·공통 V_n 합성·케이스 셋 구조) — Ch1 부록 B/D 관행.
