\subsection*{기호 --- 계승과 신규}
\textbf{계승}(Chapter 1$\cdot$2 정의 그대로 --- 재정의 없음): 전하 보존 반전~\eqref{eq:sm-mc-balance}
과 그 유일근 논증·평형 진행률 logistic~\eqref{eq:logisticsolve}·폭 $w_j$·용량 $Q_j$·방향 부호
$\sigma_d$·히스 gap 골격~\eqref{eq:dUhys}(적용 한계는 본문 생존 지도 참조). \textbf{신규}(본 장
도입): Si 분율 $f_\mathrm{Si}=Q_\mathrm{Si}/Q$(혼합 음극 용량 몫, $0$--$0.3$ 검토 범위)·활물질
케이스 표지(원소 Si / SiO$_x$ / Si--C 복합)·응력--전위 결합 계수(생존 지도 \S\ref{ssec:si-gap} ---
골격 밖 새 물리로 선언). 케이스별 파라미터 셋과 블렌드 합성 기호는 해당 절 도입부가 정의 기준이다.
