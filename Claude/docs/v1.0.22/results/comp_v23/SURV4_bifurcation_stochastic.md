## 1. 기법별 정직 적합도 판정

### 1) 파국이론·정규형 (catastrophe theory / cusp normal form) — **등급 B**

- **문건 자리**: §2.2b `eq:sm-thresh`(개시 `Ω=2RT`) · §4.1 `eq:spinodal`(fold) · §4.2 `eq:dUhys`(gap `∝u³`).
- **무엇을 더하나**: 문건의 이중웰 자유에너지 `g_j(ξ)`(식 `eq:gxi`)는 ξ=½ 근방에서 **cusp 파국의 정규형 `V=x⁴/4+a x²/2+b x` 그 자체**다 — 제어변수 `a↔(2RT−Ω)∝(T−T_c)`, 場 `b↔` 전기화학 구동력. 그러면 **spinodal `ξ_s^±`=cusp 의 두 fold(접힘)선**, **개시 `Ω=2RT`=cusp 첨점(organizing center)**, 그리고 결정적으로 **문건이 이미 유도한 `ΔU^hys∝(T_c−T)^{3/2}`=cusp 의 보편 fold-분리 지수**다. 이는 재-명명이 아니라 **보편성 진술**을 준다: 3/2 지수는 Bragg–Williams 평균장의 우연이 아니라 첨점 근방 모든 gradient 계에 공통 → 문건의 `u³` 소멸이 근사에 강건하다는 근거.
- **검산(python)**: 문건 정확식의 `d ln(gap)/d ln(Ω−2RT)=1.4987`, 독립적으로 cusp 정규형 fold-분리 폭 지수 `=1.5000` — **일치**. 문건의 `(8/3)u³` 연성 개시가 정확식과 threshold 극한에서 비 →1(수치 재현). 문건의 `★Taylor 함정`(두 급수 동시전개 안 하면 `−4/3 u³` 부호 뒤집힘)은 cusp 언어로 "unfolding 을 완전히 전개해야 fold-분리가 양수"로 자연 설명.
- **안 되는 이유 / caveat**: (a) **새 예측 0** — 닫힌 식도 3/2 지수도 문건이 이미 가짐; cusp 는 그 위에 "보편"이라는 라벨만 얹음. (b) **팽창 리스크 중** — Thom 분류·전개(unfolding) 술어를 피팅 방법론 문서에 들이면 수학 과시로 읽힐 수 있음. (c) cusp 는 gradient(potential) 계 분류라 `g(ξ)` 에 정확히 적용되나, 문건 히스는 준안정 *과주행*(비평형 경로)이라 정적 cusp 이 경로 자체를 주진 않음.
- **권고**: §4.2 `eq:dUhys` 의 `(T_c−T)^{3/2}` 소멸 문장에 **1–2문장 각주** — "이 3/2 는 cusp 파국의 보편 fold-분리 지수이며 평균장 특유가 아니다". 절 신설 금지. 차원 0.

### 2) Preisach 모델 (히스테리시스 표준 엄밀 프레임) — **등급 B(분류 노트) / D(연산자 채택)** ★정밀 판정

브리핑의 최우선 판정 대상. "정합하면 강 후보"에 대한 **정직 결론: 형식적으로 정합하나 원자적(atomic) 극한이며, 승격 가치는 낮다.**

- **문건 자리**: §4.3 `eq:Ubranch` `U_j^d=U_j+½σ_d h_{η,j}γ_j ΔU^hys`, 특히 부분cycle 인자 `h_{η,j}`.
- **정합성 정밀 판정(핵심)**: Preisach 릴레이 히스테론 `γ̂_{αβ}`(상향 임계 α, 하향 임계 β)에 문건을 매핑하면 —
  ```
  방전 가지 중심(σ_d=+1): U_j + ½γ_j h ΔU^hys  ≡  α_j  (상향 스위칭場)
  충전 가지 중심(σ_d=−1): U_j − ½γ_j h ΔU^hys  ≡  β_j  (하향 스위칭場)
  히스테론 폭 α_j−β_j = γ_j h_{η,j} ΔU^hys  (γ=h=1 에서 spinodal 상한)
  ```
  곧 **문건의 전이 하나 = Preisach 릴레이 히스테론 하나**. N_p 전이 전체 = **원자적(atomic) Preisach 밀도** `μ(α,β)=Σ_j Q_j δ(α−α_j)δ(β−β_j)`. python 검산: γ=0.7, ΔU=54.7mV → α=+119.2, β=+80.8 mV-off, 폭 38.3mV — 정확히 재현. **⇒ 문건 히스는 Preisach 와 모순되지 않는다; Preisach 의 이산 델타-밀도 극한으로 정확히 embed 된다.**
- **그런데 왜 강 후보가 아닌가(승격 가치 판정)**: Preisach 가 원자적 극한 위에 더 주는 것은 **연속 밀도** `μ(α,β)`(각 히스테론을 스위칭場 분포로 번짐) 하나뿐인데 —
  1. **차원 폭발**: 연속 밀도는 *함수*(이산화하면 다수 파라미터)라 하드 제약 "새 파라미터 0" **정면 위반**. 문건은 이 번짐을 이미 **전이당 스칼라 `h_{η,j}`(부분cycle 축소, 기본 1) 로 붕괴**시켰다 — Preisach 의 minor-loop/FORC(1차 되돌이 곡선) 정보를 1-스칼라로 접은 것이 정확히 `h_{η,j}` 다.
  2. **스코프 벽**: Preisach 의 본질(wiping-out·return-point memory·minor-loop 중첩)은 **경로의존 메모리**인데, [C-92]·SM2 축 C 가 경로의존 정량을 *범위 밖*으로 확정. 피팅 워크플로가 full-sweep(또는 `h_η` 보정 부분sweep)만 쓰므로 wiping-out 은 **행사되지 않는 잉여 용량**.
  3. **성격 불일치**: 문건 히스는 *열역학적 분기*(2-가지, 영전류 잔존, γ_j)이지 *메모리 연산자* 히스가 아니다. "우리 히스는 Preisach"라 부르면 문건이 **일부러 두지 않은** 히스테론-중첩 메모리 구조를 가진 것처럼 오도.
- **안 되는 이유 / caveat**: 연속-밀도 Preisach 채택 = 차원+스코프 이중 위반(**D**). 반면 **"각 전이의 2-가지 gap 은 단일 Preisach 릴레이 히스테론이고, 모델은 그 원자적 밀도 극한이며, `h_{η,j}` 가 연속 밀도(minor-loop)를 스칼라로 접은 것"이라는 1–2문장 분류 노트**는 옳고·교과서 정합·정직(왜 full operator 가 불필요한가의 경계표시)해서 **B**. CNT 가 γ_j 의 *근거*이되 예측식이 아니듯(§4.3 선례), Preisach 는 `h_{η,j}` 의 *분류학적 자리*이되 도입 대상이 아니다.
- **권고**: §4.3 `h_{η,j}` 정의 문장에 CNT_LINK 와 **동형의 경량 명명 노트**(서술형, 무라벨). 연산자·밀도·FORC 도입 금지.

### 3) 중심다양체 축약 (center manifold reduction) — **등급 D**

- **문건 자리**: 후보 삽입처 = §8 `eq:kuniv` 완화 동역학 / §4 spinodal 근방.
- **무엇을 (안) 더하나**: 중심다양체는 **고차원 결합 동역학계**를 분기 근방 저차원 slow-manifold 로 축약하는 도구다. 그러나 문건 동역학(§8)은 **이미 전이별로 분리된 스칼라 1계 선형 ODE** `dξ_j/dt=k_j(ξ_eq−ξ_j)` — 축약할 고차원 결합계가 없다. 유일 분기는 **정적 gradient fold**(cusp/평형, 이미 닫힌 식)이지 동역학적 분기(Hopf·리미트사이클·진동)가 아니다.
- **안 되는 이유 / caveat**: 축약 대상 부재 + 정규형 동역학 분기 부재. 유일한 잠복 고리 = spinodal 에서 `g''→0` 에 의한 **임계 감속**(k→0, 조성 요동 완화율 소멸)인데, 이는 (a) 문건이 §8 지연을 열역학 분기와 **의도적으로 분리**(전이당 스칼라 `L_V`)한 설계 밖이고, (b) 도입 시 kinetics↔`g''` 결합이라는 **신규 물리·신규 파라미터**라 스코프+차원 위반. 문건이 갖지 않은 문제를 위한 도구.
- **권고**: 비도입. (원하면 §8 또는 부록 Cahn–Hilliard 문단에 "spinodal 임계 감속은 fold 의 동역학적 그림자이나 분리-지연 설계 밖" 1문장 각주가 상한 — 그래도 C 미만.)

### 4) Fokker–Planck · 1차 통과시간 (Kramers escape) — **등급 B(부록 한정)**

- **문건 자리**: 부록 §A.6.1 `eq:app-rstar`(`ΔG*`·`rate∝exp(−ΔG*/k_BT)`) · §4.3 CNT 문단 · §8 `eq:kuniv`(완화).
- **무엇을 더하나**: 부록이 **단언**한 핵생성률 `∝exp(−ΔG*/k_BT)` 은 정확히 **Kramers 1차 통과시간의 고장벽 극한**(`τ~exp(ΔG*/k_BT)`, prefactor=시도 진동수)이다 — FPT/Kramers 는 부록이 *주장*한 것을 *유도*(+prefactor)해 자족성을 높인다. 더해, **통일 언어 payoff**: §8 완화(`dξ/dt=k(ξ_eq−ξ)`=조화웰 OU/선형 Fokker–Planck)와 CNT 핵생성(장벽 탈출 Kramers)이 **하나의 FPT 그림의 두 극한**(웰 바닥 완화 vs 장벽 넘기)임을 명시 — 사용자 "절간 연결" 지향 정면.
- **검산(python)**: `exp(−ΔG*/k_BT)` 지수 지배 재현(ΔG*/kT=5,10,20,40 → 6.7e-3 … 4.2e-18). Kramers MFPT 고장벽식이 부록 지수를 정확히 재현.
- **안 되는 이유 / caveat**: (a) 전적으로 **부록(이미 본문 범위 밖·독립 컴파일 문서)** 안에 삶 — 마스터 tex 는 상분리 부록을 `\input` 안 함(CNT_LINK 헤더 확인). 본문 모델 차원 무영향. (b) prefactor/시도진동수는 피팅이 **쓰지 않는** 미시 상세(γ_j 가 현상학 흡수). (c) 값어치는 "단언→유도"의 자족성 향상 + 완화↔핵생성 연결로 *중*.
- **권고**: 부록 §A.6.1 에 **Kramers 각주**(MFPT 고장벽 극한 = 부록 지수의 유도) + §A.7 "본문과의 연결"에 "§8 완화와 §A.6.1 핵생성 = 한 Fokker–Planck 의 두 극한" 1문장. 본문(Ch1) 무변경.

### 5) 대편차이론 (large deviations, LDT) — **등급 C**

- **문건 자리**: §6 `eq:eqpeak`+SM2A(`var(N)` 감수율) · §2.5 SM2B(`1/√M`) · §7 CLT · 부록 CNT(`ΔG*`).
- **무엇을 더하나("SM2 감수율의 확장")**: LDT 는 CLT(요동 *중심*, Gaussian, √M)를 넘어 **요동 *꼬리*(rare event)** `P~exp(−M·I(x))` 를 준다. 결정적 연결: **rate function `I(x)` 의 최소점 곡률 = SM2A 의 var(N) 감수율**(Gaussian 은 `I` 의 2차 전개)이고, **핵생성 장벽 `ΔG*/k_BT` = `I` 의 꼬리 비용**. 곧 LDT 는 **SM2(Gaussian 벌크) + CNT(지수 꼬리)를 하나의 rate-function 지붕으로 통일** — `var(N)`↔`ΔG*` 를 한 함수의 곡률과 꼬리로.
- **안 되는 이유 / caveat**: (a) **무거운 기계**(Cramér·Gärtner–Ellis·Varadhan)를 피팅 방법론에 들이는 팽창 리스크 최상. (b) **모델 payoff 0** — var(N)·CNT 이미 있음; LDT 는 재-지붕만. (c) **정직 한계**: SM2A/B 의 var(N)·`1/√M` 은 **자리 독립** 전제 위인데(SM2B 명시 가드), LDT 가 겨냥할 상분리 창은 정확히 **Ω>2RT 강상관**이라 독립-자리 rate function 이 거기서 깨진다 — LDT 를 정직하게 쓰려면 이 붕괴를 먼저 처리해야 함. (d) [C-92] 경로의존 벽과 인접.
- **권고**: 절 신설 금지. 최대 상한 = SM2A 또는 SM2B bgbox 에 **단일 연결문장** — "Gaussian 감수율(var N)과 핵생성 장벽(ΔG*)은 한 대편차 rate function 의 곡률과 꼬리이며, 후자는 독립-자리 전제가 깨지는 Ω>2RT 창의 문제". 그 이상은 C→D 로 하락.

### 6) Langevin · 확률미분방정식 (SDE) — **등급 D(#4로 병합)**

- **문건 자리**: 후보 = §8 완화에 잡음 추가 `dξ=k(ξ_eq−ξ)dt+√(2D)dW`(OU) / 부록 핵생성.
- **무엇을 (안) 더하나**: §8 결정론 완화가 **유한입자 Langevin 의 M→∞(무잡음) 극한**임을 명시하고 잡음 진폭을 var(N)에 연결하려는 것.
- **안 되는 이유 / caveat**: (a) **#4(FPT)와 동일 내용**(Langevin↔Fokker–Planck 는 같은 물리, 두 언어) — 별도 후보 아님. (b) **문건 자체 결과로 자멸**: 벌크 감수율에서 잡음은 SM2B 의 `√var(N)/⟨N⟩~1/√M→0`(마이크론 흑연 M≫10⁸)로 **거시적 무의미** — 문건이 이미 증명. 잡음이 주역인 유일 국면은 **핵생성 꼬리**(장벽 구동 탈출)인데 그건 **#4 그 자체**. (c) 신규 잡음 파라미터(D/M)·의도적 결정론 kinetics 설계 위반.
- **권고**: 독립 도입 비권장. 확률화가 값을 갖는 유일 자리(핵생성 탈출)는 #4 Kramers 노트가 부록에서 이미 흡수.

## 3. v1.0.23 값어치 종합 — 교과서 정합 vs 비대화 리스크

- **A 등급 0건이 정직한 결론.** 이 클러스터에서 문건은 이미 **닫힌 spinodal 근·cusp 보편지수(3/2)·원자적 Preisach 구조·Kramers-극한 CNT 율·CLT broadening·요동–소산 감수율**을 자력 보유한다. 신규 기법이 줄 수 있는 것은 대부분 **새 물리가 아니라 이미 있는 결과의 교과서 명명·절간 연결**이다. 이는 문건의 성숙도에 대한 긍정 신호이자, "고등수학을 더 넣어 깊이를 키운다"는 기대에 대한 **감가 경고**다.
- **권장 3건(B)은 모두 "명명·연결" 유형**이고 셋 다 **모델 차원 0**이다:
  - **#1 FPT(부록)** = 자족성 향상 + 완화↔핵생성 연결(사용자 "절간 연결" 정면), 본문 무영향이라 최저 위험.
  - **#2 Preisach(분류)** = 브리핑 최우선 기법의 정직한 착지점 — *정합하나 원자적 극한, 연산자 채택은 D*. `h_{η,j}` 에 표준 프레임 이름을 주되(CNT↔γ_j 선례와 동형) 메모리 연산자는 들이지 않음.
  - **#3 cusp** = `(T_c−T)^{3/2}` 의 보편성 조명. 각주 상한, 절 신설 시 팽창.
- **비대화 리스크 지도**: LDT(#4)·중심다양체·Langevin 은 **무거운 기계 대비 payoff 0** 또는 **문건 자체 결과·스코프 벽으로 자멸**. 특히 **경로의존 계열(Preisach 연속밀도·LDT-소산·요동정리)은 [C-92]·SM2 축 C 가 이미 친 스코프 벽**에 정면으로 걸린다 — 이 벽을 넘는 도입은 확정 경계 위반.
- **P3 검수 정합**: 권장 3건 모두 재해석·명명(모델 차원 0, 새 파라미터 0, 무라벨/서술형)이라 P3 2항(전하 보존 중심식)·6항(Ch1↔Ch2)·7항(ver↔Chapter) 무충돌. #1 은 부록(독립 문서)이라 본문 라벨/식번호 재배열 0.
- **집행은 사용자 GO 후.** 실집행 시 CNT_LINK/CLT_DRAFT 와 동형의 **제거 용이 독립 블록**(무라벨/서술형) 원칙 준수. 자산 앵커(제안): `[V23-S4-FPT]`·`[V23-S4-PREISACH]`·`[V23-S4-CUSP]`.
