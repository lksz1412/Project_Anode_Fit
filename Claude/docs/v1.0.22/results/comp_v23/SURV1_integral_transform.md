## 2. 기법별 소절

### 기법 1 — Volterra vs Fredholm 재유도 (본체 #0 을 위한 정식화) · **B**

**문건 자리.** §9 eq:memory–eq:lag(인과 기억) · §7 eq:ensavg(앙상블 평균) · #0(JCP self-consistent 되먹임).

**재유도·판정.** 세 적분구조를 종으로 가르면:
- **§9 lag = Volterra 2종**(인과·1측). eq:memory-Vaxis `r(V)=∫_{-∞}^V e^{-(V-u)/L_V}(dξ_eq/du)du` 는 상한이 평가점 `V`(가변)라 정의상 Volterra. 완화 ODE(eq:Lq)의 적분방정식판이며, **1측성=인과성**이 이 종에 각인돼 있다. eq:reversal 의 충전 커널 반전은 "진행 방향 과거를 훑는다"는 인과 요건을 종 안에서 강제하는 것과 동치 — 이 대응을 명시하면 §9 부호 논증(방향 어기면 비인과)이 **적분방정식 종 언어로 한 줄** 정당화된다.
- **§7 앙상블 = Fredholm 1종**(고정 구간·역산 ill-posed). 문건 scope(ii)가 이미 명시(`A08_REVIEW.md:237,373` 이력).
- **#0 되먹임 = Fredholm 2종**(미지량이 적분 안팎 동시 등장 → ratio-닫힘).

**더하는 것.** "이 문서에는 종이 다른 세 적분방정식이 공존하고, **well-posedness 가 종에서 직접 읽힌다**"(Volterra-2 = 인과·항상 가해; Fredholm-1 = 역산 불안정; Fredholm-2 = ratio-닫힘 대상)는 통일 표. 검수 7항목 ③④(순환 의존성 4분류)와도 정합 — #0 되먹임을 "정의상 implicit(Fredholm-2)"으로 분류하는 근거를 적분방정식 종론으로 보강.

**정직 caveat.** (i) lag 은 §9 가 **이미 정확히 닫음** — Volterra 라 명명해도 **새 해법·새 수치 0**. (ii) 기법 1 이 언급하는 "ratio 보정의 수렴/오차 한계 부등식"은 lag(정확해·오차 없음)이 아니라 **#0 의 Fredholm-2** 에서만 의미 있고, 그 부등식의 실제 유도는 **기법 4**(Neumann/resolvent)가 한다. 곧 기법 1 은 **분류·정식화**까지가 값어치이고, 오차 부등식은 기법 4 로 넘긴다.

**v1.0.23.** 넣기(소). §7 scope(ii) 또는 §9 도입에 **3종 대조 keybox** 1개(3–4행). 모델 차원 0, 비대화 낮음.

### 기법 2 — Laplace/Fourier 변환 → 전달함수 · **A (핵심 추천)**

**문건 자리.** §8–9 동결 `L_V` 합성곱(`_causal_memory_pointwise`) · §7 eq:widthbudget(대칭폭 `w_j` vs 비대칭꼬리 `L_V`) · §6 감수율.

**재유도.** 동결 `L_V` 는 상수계수 LTI 라 lag 은 병진불변 합성곱 `ξ_lag=K∗ξ_eq`, `K(Δ)=(1/L_V)e^{-Δ/L_V}Θ(Δ)`. 변환영역에서 대각화:
```
K̂(ω) = H(ω) = 1/(1 + iωL_V)             (정규화 지수 = 1차 저역통과)
```
peak 모양(eq:peakshape) `r/L_V=(ξ_eq−ξ_lag)/L_V=dξ_j/dV` 를 변환하면
```
(dξ_j/dV)^ = (1−H)/L_V · ξ̂_eq = iωH·ξ̂_eq = H(ω)·(dξ_eq/dV)^
⟹  dQ/dV_apparent(ω) = H(ω) · dQ/dV_eq(ω)          ★
```
곧 **관측 dQ/dV = 평형 dQ/dV 에 1차 저역통과를 통과시킨 것**. 세 귀결:
1. **크기 `|H(ω)|=1/√(1+ω²L_V²)`** = 저역통과(차단 `ω_c=1/L_V`) → 대칭 감쇠·봉우리 낮춤.
2. **위상 `arg H=−arctan(ωL_V)`** = 인과 지연 → **비대칭 skew·봉우리 이동**. §7 이 손으로 가른 "대칭폭(②③) vs 비대칭꼬리(①)"가 **크기 vs 위상**으로 자동 분리된다 — Gaussian(②⊗③)은 **영위상**(대칭), 인과 lag(①)은 **비영위상**(꼬리). 두 축의 분리가 "손 논증"에서 **주파수영역 정리**로 승격.
3. **`L_V→0 ⟹ H→1`** = 평형 종 회수(eq:tail-limit)가 **즉각** — §9 의 지배수렴정리 논증과 동일 결론을 한 줄로.
4. **충전 = 켤레 필터** `H(−ω)=H*(ω)` → eq:reversal 의 "거울"이 곧 켤레 응답(크기 불변·위상 반전).

**검산(python, 재현).** logistic `ξ_eq`(w=1), `L_V=1.5w`, 균일 격자 FFT:
- `‖dQ/dV_app^ − H·dQ/dV_eq^‖/‖dQ/dV_eq^‖ = 1.4×10⁻⁸`(★식 **기계정밀도 정확**).
- 겉보기 봉우리 높이 **0.1955**/w, V-이동 **+1.012 w** → 문건 `fig:reversal` 자체 명기점 **(+1.01w, 0.196)** 과 **소수 3자리 일치**. 곧 전달함수 한 줄이 **문건의 기존 그림 수치를 재생산**한다(자기정합 확인).
- 차단 `ω_c=1/L_V=0.667`, `|H(ω_c)|=1/√2=0.707`(반출력점, 교과서 1차 저역통과 정합).

**더하는 것.** (i) **닫힌형 전달함수** — 봉우리 감쇠·이동을 `H(ω)`로 해석적으로 예측(수치 합성곱 없이). (ii) §7 폭 예산의 **크기/위상 엄밀 근거** — 문서의 가장 중심적인 분리(대칭 w vs 비대칭 L_V)에 주파수영역 정리를 깐다. (iii) **기기응답(instrument-response function) 해석** — lag = 평형 감수율에 씌운 기기 blur, scope(ii)의 Fredholm/DRT 언어와 같은 신호처리 계보로 연결. (iv) #5 Green 함수(`H=𝓕[G]`)·#4 분해핵·#6 정적 감수율(`χ₀=dQ/dV_eq`, `χ(ω)=χ₀H(ω)`)을 **한 box 로 묶는 허브**.

**정직 caveat.** (i) `H(ω)` 는 **동결 `L_V` 상수**를 전제 — 문건이 이미 전이당 상수로 동결(§8 "실현 미분 `∂ln L_q/∂V=0`")하므로 **새 근사 도입 0**, 전달함수는 그 동결 근사를 **그대로 상속**. (ii) **값·모델 불변** — 순수 해석 승격(교육·통일 이득이지 새 자유도 아님). (iii) **비대화 리스크**: full Fourier 절은 과함 → **compact box 1개**(H·|H|·arg H·`L_V→0`·켤레)로 한정. (iv) 검수 7항목 ②(전하 보존식 중심성) **불가침** — `H(ω)`는 이미 결정된 `ξ_eq(V_n)` **출력에만** 작용, 내부 전위 결정(전하 보존식)에는 손대지 않음. 이 점을 box 에 명기해 "OCV 읽기 회귀" 오독 차단.

**v1.0.23.** **넣기(핵심).** §9(eq:tail-limit 뒤) 또는 §7(eq:widthbudget 옆)에 **전달함수 keybox** 1개. 등급 A 근거 = 정확성(1e-8)·자기정합(문건 그림 수치 재생산)·중심 분리 승격·교과서 표준(1차 저역통과 IRF).

### 기법 3 — Wiener–Hopf · **D (기각)**

**문건 자리.** 없음(대응 문제 부재).

**판정.** WH 는 반평면 커널의 **비자명 인수분해** `H=H_+H_-` 로 반직선 위 적분방정식을 푸는 기법 — 커널이 초등적으로 닫히지 않을 때 쓴다. 여기서는 (i) 커널이 **지수(초등)**라 §9 가 적분인자로 **이미 닫음**, (ii) 단측성(인과, `u∈(-∞,V]`)은 1계 선형 ODE 가 **자명 처리**(과도항 지수 소멸), (iii) WH 가 진가를 낼 유일 자리인 **인과 deconvolution**(꼬리에서 비지수 기억 커널 역추정)은 문건이 **명시 금지**(forward-only·역산 ill-posed, scope(ii)). 곧 **WH 가 붙을 문제 자체가 존재하지 않는다.**

**정직 caveat.** WH 를 도입하면 초등 커널에 무거운 반평면 기계를 얹는 **교과서 이름 붙이기**가 되어 비대화·오독(역문제가 열린 것처럼 보임) 리스크만 커진다. **명시 기각** — 단, "인과 단측성은 지수 커널이라 WH 없이 초등적으로 닫힘"이라는 **1줄 배제 근거**는 #1 표 안에 남겨 완결성 표시.

### 기법 4 — 분해핵(degenerate/separable)·Neumann 급수·resolvent · **B**

**문건 자리.** §9 지수 커널(분해핵) · #0 ratio 보정(Neumann/resolvent).

**재유도·판정 — 두 부분.**
- **(a) 분해핵 = §9 초등 가해성의 구조 설명.** `e^{-(V-u)/L_V}=e^{-V/L_V}·e^{u/L_V}` 는 **rank-1 분리 커널**의 정본. Volterra 분해핵은 적분방정식을 **유한계 ODE 로 붕괴**시키므로, §9 가 eq:Lq(1계 ODE)↔eq:lag(적분)을 자유로이 오간 것이 **"커널이 rank-1 이라"** 라는 한 마디로 구조화된다. 교육 이득 있음(왜 초등적으로 풀리나), 분량 1줄.
- **(b) Neumann/resolvent = #0 오차 골격.** Fredholm-2 `r=g+λ∫K r` 의 resolvent 전개는 ratio-닫힘 근사에 **명시 오차 부등식**을 준다: `‖r−r_N‖ ≤ ‖λK‖^{N+1}/(1−‖λK‖)·‖g‖`(‖λK‖<1). 이것이 **기법 1 이 가리킨 "ratio 보정 수렴/오차 한계"의 실제 거처**다. #0(JCP 기법)이 확정 본체이므로, 그 ratio 치환의 **수렴 조건·1차 보정항**을 resolvent 로 적으면 본체에 엄밀 뼈대를 더한다.

**정직 caveat.** (i) **lag 에는 값어치 없음** — Volterra 커널은 준멱영(spectral radius 0)이라 Neumann 이 **무조건 수렴**, 새 정보 0. (ii) (b)의 실값은 **#0 쪽**이고 #0 은 P3-5 **확인 대기** — 사용자 JCP 원문의 ratio-닫힘이 실제로 어떤 norm·어떤 `‖K‖`에서 닫히는지 **확인 전에는 부등식 상수를 못 박음**. 곧 (b)는 "#0 확정 후 보강 후보"로 대기.

**v1.0.23.** (a) 넣기(소): §9 eq:memory 각주 1줄("지수=rank-1 분해핵 → ODE 붕괴"). (b) **보류**(#0/ref.6·7 확인 후) — 지금 넣으면 미확인 본체에 오차 상수를 선점하는 것.

### 기법 5 — Green 함수(완화 연산자) · **C (독립추가 비권장·#2 로 흡수)**

**문건 자리.** §9 eq:intfactor·eq:memory.

**판정.** 기억 커널 `(1/L_V)e^{-(V-u)/L_V}Θ(V-u)` 는 완화 연산자 `𝓛=(d/dV+1/L_V)` 의 **Green 함수**(`𝓛G=δ`). lag 해 `ξ_lag=G∗source` 는 Green 함수 해다 — **참이나**, §9 의 **적분인자 해(eq:intfactor)가 곧 이 Green 함수**라 순수 재명명이다. 1계 선형 ODE 에서 "적분인자"와 "Green 함수"는 같은 대상.

**정직 caveat.** 독립 신규 0. 유일한 값어치는 **연결 어휘** — #2 전달함수(`H(ω)=𝓕[G]`, 곧 전달함수는 Green 함수의 변환)와 #4 분해핵(`G`=rank-1)을 잇는 다리. 따라서 **독립 절 추가는 비권장**, #2 box 안에 **"이 커널은 완화 연산자의 Green 함수이고 `H=𝓕[G]`" 1절**로 흡수하는 것이 정직.

**v1.0.23.** 흡수(추가 분량 ≈0). 독립 추가 시 중복·비대화만 증가.

### 기법 6 — 요동–소산(Kubo)/선형응답 형식화 · **B (정적 1줄 · 동적은 경고)**

**문건 자리.** §6 SM2-A(`dQ/dV_eq=e²∂⟨N⟩/∂μ=(F²/N_ART)var(N)`).

**재유도·판정 — 정적/동적 분리.**
- **정적(수용).** SM2-A 는 이미 **정적 요동–소산 관계**(등온 감수율 = 분산). 이를 **"Kubo 선형응답의 ω→0 정적 극한"**으로 명명하는 것은 교과서 정합·1줄. `∂⟨N⟩/∂μ=β var(N)`는 정적 감수율 = 정적 구조인자(=적분 상관)의 표준형.
- **동적(경고).** 완화 과정의 동적 감수율은 Debye 형 `χ(ω)=χ₀/(1+iωτ)`. 여기 `χ₀=dQ/dV_eq`(§6), 그리고 `1/(1+iωτ)`가 **#2 의 `H(ω)`와 형식적으로 정확히 일치** → `χ(ω)=χ₀·H(ω)=dQ/dV_apparent(ω)`. 곧 "겉보기 dQ/dV = 동적 감수율"이라는 매혹적 통일이 **수학적으로는** 성립.

**정직 caveat(핵심 — 물리 가정 충돌).** 동적 승격은 **FDT 의 적용 요건을 위반**한다:
- FDT/Kubo 는 **평형 요동 ↔ 미소 섭동에 대한 선형응답**을 잇는다. 그러나 문건의 동역학은 **유한 전류가 구동하는 결정론적 kinetic lag**(`dξ/dt=k(ξ_eq−ξ)`, eq:Lq)이지 평형 요동의 완화가 아니다.
- `L_V∝|I|`(§8)이라 **평형(`|I|→0`)에서 `L_V→0`**. 반면 Kubo 상관시간 `τ`는 평형에서 **유한**. 곧 `L_V↔τ` 대응은 **평형 상관시간이 아님** — 구동 프로토콜의 비평형 지연을 평형 상관시간으로 부르는 **범주 오류**.
- 따라서 `χ(ω)=χ₀H(ω)`의 Debye 일치는 **형식적 유비**(같은 1극 필터 대수)일 뿐, 그 `χ(ω)`를 **진짜 평형 FDT 동적 감수율로 읽으면 물리 가정 충돌**(검수 7항목 ④의 "물리 가정 충돌" 유형에 정확히 해당).

**v1.0.23.** 정적 1줄 넣기(§6 SM2-A 에 "= ω→0 Kubo 정적 극한" 명명). 동적 `χ(ω)`는 **넣더라도 #2 box 의 정직 각주로만** — "구동 lag 이라 평형 FDT 아님·Debye 일치는 형식 유비"를 **반드시 병기**. 무경고 도입은 확정 경계 위반.
