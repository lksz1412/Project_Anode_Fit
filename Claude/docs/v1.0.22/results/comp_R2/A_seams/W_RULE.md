# W_RULE — "본 문건/이 문건" 어휘 일괄 규칙안 (v1.0.22 R2 · O-A 초안)

> 지위: `comp_R2/A_seams/` 신규 초안. 마스터 소스 미수정. 대상 = `R1B_SWEEP_LIST.md` 의 W 29건.
> W 처분(정본): "장 단독 배포는 유효하나 병합(단일 문건) 대비 '본 장/본서' 정합 검토" — 신Ch1분(20건)=R2 집행, 신Ch2분(9건, `ch1_sec11~17`)=R3 집행. 본 규칙은 29건 **전체의 일괄 규칙안**이며 R2/R3 공용.

## 0. 왜 W 인가 (문제 정의)

"본 문건/이 문건" 은 구 v1.0.21 단일 문건 시절의 자기지칭이다. v1.0.22 는 활물질별 3장으로 분리 컴파일되므로:
- 장 단독 PDF 로 읽으면 "이 문건" = 그 장 = 여전히 유효(오독 없음).
- 그러나 병합 빌드(단일 문건, 사용자 별도 세션)에서는 "이 문건" 이 3장 전체를 가리켜 **스코프가 뭉개진다**.
- 따라서 **지칭 스코프를 명시**하는 일괄 치환이 필요하다. T1/T2/T3 와 달리 라벨 전환이 아니라 **어휘 스코프 명확화**다.

## 4. 재현 선언 3곳 — 의미 보존 판정

대상: `ch1_sec00:19`·`ch1_sec10:126`·`ch1_sec18:16` (+ 동종 `ch1_sec10:18`).
원 의도 = **"이 문서 하나만 보면 곡선 재현 코드를 짤 수 있다"**(자기완결 = 외부 의존 0).

**판정 = 동치(원 의도 보존). "본 장" 적용 가.** 근거:
1. 곡선 재현에 필요한 전부가 신 Ch1 안에 있다 — 파라미터표 `tab:staging`(§10)·유도(Part I §1~10)·평형식(Part 0 §2)·6단계 절차(§18 keybox, `eq:n0map`~`eq:sum` 전부 Ch1 라벨). 신 Ch2(LCO)·Ch3 를 참조하지 않는다.
2. `ch1_sec10:159~162` verifybox 가 "본 예제에 쓰인 것은 `tab:staging` 와 식 4개뿐 --- 다른 입력이 없으므로" 라고 이미 자기완결을 못박음. 가역 발열 예제만 "Part T 의 계산 예제(`\S\ref{ssec:worked}`)" 로 넘기는데, 이는 **곡선 재현이 아니라 열특성**이라 재현 선언 스코프 밖. → 곡선 재현은 여전히 장 자기완결.
3. 따라서 "이 문건만 보고 곡선 재현" → "본 장만 보고 곡선 재현" 은 스코프를 좁히지도 넓히지도 않는다(구 문건 = 흑연 곡선 문건 ≈ 신 Ch1 의 곡선 파트, 재현 대상 동일).

**주의(마스터 재량 경계)**: 만약 마스터가 "본 장" 이 Part T(열특성)까지 포함해 "곡선만" 이라는 원 뉘앙스를 흐린다고 판단하면, 대안은 "본 장 Part I 만 보고" 또는 원문 유지. O-A 권고 = **"본 장"**(뉘앙스 훼손 없음 — 재현 대상이 "곡선" 으로 문장에 이미 명시됨). 브리핑 중단조건("어색하면 원문 유지")대로 최종은 마스터 판단.
