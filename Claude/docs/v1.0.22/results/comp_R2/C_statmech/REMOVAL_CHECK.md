## 1. CLT 블록 (`CLT_DRAFT.tex`) — §7.2 (iii-b), eq:ensavg 직후

### 1.1 제거 용이성

| 점검 | 결과 | 근거 |
|---|---|---|
| **참조 고아** (블록 삭제 시 깨지는 `\ref`/`\eqref`) | **0** | 블록이 **참조하는** 라벨(`eq:ensavg`·`eq:widthbudget`·`sec:broadening-scope`)은 전부 **기존**이라 블록 없이도 유효. 블록이 **신설하는** 라벨(`eq:clt-bell`·`eq:clt-var`)은 **블록 내부에서만** 등장 — 본문 다른 곳에서 이 둘을 참조하지 않음. ⇒ 삭제해도 고아 참조 0. |
| **논지 공백** (삭제 시 본문 흐름 단절) | **0** | 삭제하면 §7.2 는 블록 삽입 **이전 상태** 로 정확히 복귀: `eq:ensavg` → (현행 90행) "곧 단일 입자 응답… forward 합성" 문단으로 연속. 세 출처 예산·forward-only·`eq:widthbudget` 분산 가법(기존의 "합성곱 항등식" 정당화)은 **블록과 독립**으로 모두 성립. CLT 블록은 ③항 $\sigma_\eta^2$ 의 **형상 근거**만 얹은 순증분. |
| **그림 의존** | **0(권고 판형)** | `fig:widthbudget` 캡션 승격은 **제안만**(미집행). 캡션 현행 "Gaussian 예시"는 블록 유무와 무관하게 그대로 성립 → 블록 삭제해도 캡션 무영향. |
| **식번호 재배열** | 판형 (b) **0** / 판형 (a) 경미 | ★**권고 판형 (b) bgbox** 는 식이 **무라벨(unnumbered)** 이라 `eq:widthbudget` 이하 식번호 재배열 **0**. 판형 (a)는 신규 번호식 2개가 이후 식번호를 한 칸씩 밀지만 **라벨 참조는 자동 보정**되어 깨지지 않음(표시 번호만 이동). ⇒ 제거 용이성 관점에서 (b) 우세. |

**결론:** 판형 (b) 선택 시 CLT 블록은 **완전 독립·통삭제 가능**(본문·식번호·그림 무영향).

### 1.2 혼란 가중 리스크

- **선행 지식:** 중심극한정리·Lindeberg 조건 — 학부 확률론 표준 정리라 무인용 관용(rubric). 독자 부담 낮음. 블록이 CLT 자체를 정의하지 않고 "다수 독립 원천의 합 → 종형"이라는 **결론만** 부르므로 자족적.
- **Part 0 중복 여부 — 표면 유사 있음 / 내용 중복 없음:**
  - Part 0(`ch1_sec02a` `eq:sm-flucres`·`ch1_sec02b` `eq:sm-mc-fluc`)에 "분산 가법"이 이미 있음: `var(N)=Σ_j M_j θ_j(1-θ_j)` = **자리 점유 분산**의 클래스 합 → **입자수 감수율** `∂⟨N⟩/∂μ=β·var(N)`(유일근 보증). 이는 후보 (i)/축A 영역.
  - CLT 블록의 분산 가법 `σ_η²=Σ_k σ_{η_k}²` 은 **독립 η-원천 분산**의 합 → **겉보기 이동 산포의 형상**(broadening). **대상이 다름**(자리 점유 vs η-원천), **물리량이 다름**(전하 감수율 vs 폭), **자리가 다름**(Part 0 대정준 반전 vs §7.2 broadening).
  - 공유되는 것은 "독립 합의 분산은 가법"이라는 표준 사실뿐 — **내용 중복 아님**. Part 0 에는 CLT·Gaussian 형상이 **전무**(정독·grep 확인: 중심극한/CLT/Gaussian/정규분포/종형 0건).
  - **리스크:** 독자가 두 "분산 가법"을 혼동할 여지. **완화:** 블록이 $\sigma_\eta^2$ 를 명시적으로 "$\eta$-원천 / `eq:widthbudget` ③항"에 국한해 서술하므로 자리 점유 분산과 혼선 소지 낮음. 잔여 리스크 **낮음**.
- **4가드 각인:** forward-only·흑연 두-상 국한·비-크기 전용·②≠Gaussian 을 블록 안에 모두 담아, 삭제 안 해도 (iii-b) 확정 자산(역산 금지 등)을 **훼손하지 않음**.

## 2. CNT 링크 (`CNT_LINK_DRAFT.tex`) — §4.3 (sec:hys-branch), γ_j 지위 문단 직후

### 2.1 제거 용이성

| 점검 | 결과 | 근거 |
|---|---|---|
| **참조 고아** | **0** | 부록이 마스터에 **미포함**(독립 컴파일)이라 링크는 **서술형 참조**(부록 명칭·절 제목 텍스트)로 작성 — `\ref`/xr **미사용**. 신규 라벨·신규 식 **0**. ⇒ 삭제해도 깨질 링크 자체가 없음. 유일한 `\S\ref{sec:hys}` 는 같은 §4 자기 참조(기존 라벨)라 무관. |
| **논지 공백** | **0** | 삭제하면 §4.3 은 현행 그대로: γ_j∈[0,1] 지위(spinodal 상한 vs 실측 안쪽, "국소 요동·불균일 핵생성으로 가지 이탈")가 **이미 산문으로 완결**(현행 247–251행). CNT 링크는 그 산문에 **미시 근거(exp 억제)** 만 연결한 순증분 — 삭제 시 근거만 빠지고 논지는 온전. |
| **부록 측 영향** | **0** | 부록 "본문과의 연결" 절(app:link)이 부록→§4 방향 서술을 **독립적으로** 이미 보유. 본 링크는 §4→부록 **반대 방향** 다리라, 삭제해도 부록 서술 무영향(양방향 독립). |

**결론:** CNT 링크는 **완전 독립·통삭제 가능**(서술형이라 잔존 참조 0).

### 2.2 혼란 가중 리스크

- **선행 지식:** 고전 핵생성 이론(ΔG*·임계핵·exp 억제) — 부록에 **자족 유도 완비**. 링크는 결과(ΔG*·exp 억제)만 부르고 **재유도하지 않음** → 본문 독자 부담 최소. 부록 미열람 독자에게도 "핵생성 장벽이 exp 로 억제 → 과주행"의 정성 그림은 자족적.
- **Part 0 중복 여부 — 없음:** Part 0·본문 §4 에 핵생성/CNT 서술 **전무**. 겹침은 **부록**과의 것(의도된 다리 — 중복이 아니라 연결).
- **기호 함정 2건 — 링크 안에서 명시 처리:**
  1. **γ vs γ_j:** 부록 γ = 계면 에너지[J/m²] ≠ §4 분기 축소 인자 γ_j. 링크가 γ³/(Δg_v²) 형태식을 **부르지 않고** ΔG*·exp(-ΔG*/k_BT) 만 인용해 충돌 회피 + 말미에 "별개 기호" 명시(§7 이미 같은 구분 보유).
  2. **ξ 배향:** 부록 ξ=Li 점유(=본문 θ) vs 본문 ξ=진행률(1-θ), 여집합. 링크가 ξ-의존식을 **부르지 않아** 부호 함정 원천 제거 + 말미에 여집합 관계 명시.
  - 잔여 리스크 **낮음**(P3 검수 1·7항 기호 일관성 가드 충족).
