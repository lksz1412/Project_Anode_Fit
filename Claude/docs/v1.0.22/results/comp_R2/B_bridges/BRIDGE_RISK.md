# BRIDGE_RISK — 인용 다리(흑연분) 충돌·중복 점검표 (v1.0.22 R2 · O-B)

> 각 다리가 기존 서술과 충돌할 위험을 세 축으로 점검: **기호 충돌** · **중복 서술** · **자산 앵커 인접**. 마스터 체리픽 판정용. 등급 = 상(집행 전 처리 필요) / 중(집행 시 유의) / 하(경미).

## 0. 전역 점검(게이트 대응)
- **우리 기호 유지**: 8개 스니펫 전건 — 본문 삽입 서술문은 우리 기호(𝒜_j·χ·U_j·ξ·U_oc·V_n·ΔS·S_vib 등), 논문 기호(α·η·E₀·x·entropic potential 등)는 **대응표 우측 열에만**. 게이트 "우리 기호 체계 유지" 충족.
- **원장 V1 외 인용 0**: 인용 키 = bazant2013·dreyer2010·dreyer2011·mckinnon1983·weppner_huggins1977·baek_pilon2022·bernardi1985·allart2018·reynier2003·jpcc2021 — 전건 ch1v22_bib 수록 V1(승계 원장). 신규 키 0.
- **신규 라벨**: `eq:br-bazant2013-1` 1건(제안) — br- 접두라 기존 eq: 라벨과 충돌 없음. equation 카운터 +1(1.x 번호 1개 증가) → 마스터 빌드 시 후속 참조 영향 없음(신규 라벨만 참조).
- **삽입 위치 8곳 전건 mid-section** — 절말 자산 블록(sec05:378·sec07:307·sec02a:365·sec03:96·sec09:42)과 물리적 비인접. 예외 1건(ch1_sec01:219 [V21-Q5-02]) = §4 항 참조.

## 2. 상급 위험 처리 권고(마스터 판정용)

- **[R-1 · weppner+baek_pilon 스택·앵커 인접]**(상). 두 다리(#4·#5)가 같은 측정원리 bgbox(204–218) 직후에 연달아 삽입되면 (a) [V21-Q5-02] 자산 주석(219)과 물리적으로 끼이고 (b) srcbox 2개가 bgbox 뒤에 스택된다. **권고**: 두 다리를 **하나의 "측정 다리" srcbox 로 병합**(항 (i) weppner·(ii) baek_pilon 소구획)하고, **[V21-Q5-02] 주석(219) 뒤에 삽입**해 앵커를 bgbox 에 붙여 둔다. 또는 bgbox 항(i)/(ii) 내부에 대응표만 in-line 증축.

- **[R-2 · bernardi ③ 중복]**(상). 다리 ③(혼합·상변화 소거 논리)이 현행 22–25행과 거의 동일. **권고**: 집행 시 ③을 재서술하지 말고 **항별 대응표(①)+방법(②)만 채택**, ③은 "현행 22–25행 참조" 포인터로 축약(스니펫은 이미 "취지 동일" 명기 — 마스터가 문장 중복만 제거).

- **[R-3 · baek_pilon 대응표 중복]**(상). 대응표가 bgbox 항(ii) 나열의 재판. **권고**: 대응표를 **가정차(tier B 1차병기 규약, 이것이 유일 신규분)** 중심으로 트림하거나, 항(ii) 문장에 각주로 흡수. load-bearing 판정 자체는 유지(중), 다리 분량은 하향 가능.

- **[R-4 · η 글자 중복]**(중, bazant). 대응표의 η(Bazant 과전위)와 본문 §7 의 η(apparent-U 국소 과전위, U_app=U_j+η)가 같은 글자. §5(다리 위치)와 §7(η 사용처)이 떨어져 있어 즉시 충돌은 아니나, **권고**: 표 좌측 캡션에 "논문 η(과전위) --- 본문 §7 의 η(국소 과전위)와 무관" half-line 가드 추가(스니펫 채택 시 마스터 재량).

- **[R-5 · mckinnon x 글자 중복]**(중). 표의 x(자리 점유율)가 본문 x̄(조성)와 글자 공유. **권고**: 표에서 "삽입 분율 x(자리 점유율, 본문 θ/ξ 에 해당)"로 괄호 가드(스니펫에 일부 반영 — 마스터 확인).

## 3. 물리 가드(가정차 정확성 — 집행 전 마스터 재확인 권고)
- bazant2013 ③: 검증완료(식[29] γ‡ 구조 확인). 안전.
- dreyer/mckinnon/bernardi: ②(방법)는 검증, ③(가정차)는 우리 본문 전제(공통 μ·q(T)·저율 소거)와 대비로 물리적으로 타당하나, **원 논문 식 미재현**(★확인 필요 3건)이므로 "논문이 그렇게 유도한다"는 단정 대신 **결과/항 대응 수준** 어법 유지 확인(스니펫은 준수).
- baek_pilon/allart/reynier: 논문 식 미재현·방법은 제목/주제로 검증. 수치(예: Allart +3~4 mV/K, ch2_sec02:139)는 **다리에 재현하지 않음**(원문 값 미확인) — 기존 본문 tier 인용만 유지.

## 4. 결론
- 즉시 처리 권고 = **상 3건(R-1·R-2·R-3, 전부 중복/스택 축소)** + 중 2건(R-4·R-5 글자 가드). 전부 **분량·중복** 조정이지 물리·서지 결함 아님.
