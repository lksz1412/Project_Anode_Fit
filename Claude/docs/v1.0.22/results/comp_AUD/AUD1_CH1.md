## 상세 (재유도 주요분 — 적대적 재검산)

### C-043 SM2-A 가드 — 핵심 반례 검증
평균장 격자기체 `μ=μ⁰+k_BT ln[θ/(1−θ)]+ω(1−2θ)` 에서
`var(N)=k_BT ∂⟨N⟩/∂μ = M θ(1−θ)/[1−2ωθ(1−θ)/k_BT]`.
- Ω=0: `var=Mθ(1−θ)` (독립-자리형 정확).
- Ω≠0: 분모≠1 → 독립-자리형은 **근사**. 조건은 n_j=1 이 아니라 Ω=0. (현행 정정 정확)
- **반례**(task 요구): θ=½·Ω>2RT 면 `1−ω/(2k_BT)<0` → 형식상 var<0 = spinodal 불안정. 곧 "n_j=1 에서 정확"은 반례 존재 → 원 서술 오류가 실재했고 정정이 맞음.
- FDT 등식 `var(N)=k_BT∂⟨N⟩/∂μ` 자체는 대정준 일반(현행 (iii) 서술 정확).
- 배율: w_j=n_jRT/F → 폭 ∝n_j·높이 Q/(4w)∝1/n_j. "폭 n_j·높이 1/n_j" 정정 방향 맞음.

### C-044 eq:kuniv — 이중계산 폐색 재추적
현행 exponent bare `−(ΔG_{a,j}−χ_d𝒜_j)/RT`, 완전 `𝒜_j=sF(V−U_j)−Ω_j(1−2ξ_j)`. 심층꼬리(ξ→1) `𝒜_j→𝒜+Ω_j`.
`ΔG_{a,j}−χ_d(𝒜+Ω_j) = [ΔH_a−χ_dΩ_j]−TΔS_a−χ_d𝒜 = ΔH_a^eff−TΔS_a−χ_d𝒜 = ΔG_a^eff−χ_d𝒜` (eq:dHeff: ΔH_a^eff=ΔH_a−χ_dΩ, ΔS_a 불변).
→ sec08 equation*(77-84)의 두 표기 등가 성립. bare+완전 인쇄형이 eff+이상 평가형과 **대수 항등** = 이중계산 0. eq:Lqfull/eq:LV 무결.
- 미세 관찰(비-오류): 역인자를 eq:kuniv 는 완전 `(1+e^{−𝒜_j/RT})`, eq:Lqfull 은 이상 `(1+e^{−𝒜/RT})` 로 씀. 컷 동결 근사(§8.4 line 100-102 명시)이며 𝒜≳4RT 에서 역인자≈1.0x 라 Ω 몫 낙차 무시가능. 전방(주항) 이중계산과 무관.

### C-045 A13 trio — 수치 재계산 (python 확인)
- `S_vib/R(u=2.3478)=0.34857` vs `1+ln(298.15/700)=0.14652` → **비 2.379 ≈ 2.4배**. 마스터 0.3486/0.1465 재현. "동결 기준=닫힌형" 정정이 캡션(●=0.35R)과 정합.

- fig(b) 마커 −3.738/0/+3.700/+9.138 μV/K = 캡션 −3.74/0/+3.70/+9.14 정확. (참고: 고전극한 편차 13.36>준양자 9.14 → "u→0 최대" 서술 수치 정합)

### C-046 ② TST 전치인자 — 고온극한 재유도
반응물 N모드·활성복합체 N−1모드(반응좌표 1개 제거). q_vib→k_BT/(hν).
`q‡/q_R = (h/k_BT)·∏_{1..N}ν^R/∏_{1..N−1}ν^‡ ∝ 1/T`.
`(k_BT/h)(q‡/q_R) → ∏ν^R/∏ν^‡` (T·m* 무관 유한 = Vineyard 고전 시도빈도). 현행 (ii) 문면과 정확 일치. 원 "비가 고전값 수렴·ΔS_a 유한"은 `ΔS_a=R ln(q‡/q_R)→−∞` 라 오류였고 정정이 맞음.

### SM2 3본 무라벨 식 = 참조 식 항등 확인
- **SM2-A**(sec06): 두 무라벨식 모두 재유도 성립. eq:sm-mc-fluc(β·var), eq:sm-mc-occ(⟨N⟩=ΣM_jθ_j) 실재·일치.
- **SM2-B**(sec02b): var(N)=ΣM_jθ(1−θ)=O(M)·⟨N⟩=O(M) → 상대요동 1/√M. C-043 의 독립-자리(Ω=0)형과 상호 정합(가드 동일 Ω>2RT).
