## 배경: 블렌드 음극의 이론 골격 (R5에서 저작)

마스터플랜 §7 R5 에서:
> "혼합 음극 = 같은 전위 손잡이(μ 하나)를 공유하는 두 host의 자리 클래스 합집합 — 대정준 반전식이 $\sum_{\mathrm{host}}\sum_j Q_j\xi_{\eq,j}(U_{\mathrm{oc}},T)=Q\bar x$, $Q=Q_{\mathrm{gr}}+Q_{\mathrm{Si}}$, $f_{\mathrm{Si}}=Q_{\mathrm{Si}}/Q$"

즉 **두 활물질이 같은 전위(내부 전위 V_n)에서 동시에 반응**한다는 가정입니다. 이것이 성립하려면:
1. 두 음극의 **전위 곡선(OCV)이 f_Si 비율에서 얼마나 겹치는가**
2. **리튬 저장소 공유 (호스트 자리 클래스 합집합)의 물리**
3. **용량 배분 (Q_gr : Q_Si)의 f_Si 의존**

이 3가지가 실측으로 검증되어야 합니다.

## 검색 전략

흑연+Si 블렌드의 전위 정렬·용량을 직접 측정한 문헌은:

1. **측정 대상**: Gr-Si 물리 혼합, 또는 복합체 음극; f_Si 또는 Si 질량분율 명시
2. **측정 데이터**:
   - **갈바노스태틱 충방전** (potential vs time/capacity) → dQ/dV 또는 dV/dQ
   - **전위 곡선 비교**: 순 흑연 vs 순 Si vs 블렌드 (f_Si = 10%, 30%, 50% 등)
   - **가역 용량 배분** (Q_gr vs Q_Si vs 블렌드)
   - **히스테리시스 크기** (Gr: ~수십 mV, Si: ~수백 mV)
3. **보고 형태**: dQ/dV 곡선 그림 또는 표

## 현행 상태 정리

### 이론적 필요성 (R5에서)

블렌드 공통-μ 이론을 실증하려면 **적어도 다음 중 하나**:
- (A) 실측 dQ/dV: 흑연, Si, 블렌드(f_Si=10/20/30/50%) 의 전위 곡선 overlay → 공통 전위 영역 확인
- (B) 이론 계산(DFT 또는 대정준 시뮬레이션): 두 host의 화학포텐셜 일치 조건(f_Si, T, x 함수)
- (C) 혼합열역학: 활동도 계수(interaction parameter) 실측 → 용량 배분 예측

### 현재까지 확보된 것
- **Si 단일 전위**: DIRECTION_SI_LCO_REPORT §1의 17건 문헌 (경사, 평균 0~0.5 V, 히스테리시스 수백 mV)
- **흑연 전위**: v1.0.21 Ch1 (plateau, 평균 0.1~0.2 V, 히스테리시스 ~수십 mV)
- **차이점**: 전위 곡선 형태·히스테리시스 크기 완전히 다름 → **공통 전위 영역이 얼마나 되는가?**

### 미확보
- **Gr-Si 블렌드의 실측 dQ/dV** (f_Si 비율별)
- **용량 배분의 f_Si 의존성**
- **공통 전위 영역의 f_Si 의존**
