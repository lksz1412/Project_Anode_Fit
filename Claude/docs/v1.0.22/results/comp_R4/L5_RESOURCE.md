## 재소싱 대안 1: reynier2004 (L2 후보 C1)

### 서지·확인 상태

| 필드 | 정보 | tier |
|---|---|---|
| **서지** | Y. Reynier et al., *Phys. Rev. B* 70, 174304 (2004); 10.1103/PhysRevB.70.174304 | A |
| **측정법** | 평형전압 온도 의존 (dU_oc/dT) → ΔS_rxn = F·dU/dT | A |
| **실측 대상** | LCO O3 x = 0.5 < x ≤ 1.0 전 구간 | A |
| **질서상 명기** | x = ½ **및** x = **5/6** 중간상 명시 | A |
| **엔트로피 분해** | Config(Li·빈자리) + phonon(baseline) + electronic(MIT 서) | A |

### 원문에서의 수치 · 가능한 대응

reynier2004 는 **x = ½ 와 x = 5/6 의 두 질서상**을 명기합니다. 현행 본문의 T2/T3 와의 대응:

| 본문 슬롯 | 현행 x | 조성 근거 | reynier2004 대응 | 재소싱 가능성 |
|---|---|---|---|---|
| **T2 (MIT 위, ~4.05 V)** | ≈0.55 | 초기값 | **x=½** 질서상 직접 제시 | ✓ **가능**(x=½ 사용) |
| **T3 (MIT 아래, ~4.17–4.20 V)** | ≈0.48 | 초기값 | **x=5/6**(≈0.83) 질서상 제시 | ⚠ **부분*(x=5/6으로 갱신 필요) |

### 재소싱 판정

**reynier2004 로부터:**
- **T2 config ΔS** = x=½ 질서상의 config 엔트로피 성분 (원문 수치 확인 필요 — 전문 접근)
- **T3 config ΔS** = x=5/6(current 본문의 x=⅔ 대신) 질서상의 config 엔트로피 성분

**효과:**
- T2 슬롯: x=½ 조성 **일치** → tier-C → **tier-A 승격 가능**
- T3 슬롯: x=⅔ → x=5/6 **갱신 필요** (물리적으로 실측 질서상과 일치)

**단점:** 원문 전자접근 필요(현재 ASU pure 초록만 확인) → **구체 수치는 R3 마스터가 전문 대조 필요.**

## 재소싱 대안 2: EES-entropymetry2020 (L2 후보 C3)

### 서지·확인 상태

| 필드 | 정보 | tier |
|---|---|---|
| **서지** | H. J. Kim et al., *Energy Environ. Sci.* 13, 286–296 (2020); 10.1039/C9EE02964H | A(서명) |
| **측정법** | Entropymetry (electrochemical impedance × temperature) → ΔS(x) 프로파일 | A |
| **실측 대상** | LCO O3 ΔS(x) 전 구간 | A |
| **핵심 발견** | **x≈0.5–0.6 에서 ΔS 기울기 반전** = monoclinic 중간상 order–disorder 전이 | A(서명) |
| **조성-온도 의존** | Ni 도핑으로 질서 강도 변조 | A |

### 원문에서의 수치 · 가능한 대응

EES 초록(403 미접근)에서:
- **qualitative**: ΔS 기울기 반전 (order–disorder 서명) 명확
- **quantitative**: 정량 ΔS 값 **미확보**(전문 필요)

### 재소싱 판정

**EES-entropymetry2020 로부터:**
- **물리 근거**: monoclinic order–disorder 의 config ΔS 변화를 1차 실측으로 입증 ✓
- **수치 값**: 초록 미접근 → 원문 전자 또는 별쇄 필요

**효과:**
- T2/T3 의 config ΔS 물리기원(질서/무질서)을 **최신 실측으로 근거 제시** 가능
- 다만 **정량값은 여전히 미확보** 가능성 (그렇다면 tier-B/C 유지)

**단점:** 정량 ΔS 값이 초록에 미표출 가능 → R3 마스터가 EES 전문 직접 대조.
