## 보고 양식 (기계 적용 가능하게)
발견별: `| ID | 파일:행 | 유형(보완/논리/설명/수식화) | 등급 H/M/L | 현행(축자 — 복사) | 제안(완성 LaTeX) | 근거 |`
- **현행 열은 원문 축자 복사 의무**(의역 금지 — 마스터가 기계 매칭). 여러 행 걸치면 개행 포함 그대로.
- H=논리/물리 오류·오귀속, M=의미·이해 실질 개선, L=문체. 등급별 절 정리 + 말미 4-tier(확정/추정/미검증) + "무발견 축"도 명시(검토했으나 문제 없음).
- 제안이 기존 자산·수식을 삭제하는 경우 금지 — 대체·보강만.
