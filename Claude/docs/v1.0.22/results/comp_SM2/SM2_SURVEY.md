## 1. DIRECTION 보류/미집행 후보의 현 시점 재평가

### (v) 단상 폭 `w_eff(Ω)=(RT/F)(1-Ω/2RT)` — **보류 유지(재확인)**
- **재평가**: 수식은 여전히 옳다(정규용액 등온선 `eq:Veq_BW` 의 θ=½ 음함수 미분). 그러나 (a) 결과는 §5.2·Part T 파생 C(`ch2_sec05` `ssec:weff`)에 **산문으로 이미 존재**("`(1-Ω/2RT)` 배 좁아진 유효 폭, 그 값도 평형이 결정")하고, (b) 정책 충돌 위험이 **오히려 굳어졌다** — Part T 파생 C 의 `warnbox`(`[C-87]`)와 자산 `[C-2]`가 **두-상** `w_eff(Ω)=w(1-Ω/2RT)` narrowing 재도입을 명시 금지하는데, 단상 유도식은 **글자까지 동일**해 독립 박스화 시 두-상 금지식과 시각 구별이 안 된다.
- **교육 이득**: 낮음(이미 산문 존재). **모델 차원**: 0. **분량**: 각주 4–6행. **삽입처**: §5.2 단상 문단 각주. **혼란 리스크**: **최상**(C-2/C-87 재도입 오독).
- **판정**: **집행 비권장(현행 산문 유지).** DIRECTION 5순위 그대로.

### 축 A(거시 요동–응답 = dQ/dV 감수율) — **절반만 집행됨 → 신규 후보 SM2-A 로 완성**
- DIRECTION 은 축 A 를 후보 (i) 소절에 verifybox 로 병합 권고했으나, **실제 §2.5 verifybox 는 유일근 증명(`∂⟨N⟩/∂μ=β var(N)>0`)만** 담았다. "각 전이 종의 합 = ∝var(N), 곡선 전체 = 요동 지문"이라는 **감수율(dQ/dV↔요동) 읽기는 미집행**이다(§6 `eq:eqpeak` 와 `var(N)` 을 잇는 항등식이 본문 어디에도 없음).
- → **신규 후보 SM2-A 로 승격**(아래 §2).

### 축 B(staging 상관함수·구조 인자) — **범위 밖 유지**
- staging 을 축간 상호작용 lattice gas 로 미시 유도(safran 계열). 여전히 문서 성격(피팅 방법론)을 미시 상도표 이론으로 팽창시켜 스코프 위반. **집행 비권장(명시만).**

### 축 C(히스 소산의 비평형 요동정리 Jarzynski/Crooks) — **범위 밖 유지·비권장**
- Part T 파생 D `warnbox`(`[C-92]`)가 히스 gap 소산을 "비가역, `∂/∂T` 와 별개, 경로의존 정량은 범위 밖"으로 **명시 선언**. 무리한 도입은 확정 경계 위반. **공백 유지가 옳음.**

### 축 D(annealed vs quenched) — **집행됨**
- R2 CLT bgbox 가 이미 "quenched(정적)/annealed 구분" 1문장 포함. 완료.

### flow-1/2/3(유도 매끄러움) — **부분 집행·flow성(심화 통계역학 아님)**
- **flow-1**(Part 0 사다리↔결과 사슬 1:1 대조표): §2b keybox `Part 0 사다리` 가 식 참조 목록은 주나 **1:1 표는 미집행**. 유효하나 flow성.

## 2. 신규 심화 후보 — 본문 정독에서 도출

> 각 후보: [교육 이득 / 모델 차원 영향(0) / 분량 / 삽입처 / 혼란 가중 리스크]. ★= 초안 작성 대상.

### ★SM2-A — 평형 peak = 등온 입자수 감수율(요동–소산 읽기) **[최우선]**

- **도출 근거(빈틈)**: Part 0 이 `var(n)=θ(1-θ)`(§2.2 `eq:sm-flucres`)·`∂⟨N⟩/∂μ=β var(N)`(§2.5 `eq:sm-mc-fluc`)를, §6 이 `(dQ/dV)^eq_j=Q_j ξ(1-ξ)/w_j`(`eq:eqpeak`)를 각각 세웠으나, **둘을 잇는 항등식이 없다**. §2.2 는 자리 하나에서 "peak 종 = 분산"을 예고만 하고, §6 은 관측식을 요동으로 읽지 않는다.
- **핵심 수학(검산 완료)**: 이상 폭 `w_j=RT/F`(n_j=1), `Q_j=(F/N_A)M_j`, `θ(1-θ)=ξ(1-ξ)` 에서
  ```
  (dQ/dV)^eq_j = Q_j ξ(1-ξ)/w_j = (F²/N_A RT) M_j θ_j(1-θ_j) = (F²/N_A RT) var_j(N)
  ```
  전 전이 합 + `var(N)=kT ∂⟨N⟩/∂μ`·`e=F/N_A`·`k_B/R=1/N_A` 로
  ```
  (dQ/dV)^eq = (F²/N_A RT) var(N) = e² ∂⟨N⟩/∂μ      (이상 폭)
  ```
  차원 검산: `e²[C²]·∂⟨N⟩/∂μ[1/J] = C²/(C·V) = C/V` ✓. 합규칙 `∫(dQ/dV)^eq_j dV = Q_j`(면적=용량=적분 감수율) ✓.
- **교육 이득**: **최상.** 사용자 핵심 피드백("통계역학이 초반 반짝하고 끝난다")의 **정면 해소** — 문서의 \*\*최종 산출물 dQ/dV 자체가 통계역학 요동량\*\*임을 항등식으로 못박아 §2.2→§2.5→§6 의 호를 닫는다("봉우리 하나=요동 하나, 곡선 전체=요동 지문"). DIRECTION 축 A 의 미집행 절반을 완성.
- **모델 차원 영향**: **0**(재해석 — 새 파라미터·식번호 재배열 없음; boxed 식은 무라벨).
- **분량**: bgbox 1개 ~22행(항등식 2식 + verifybox 4검산).
- **삽입처**: §6 `ch1_sec06_eqpeak`, `eq:eqpeak` 읽기 문단 직후(현행 35행 뒤). 대안 = §2.5 verifybox 확장(DIRECTION 원안 위치이나 §2.5 이미 밀도 높음 → §6 payoff 위치 권장).
- **혼란 리스크**: **낮음.** 표준 요동-소산/감수율. 유일 가드 = "이상 폭 n_j=1 에서 정확, 두-상 현상학 폭은 1/n_j 재척도"(폭 이중지위 §5·§7 와 정합) — verifybox 에 명시.

### ★SM2-B — 앙상블 동등성 = 상대 요동의 1/√M 소멸 **[차선]**

- **도출 근거(빈틈)**: §2.3 이 대정준↔셈법 두 경로가 같은 `μ(θ)` 에 닿음을 "앙상블 동등성"이라 **명명만** 하고(`eq:sm-mucount`), §2.5 가 반전을 대정준→정준 Legendre 반전이라 부르나, **왜 두 앙상블이 같은 열역학을 주는지(메커니즘)는 미서술**. Part 0·전문에 `1/√N`·상대요동·앙상블 동등성 유도 **전무**(grep 확인).
- **핵심 수학**: 방금 계산한 `var(N)=Σ M_j θ_j(1-θ_j)=O(M)`, `⟨N⟩=O(M)` 에서 `√var(N)/⟨N⟩ ~ 1/√M → 0`(거시 M). 곧 μ 고정 시 함량이 한 값에 못박혀 정준과 동일 강도적 열역학 → 앙상블 동등성 = 이 소멸의 따름. §2.5 반전의 정당성 근거.
- **교육 이득**: **높음.** 표준 통계역학의 정수(캡스톤)를 문서 자산(`var(N)`)으로 직접 닫아 §2.3 명명·§2.5 반전을 물리로 완결. "대정준↔정준 동등성"(과제 예시) 정면.
- **모델 차원 영향**: **0**.
- **분량**: bgbox 1개 ~15행(무라벨 식 1).
- **삽입처**: §2.5 `ch1_sec02b_part0`, `eq:sm-mc-fluc` verifybox 직후·소절 맺음 문단 직전(현행 380행 뒤). `var(N)` 이 바로 위에서 계산되므로 최적.
- **혼란 리스크**: **낮음.** 유일 가드 = "자리 독립 전제 위; 이중웰 Ω>2RT 상분리 창은 강상관이라 상대요동 커짐 → §4 소관"(이미 §2.5 verifybox 가드와 정합).

### ★SM2-C — 한 자유에너지의 두 응답: dQ/dV(∂_V)와 가역 발열(∂_T) **[3순위·Part 0↔Part T 다리]**

- **도출 근거(빈틈)**: §2.2 `s_int=k_B ∂(T ln q)/∂T`, §2.6 Nernst, §6 peak, §2.5 mixing config 항, §2.7 가역 발열이 각각 있으나, **"관측 dQ/dV(∂/∂V)와 가역 발열(∂/∂T)이 같은 평형 전위 `U_oc(x,T)`=같은 분배함수의 두 1계 응답"이라는 켤레-쌍 통일 서술이 없다**. 배치 Nernst 항 `(RT/F)ln[ξ/(1-ξ)]` 을 두 축으로 미분하면 peak(∝var)과 부분몰 배치 엔트로피가 한 항에서 갈라진다는 명시가 미집행.
- **핵심 수학**: `∂/∂V[(RT/F)ln(ξ/(1-ξ))] = (1/w)·ξ(1-ξ)/…`(peak, `eq:eqpeak`), `∂/∂T[(RT/F)ln(ξ/(1-ξ))] = (R/F)ln[ξ/(1-ξ)]`(= §2.5 완전식 config 항 `eq:single_config` = 가역 발열 배치 몫). 곧 전하 감수율↔열 감수율 켤레 쌍; `s_int`(§2.2)는 vibrational 축의 같은 ∂_T 응답, 세 분포 ∂_T 합 = ΔS(x)(§2.7).
- **교육 이득**: **높음(연결형).** 사용자 두 번째 피드백("절간·챕터간 연결 자연스럽게")의 정면 — Part 0/Part I(dQ/dV)↔Part T(가역 발열)를 한 켤레 쌍으로 묶어 DIRECTION 축 A 가 예고한 "열 감수율·전하 감수율의 쌍"을 완성(SM2-A 의 열역학 짝).
- **모델 차원 영향**: **0**.
- **분량**: bgbox 1개 ~17행(파생 쌍 1식군).
- **삽입처**: §2.7 `ch2_sec07_revheat`, `ssec:revinterp` 맺음 직후(현행 72행 뒤). Part T 이나 같은 ch1 빌드라 Part 0 라벨 직접 `\eqref`.
- **혼란 리스크**: **중(3후보 중 최고).** §2.5 파생 B(config 이중계산 분리)·§2.7 revinterp(분포 재배열 해석)와 **표면 인접** — 신규 기여는 "∂_V↔∂_T 켤레 쌍" 통일뿐이므로 박스가 그 한 점에 국한해야 함. 가드 = "가역 응답에 한함; 히스 gap 비가역 소산은 별개"(§2.5 파생 D warnbox 정합).

### SM2-D — Lindeberg 조건의 실측 대응(CLT 블록 확장) — **저순위·미초안**
- **내용**: R2 CLT bgbox 가 "Lindeberg 붕괴 → 종형 이탈"을 이미 서술. 실측 대응(한 원천 지배 시 왜도·비-Gaussian 꼬리로 나타남; 예 = 불량 접촉저항 소집단)을 1–2문장 추가.
- **교육 이득**: 낮음(한계효용). **모델 차원**: 0. **분량**: 2–3행. **삽입처**: §7.2 CLT bgbox 내부. **혼란 리스크**: 중(CLT 블록과 직접 중복 — 별도 블록화 부적합).
- **판정**: 초안 비대상. 필요 시 기존 CLT bgbox 1문장 증보로 충분(신규 블록 아님).

### SM2-E — 겹침 가중 = 감수율 가중(파생 A 계) — **SM2-A 에 흡수·미초안**
- **내용**: §2.5 mixing 겹침 가중 `Q_j g_j/Σ Q_k g_k`(`eq:weighted`)의 `g_j=ξ(1-ξ)/w_j` 가 곧 전이별 감수율/요동이라 "그 SOC 에서 어느 요동이 지배하는가"의 가중. SM2-A 의 §2.5 적용 따름정리.

### 핵심 관찰
- **세 초안(A·B·C)은 모두 이미 놓인 `var(N)`·Nernst 배치항의 \*미완성 상위 항등\*을 명시할 뿐, 새 물리·새 파라미터를 도입하지 않는다**(모델 차원 3건 전부 0).
- **A + C 가 짝**: A = 전하 감수율(dQ/dV↔요동), C = 열 감수율(가역열↔엔트로피). DIRECTION 축 A 가 예고한 "전하·열 감수율의 쌍"을 두 초안이 완성.
- **B 는 A·C 의 근거**: 상대 요동 소멸이 대정준 감수율을 정준 관측으로 읽어도 되는 근거(A)이자 반전 유일근의 거시 정당성(§2.5·B).
