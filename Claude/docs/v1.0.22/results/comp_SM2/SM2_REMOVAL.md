## 2. ★기존 CLT/CNT 블록과의 중복 여부 (과제 지정 항목)

### 2.1 CLT 블록(§7.2 `[겉보기 이동의 CLT 종형]`, 자산 A-011)과의 대조

| 축 | CLT 블록(§7) | SM2-A(§6) | SM2-B(§2.5) | SM2-C(§2.7) |
|---|---|---|---|---|
| **대상 분산** | η-원천 분산 `σ_η²=Σσ_ηk²` | 자리 점유 분산 `var(N)=ΣM_jθ(1-θ)` | 같은 `var(N)`(상대 요동) | (분산 아님) 배치 엔트로피 |
| **물리량** | 겉보기 이동 산포의 \*형상\*(폭) | 전하 감수율(peak 높이) | 앙상블 동등성(1/√M) | 열 감수율(가역 엔트로피) |
| **자리** | broadening 층(§7.2 iii-b) | 평형 peak(§6) | 대정준 반전(§2.5) | 가역 발열(§2.7) |
| **공유** | — | "독립 합의 분산 가법"이라는 표준 사실뿐 | 좌동 | — |

- **SM2-A vs CLT**: **중복 아님(상보).** CLT 는 `σ_η²`(η-원천 → 폭 형상), SM2-A 는 `var(N)`(자리 점유 → 전하 감수율). 대상·물리량·자리 전부 다름. R2 `REMOVAL_CHECK.md` §1.2 가 이미 "자리 점유 분산(축 A) vs η-원천 분산(broadening)은 대상·물리량·자리가 다름"을 명시했는데, SM2-A 는 그 **축 A 쪽을 관측식으로 완성**하는 것이라 CLT(broadening 쪽)와 정확히 분업. 두 블록이 "분산 가법"을 공유하나 **다른 분산**이므로 내용 중복 없음. (혼동 완화: SM2-A 는 `var(N)` 을 명시적으로 "저장 전하 요동/감수율"에, CLT 는 `σ_η²` 를 "η-원천/폭"에 국한.)
- **SM2-B vs CLT**: **무관.** B 는 앙상블 동등성(1/√M), CLT 는 broadening 형상. 접점 없음.
- **SM2-C vs CLT**: **무관.** C 는 가역 엔트로피(∂_T), CLT 는 폭 형상. 접점 없음.

### 2.2 CNT 링크(§4.3, 자산 A-012)와의 대조

- **세 초안 모두 CNT 와 무관.** CNT 링크는 히스 분기 축소 인자 γ_j 의 미시 근거(핵생성 지수 억제). 세 초안(감수율·앙상블 동등성·가역 발열 켤레)은 핵생성·γ_j 를 부르지 않음. **중복 0.**

## 3. 초안 상호 중복 · 기존 통계역학 내용과의 중복

- **SM2-A vs SM2-B**: 둘 다 `var(N)` 사용하나 **읽는 방향이 다름** — A = `var(N)→감수율(관측 dQ/dV)`, B = `var(N)→상대요동 소멸(앙상블 동등성)`. 대상 같고 결론 다름 → 상보(같은 자산의 두 활용). 별도 삽입처(§6 vs §2.5)라 독립.
- **SM2-A vs §2.2 `eq:sm-flucres`**: §2.2 는 자리 하나에서 "peak 종 = 분산"을 \*형상으로 예고만\*. SM2-A 는 물리단위 관측식 `(dQ/dV)^eq=e²∂⟨N⟩/∂μ` 로 **거시·차원 완성** → 중복 아님(payoff). 전문 grep: `e²∂⟨N⟩/∂μ`·"감수율(susceptibility)" 형태의 관측식 항등은 본문 전무.
- **SM2-B vs §2.3·§2.5**: §2.3 "앙상블 동등성"·§2.5 "Legendre 반전"은 **명명만**. B 는 `1/√M` 메커니즘 → 중복 아님. 전문 grep: `1/√N`·상대 요동·`√var/⟨N⟩` 서술 전무.
- **SM2-C vs §2.5 파생 B·§2.7 revinterp**: **표면 인접·내용 비중복.** §2.5 파생 B 는 config 이중계산 분리(`eq:single_config` 도입), §2.7 revinterp 는 "가역열 = 분포 재배열" 해석. SM2-C 신규 기여 = **"∂_V(peak) ↔ ∂_T(가역열)가 같은 자유에너지의 켤레 쌍"이라는 통일 서술**뿐 — 두 기존 서술 어디에도 이 V-T 응답 쌍의 명시 없음. **단 3후보 중 인접도 최고** → 박스가 그 한 점(켤레 쌍)에 국한해야 재진술 인상 방지(초안은 국한 준수).
