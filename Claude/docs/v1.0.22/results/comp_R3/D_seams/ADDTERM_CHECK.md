## 2. 받는 골격 vs 추가 텀 — 이중계산 경계 확인

"추가 텀만" 이 물리적으로 닫히려면 받은 골격과 새 텀이 겹치지 않아야 한다. 절들이 이를 식으로 고정하는지 확인:

- §2.4 슬롯 규칙(eq:lco-slots·eq:lco-decomp): config 는 **중심 표준값만**(혼합 분포항은 폭 $w_j$ 가 이미 담음 — 재기입 금지), elec 는 **게이트 골만** → 이중계산을 식 자체로 차단. **정합**.
- §2.4:56–58(OK-S8): 신 Ch2 의 성분 $\Delta S^0_j$(슬롯) vs Ch1 Part T 의 총합 $\Delta S^0_j$(성분 미분해) 혼동 금지 명시 — **정합**(정당 지칭, SEAM_PLAN §3b).
- §2.5 전자항: 흑연에 없던 항($\Delta S_e$)만 신설, 골격식(전하 보존·평형 종·합산)은 무개작 → "추가"의 정의 충족. **정합**.
- §2.6: 신 텀 0(파라미터 교체만) — "같은 식 반복"의 순수 사례. **정합**.

이중계산 경계가 절별로 식화되어 있어 "추가 텀만" 방식이 물리적으로도 닫힌다.

## 3. 받는 방식 명명 일관성 — 관찰(비강제)

방식은 성립하나, **Ch1 을 수신 지칭하는 명명이 절마다 다르다**:

| 수신 명명 | 사용 절 | 성격 |
|---|---|---|
| "흑연" + live \S\ref/\eqref | §2.3·§2.4·§2.5·§2.6 | 가장 명확(다수·권장형) |
| "Part I(흑연)" / "Part I" | §2.1·§2.2 | Ch1 내부 파트명 — \S\ref/\eqref 로 anchor 되나 병합 시 "Chapter 1" 접두 부재 |
| 축약형 "Ch1" | §2.7(+ §2.4:70,79) | 정확하나 축약(표기 정규화 후보) |

### 3-a. "Part II" 자기지칭(§2.1)
`ch1_sec11` 은 자기 절을 "Part II 도입"(절 제목)·"Part II 전체가 전제"·"Part II 종반" 으로 부른다. 신 구조에서 이 장은 **Chapter 2** 이므로 "Part II" 는 역사적 명칭 잔재다.
- **P5 경계**: 절 제목·역사적 인계 명칭은 사용자 지시 없이 임의 변경 X(글로벌 P5·CLAUDE.md P5-3). "Part II" 자기지칭은 **보존 대상일 수 있으므로 O-D 미집행**.
- 관찰만: 병합 문건에서 "Chapter 2 §2.1 Part II 도입" 은 독자에 다소 혼동. 마스터가 역사적 명칭 보존 vs 신 구조 정합 중 택일(판단 보류).

### 3-b. 수신 명명 조화(선택)
받는 방식을 다수형 "흑연 + live 참조" 또는 "Chapter 1 …" 로 통일하면 병합 정합이 오르나, 대상이 모두 정확하므로 **정정이 아닌 정합성 개선**이다. 강제 X — 마스터 결정(SEAM_PLAN §5 축약형 Ch1 정규화와 연동).
