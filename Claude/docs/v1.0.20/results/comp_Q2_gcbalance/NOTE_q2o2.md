## 2. 재유도 검산 (모든 신규 식 자체 재유도)

### (E1) `eq:sm-mc-factor` : Ξ = Π_j Ξ_{1,j}^{M_j}, lnΞ = Σ_j M_j lnΞ_{1,j}
대정준 합 `Ξ=Σ_{{n_site}} e^{-βΣ_site(ε̃_site-μ)n_site}`. 자리가 (클래스 간·클래스 내 평균장) 독립이면 지수의
합이 곱으로 갈라져 `Ξ=Π_site [Σ_{n=0,1} e^{-β(ε̃_site-μ)n}] = Π_site Ξ_{1,site}`. 클래스 `j` 의 `M_j` 자리는
동등(같은 `ε̃_j`)하므로 그 곱이 `Ξ_{1,j}^{M_j}`, 전체는 `Π_j Ξ_{1,j}^{M_j}`. 로그를 취하면
`lnΞ=Σ_j M_j lnΞ_{1,j}`. 이상 극한 단일 자리 인수는 `eq:sm-baresum`/`eq:partfn`(내부 자유도 `q` 는 `ε̃_j`
로 흡수, `eq:sm-epstilde`)에서 `Ξ_{1,j}=1+e^{-β(ε̃_j-μ)}`. **검산 OK** — `eq:sm-factor`(단일 클래스
`Ξ_M=Ξ_1^M`)의 자명한 다클래스 확장.

### (E2) `eq:sm-mc-navg` : ⟨N⟩ = k_BT ∂lnΞ/∂μ = Σ_j M_j θ_j
`eq:sm-gc` 셋째 식 적용. `∂lnΞ/∂μ = Σ_j M_j ∂lnΞ_{1,j}/∂μ`. 각 항:
`∂lnΞ_{1,j}/∂μ = (1/Ξ_{1,j})·(β e^{-β(ε̃_j-μ)}) = β·[e^{-β(ε̃_j-μ)}/(1+e^{-β(ε̃_j-μ)})] = β θ_j`,
`θ_j = 1/(1+e^{+β(ε̃_j-μ)})`. 따라서 `⟨N⟩ = k_BT·Σ_j M_j·β θ_j = Σ_j M_j θ_j` (`k_BT·β=1`). **검산 OK**
— `θ_j` 는 단일 자리 `eq:fermifn`(`θ=1/(1+e^{+βΔμ})`, `Δμ=ε̃-μ`)의 클래스별 복제.

### (E3) `eq:sm-mc-charge` : Σ_j Q_j ξ_eq,j(U_oc,T) = Q x̄
`ξ_j=1-θ_j`. 잔여 Li `⟨N⟩=Σ_j M_j θ_j`, 총 자리 `M=Σ_j M_j`. 추출 Li `= M-⟨N⟩ = Σ_j M_j(1-θ_j) = Σ_j M_j ξ_j`.
자리당 전하는 클래스 무관 공통 상수 `e`(몰당 `F/N_A`)이므로 `Q_j = e·M_j ∝ M_j`, 비(比)에서 상수 상쇄.
추출 전하 `= Σ_j Q_j ξ_j`, 총 용량 `Q=Σ_j Q_j`, 분율 `x̄ = (Σ_j Q_j ξ_j)/Q ⇒ Σ_j Q_j ξ_j = Q x̄`.
평형에서 `θ_j`(따라서 `ξ_j`)는 `μ` 를 `V` 로 옮기는 `eq:sm-eqcond`(`s=+1`)로 전위의 함수가 되고, 제약을 만족하는
전위가 `U_oc`. **검산 OK** — 결과가 `eq:implicit`(`Σ_j Q_j ξ_{eq,j}(U_oc,T)=Q x̄`)과 글자까지 동일.
`bar x ≡ 1-x`(추출 분율, `eq:implicit` 좌표)도 일치.

### (E4) `eq:sm-mc-fluc` : ∂⟨N⟩/∂μ = β Σ_j M_j θ_j(1-θ_j) = β var(N) > 0
`∂⟨N⟩/∂μ = Σ_j M_j ∂θ_j/∂μ`. `eq:sm-flucres`(`∂θ/∂(βμ)=θ(1-θ)`)에서 `∂θ_j/∂μ = β θ_j(1-θ_j)`.
⇒ `∂⟨N⟩/∂μ = β Σ_j M_j θ_j(1-θ_j)`. 한편 `N=Σ_site n_site`, 자리 독립 ⇒
`var(N)=Σ_site var(n)=Σ_j M_j·θ_j(1-θ_j)` (`eq:sm-flucres` 단일 자리 `var(n)=θ(1-θ)`의 합).
따라서 `∂⟨N⟩/∂μ = β var(N)`. `θ_j∈(0,1)` 이면 각 항 양수 ⇒ 합 양수 ⇒ `⟨N⟩(μ)` 순증가 ⇒ 역함수 유일.
**검산 OK** — 대정준 요동 정리 `⟨δN²⟩=k_BT ∂⟨N⟩/∂μ` 의 다클래스 형이며 `eq:sm-flucres` 의 거시판.

### (E5) 검산 ② `N_p=1` 회수
`N_p=1`: `Ξ=Ξ_{1,1}^{M_1}=Ξ_1^M`(=`eq:sm-factor`), `⟨N⟩=M_1 θ_1=Mθ`. §2.3 정확 회수. **확장 off OK**.

### (E6) 검산 ③ μ↔V 아핀·부호
`eq:sm-eqcond`: `μ_Li=μ^0-sF(V-U)`, `s=+1` ⇒ `dμ_Li/dV=-F<0` (단조·아핀). `μ` 축 유일근 ⇒ `V=U_oc` 유일근.
부호 정합: `⟨N⟩` 은 `μ` 증가함수, `μ` 는 `V` 감소함수 ⇒ `⟨N⟩` 은 `V` 감소함수 ⇒ 추출분율 `x̄` 는 `V` 증가함수.
곧 "V 올리면 탈리튬화"(Part 0 signbox·본론 규약)와 일치. **부호 검산 OK**.

### 규약 재확인
`Ξ`(대정준, 정준 `Z` 와 구분) ✓ · `β=1/k_BT` ✓ · `θ` 점유/`ξ=1-θ` 진행률 ✓ · `s=+1` ✓ · 신규 라벨 `eq:sm-mc-*` ✓ · 인용 `hill1960,mcquarrie1976` 만 ✓.
