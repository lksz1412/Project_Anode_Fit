- **베이스 = F3** (보존 충실도 + 관용 대조 가드 + 계산 창).

- 물리 가드 3항: 5본 전부 통과(밴드 절연체 명시·Marianetti "설명했다" 수준·불순물 준위 한정) — 가드 위반 실격 없음.
