## 후보 (i) — 전하 보존식의 대정준 유도

### 물리 타당성 (핵심 수학 스케치로 검증) — **타당(강)**

Part 0 §2.3(`sec:sm-lattice`)이 이미 **단일 클래스** M 자리에 대해 `Ξ_M=Ξ_1^M`, `⟨N⟩=k_BT ∂lnΞ_M/∂μ=Mθ(μ)` 를 닫았다. 후보 (i)의 요지는 이를 **다클래스**로 한 줄 일반화하고, 전하 보존 음함수(`eq:implicit`)가 그 대정준 구조의 '제약 반전'임을 보이는 것이다. 스케치:

**(1) 다클래스 대정준 분배함수.** 전극을 전이(클래스) `j=1..N_p` 로 나눠 클래스 `j` 가 `M_j` 개의 동등 자리(유효 자리 자유에너지 `ε̃_j`, 중심 `U_j`)를 갖는다고 하면, 자리 독립(클래스 내부는 평균장 `Ω_j`)에서 대정준 분배함수가 전 자리 곱으로 인수분해된다:
```
Ξ = Π_j Ξ_{1,j}^{M_j},   ln Ξ = Σ_j M_j ln Ξ_{1,j}
```
(`Ξ_{1,j}=1+e^{-β(ε̃_j-μ)}` 이상 극한, Ω_j≠0 이면 자기무모순 평균장형). 이는 §2.3 의 `Ξ_M=Ξ_1^M`(`eq:sm-factor`)의 자명한 다클래스 확장이다.

**(2) 평균 입자수 = 용량가중 점유합.** `eq:sm-gc` 셋째 식으로
```
⟨N⟩ = k_BT ∂lnΞ/∂μ = Σ_j M_j θ_j(μ),   θ_j = 1/(1+e^{β(ε̃_j-μ)})
```
`ξ_j=1-θ_j`(탈리튬화 진행=자리 공위), 용량 `Q_j∝M_j` 로 읽으면 이 식은 `Σ_j Q_j(1-θ_j)=Q·x̄`(추출 전하), 곧 **`Σ_j Q_j ξ_eq,j(U_oc,T)=Q x̄` = `eq:implicit` 그 자체**다.

**(3) canonical↔grand-canonical 반전 = 솔버.** 대정준은 μ 가 독립변수이고 `⟨N⟩(μ)` 이 결정된다. 실험/모델은 반대로 Li 함량(`x̄`, 통과 전하)이 **고정**되고 그에 맞는 μ(따라서 `U_oc`)를 **역으로 푼다** — 이것이 `⟨N⟩` 고정 제약을 건 canonical 기술로의 Legendre-켤레 반전이고, `ch2_appB` 의 `solve_U_oc(x̄,T)` "유일근 솔버"가 정확히 이 반전을 수행한다(μ↔V 는 `eq:sm-eqcond`).

**(4) ★유일근의 통계적 보증(payoff).** 반전이 유일근을 갖는 근거는 `⟨N⟩(μ)` 의 단조성이고,
```
∂⟨N⟩/∂μ = β Σ_j M_j θ_j(1-θ_j) = β·var(N) > 0
```
곧 **입자수 요동의 양positivity 이 솔버 유일근을 보증**한다. 이는 (F-1)의 요동–응답 관계 `eq:sm-flucres` 의 거시판이며, `ch2_appB` 가 서술한 "유일근"에 **통계역학적 존재·유일성 증명**을 부여한다. 후보 (i)의 가장 우아한 결실이 이 지점이다.

판정: 수학적으로 §2.3 의 1–2줄 확장 + 식별이며 전 과정이 엄밀. CLAUDE.md P1 목표("전하 보존식 기반 내부 전위 결정")와 P3 검수 2항("전하 보존식이 내부 전위를 결정하는 중심식으로 유지, OCV 에서 읽는 흐름으로 회귀 X")에 **직결** — Ch1 중심식의 통계역학 1급 승격이 실증된다.

## 후보 (ii) — Eyring 꼬리의 TST 분배함수 유도

### 물리 타당성 (핵심 수학 스케치로 검증) — **타당(중상)**

현행 `ch1_sec05_width` §5.1 은 `k≃k_0 e^{-ΔG_a/RT}`, `k_0=k_BT/h` 를 "전이상태 이론의 시도 빈도"로 **제시만** 하고(eyring1935 인용), `ch1_sec08_lag` 는 `ΔG_a^eff=ΔH_a^eff-TΔS_a` 로 갈라 쓰되 **활성화 엔트로피 `ΔS_a` 의 미시 의미를 주지 않는다**. 후보 (ii)는 `k_0=k_BT/h` 와 `ΔS_a` 를 분배함수로 유도해, Part 0 §2.2 의 내부 자유도 `q(T)`(`eq:partfn`) 언어와 같은 축에 놓는다. 스케치:

**(1) 정통 TST 속도식.** 반응물 분배함수 `q_R`, 활성복합체(안장점에서 반응좌표 모드를 뗀) 분배함수 `q‡`, 장벽 바닥차 `ΔE_0` 로
```
k = (k_BT/h) · (q‡/q_R) · e^{-ΔE_0/RT}
```
**(2) `k_BT/h` 의 미시 기원.** 안장점의 반응좌표 모드는 길이 `δ` 를 지나는 1D 병진이고, 그 1D 분배함수 `(2πm*k_BT)^{1/2}δ/h` 에 평균 통과속도 `⟨v⟩=(k_BT/2πm*)^{1/2}` 를 곱해 `δ` 로 나누면 통과 빈도 `k_BT/h` 가 남는다 — **`k_0` 는 임의 상수가 아니라 장벽 통과의 보편 빈도**다. 이 유도가 현행 문건의 "시도 빈도" 한 마디에 근거를 준다.

**(3) `ΔS_a` = 분배함수 비.** `ΔG‡=-RT ln(q‡/q_R)+ΔE_0` 로 묶으면 `k=(k_BT/h)e^{-ΔG‡/RT}=(k_BT/h)e^{ΔS_a/R}e^{-ΔH_a/RT}`, 곧
```
ΔS_a = R ln(q‡/q_R)|_reduced
```
**활성화 엔트로피는 활성복합체와 반응물의 분배함수 비의 로그**다 — Part 0 의 `s_int=k_B ∂(T ln q)/∂T`(`eq:sm-sint`)와 동일 언어. 활성복합체가 반응물보다 자유도가 조이면(`q‡<q_R`) `ΔS_a<0`, 풀리면 `ΔS_a>0` 의 부호 읽기가 곧바로 나온다.

판정: 표준 TST 그대로이며 물리적으로 건전. 현행 문건의 `q(T)`·`s_int` 갈고리(Part 0)와 자연 접속. 사용자 피드백("동역학 유사식만 보이다 끝난다")의 정확한 타깃 — 동역학 prefactor 에 통계역학 살을 붙인다.

## 후보 (iii) — broadening 의 앙상블 통계 유도 (CLT)

### 물리 타당성 (핵심 수학 스케치로 검증) — **타당(중), 범위 한정 필수**

`ch1_sec07_broadening` §7 은 이미 앙상블 forward 평균 `eq:ensavg`(`⟨dQ/dV⟩=∫ρ(U_app)(dQ/dV)^single dU_app`)와 분산 가법 `eq:widthbudget`(`σ_sym²=π²/3(n_jRT/F)²+σ_η²`)을 갖고 있다. **그러나 `ρ(U_app)`(=`ρ(η)`)가 왜 종 모양(Gaussian)인지 근거가 없다** — `fig:widthbudget` 캡션은 "Gaussian 예시"로 쓸 뿐이다. 후보 (iii)는 이 공백을 CLT 로 메운다. 스케치:

**(1) quenched 앙상블 평균.** 입자별 겉보기 이동 `η` 는 소인(sweep) 시간 척도에서 **정적(quenched)**이다(입자마다 고정, 한 소인 동안 불변) — 관측량 안에서 평균되는 quenched disorder 로, `eq:ensavg` 의 forward 평균이 정확히 그 quenched 평균이다.

**(2) 독립 원천 다수 → CLT.** `η` 가 성격이 다른 다수 독립 기여의 합이면
```
η = Σ_{k=1}^m η_k   (접촉저항·국소 결정성·흑연화도·turbostratic 무질서·국소 η ...)
```
Lindeberg 조건(한 원천이 지배하지 않음) 하에 CLT 로
```
ρ(η) → N(η̄, σ_η²),   σ_η² = Σ_k σ_{η_k}²
```
이 **(a) `fig:widthbudget` 의 Gaussian `ρ(η)` 형상에 미시 근거**를 주고, **(b) `eq:widthbudget` 분산 가법 `σ_η²=Σ_k σ_{η_k}²`** 를 CLT 결론으로 재유도한다(현행은 "분포 모양과 무관한 합성곱 항등식"으로만 정당화 — CLT 는 그 위에 형상까지 준다).

판정: CLT 는 교과서 결과이며 `η` 의 quenched·다원천 성격에 정확히 부합. §7 이 이미 세운 forward-only·분산가법 틀에 **형상 근거**만 보태는 정밀 보강이라 모델 차원 불변.

### 문건 내 배치 제안
- **1순위(권장)**: `ch1_sec07_broadening` §7.2 (iii-b) 안, `eq:ensavg` 직후에 **짧은 유도 단락 또는 `bgbox`[겉보기 이동의 CLT 종형]** 1개. `dreyer2010/2011`(다입자) 연결과 자연 인접.
- **연결**: `fig:widthbudget` 캡션의 "Gaussian 예시"를 "CLT 로 근거지어진 종형(본문 (iii-b))"로 참조 승격(캡션 문구는 실집행 시 사용자 판단 — 본 보고서는 수정 안 함).

### D7 정통 선행 양식 적합 형태
CLT 는 표준 정리이므로 **원형(단일 원천 극한)→확장(다원천)** 이 자연스럽다: [원형=단일 지배 원천이면 `ρ(η)` 는 그 원천의 분포(비-Gaussian 가능) → 확장 동기="원천이 다수·비지배면" → 다원천 CLT 종형·분산가법 → 회수 검산: `m=1` 또는 한 원천 지배(Lindeberg 붕괴)면 종형 이탈, `ρ→δ` 면 `③` 소멸해 `eq:ensavg` 가 단일 델타로 환원(§7 이미 서술)]. verifybox: 원천 수 `m→∞`·분산 유한이면 종형 수렴, 한 원천 지배면 이탈(한정).

### 예상 분량
유도 단락 6–10행(CLT 1식 + 분산가법 회수 1식 + 한정 2문장). `bgbox` 로 하면 +2행. **총 약 8–12행.** 신규 그림 불요(`fig:widthbudget` 재사용).

### 필요 문헌
- **기존 원장 키로 충분**: `dreyer2010`·`dreyer2011`(다입자 앙상블). CLT 자체는 교과서 상식이라 무인용 관용 가능(rubric — 표준 수학은 무인용 허용).
- **신규 서지 불필요**: quenched disorder 전용 문헌(스핀글래스 등)은 **과대인용 위험**이라 비권장 — §7 의 forward-only·현상학적 폭 흡수 기조와 맞지 않음. distribution of relaxation times(DRT) 문헌도 §7 이 이미 "동형"으로만 언급하며 역산 금지를 못박았으므로 **추가 인용 자제**가 문건 기조에 부합.

### 위험 (기존 확정 판정·자산과 충돌)
- **중.** 정책 충돌은 아니나 **경계 관리 필수**:
  - **★역산 금지 훼손 금지**: §7 (ii)가 `dQ/dV→ρ(U_app)` 역산을 ill-posed 로 배제(forward-only). CLT 는 **생성적(forward)** 정당화이지 역산 허가가 **아님** — 배경 단락이 "CLT 는 형상을 forward 로 근거지을 뿐 역산을 열지 않는다"를 명시해야 함.
  - **★흑연 두-상 국한 유지**: §7 (iii)의 구조 무질서 근거는 흑연 두-상(`LiC12`·`LiC6`)에 국한, LCO 는 일반 `η` 분포로만. CLT 배경도 이 한정을 넘지 말 것.
  - **★비-크기 heterogeneity 유지**: `ρ(U_app)` 는 비-크기 `η` 산포 전용(§7.3(i)이 사이즈 채널 정량 배제). CLT 원천 목록에 입자 반경을 넣으면 배제한 사이즈 채널이 뒷문으로 재진입 — 원천은 비-크기만.
  - **★②는 Gaussian 아님**: 내재 폭 `②`는 logistic 미분 종이라 Gaussian 이 아님. CLT 는 `③`(`ρ(η)`)에만 적용 — `②⊗③` 는 logistic⊗Gaussian 이고 `fig:widthbudget` 캡션이 이미 "같은 함수족 지름길이 실제 합성곱보다 peak 10% 높음"을 명시. CLT 배경이 이 구분을 흐리면 안 됨.
- 위 4가드를 지키면 확정 자산과 충돌 없음. 가드 누락 시 `[C-2]`계열 forward-only 원칙 훼손 위험이 실재하므로 **가드 명시가 집행 전제**.

### 권고 우선순위: **3순위**
효과는 국소(형상 근거 1개)이나 사용자 피드백("통계역학으로 뒷 내용 유도")에 부합하고 저분량. 단 4가드 관리 부담이 (i)(ii)보다 큼.

## 후보 (iv) — 준안정 과주행의 요동 통계 (CNT)

### 물리 타당성 (핵심 수학 스케치로 검증) — **타당(강), 단 대부분 이미 부록에 존재**

**★핵심 발견 (F-2)**: 후보 (iv)가 요구하는 "핵생성 지수 억제 `exp(-ΔG*/k_BT)` 로 왜 소인이 spinodal 까지 준안정 가지를 타는가"의 정량 근거는 **`appendix_phase_separation` 에 이미 완비되어 있다**:
- §A.7(`app:nucleation`): `ΔG(r)=(4/3)πr³Δg_v+4πr²γ`(`eq:app-cnt`), 임계핵 `r*=2γ/|Δg_v|`, `ΔG*=16πγ³/(3Δg_v²)`(`eq:app-rstar`), 핵생성률 `∝exp(-ΔG*/k_BT)`.
- §A.9(`app:link`)가 §4 과주행과 명시 연결: "준안정 가지에서 핵생성이 지수 억제로 지연되는 동안 소인이 전위를 계속 옮긴다."

따라서 후보 (iv)의 물리는 **신규 유도가 아니라 (a) 부록의 CNT 를 §4 본문으로 편입/연결하고 (b) `γ_j` 에 요동 통계 읽기를 부여**하는 것이다. 추가 물리 스케치(γ_j 연결):

**(1) 준안정 가지 위 장벽 프로파일.** binodal→spinodal 로 과주행이 진행하며 구동력 `|Δg_v|` 가 0(binodal 접선구동력 0)에서 자라 `ΔG*∝|Δg_v|^{-2}` 가 ∞→유한으로 급락한다. §A.7 이 이미 "binodal 근처 `ΔG*` 발산 → 분해 사실상 안 일어남 → 준안정 오래 관찰"을 서술.

**(2) `γ_j` 의 통계적 이탈 판정.** 소인이 가지를 이탈하는 지점은 소인 시간 안에 핵생성이 개연해지는 곳, 곧 `exp(-ΔG*(V)/k_BT)×(시도수)~1` 이다. §4 의 축소 인자 `γ_j∈[0,1]` 는 **이 이탈점이 spinodal(γ_j=1) 대비 얼마나 안쪽인가**를 흡수한 현상학 인자 — CNT 가 `γ_j<1`(이상 단일입자가 아닌 실재 전극이 요동·불균일 핵생성으로 조기 이탈)에 **미시 근거**를 준다. 현행 §4 는 이를 이미 산문으로 서술("국소 요동·불균일 핵생성으로 가지를 이탈")하나 CNT 식과 연결하지 않는다.

판정: 물리 건전(부록에서 이미 검증). 사용자 피드백("뒤 내용도 통계역학으로")에 부합하며, **부록↔본문 편입 결정을 진전**시키는 부수 효과(부록 지위 = "본문 편입 여부 검토 후 결정" 초안).

### 문건 내 배치 제안
- **1순위(권장, 경량)**: `ch1_sec04_hys` §4.3(`sec:hys-branch`), `γ_j` 도입 문단에 **부록 CNT 참조 다리 1–2문장** + `exp(-ΔG*/k_BT)` 억제를 `γ_j<1` 의 미시 근거로 명시. 신규 유도 없이 부록 자산을 본문에 연결(부록은 독립 컴파일 유지하므로 절 번호 텍스트 참조 — 부록 관례 준수).

## 후보 (v) — 단상 폭의 Ω 재규격화 미시 유도

### 물리 타당성 (핵심 수학 스케치로 검증) — **타당(수식은 옳음), 정책 충돌 위험 최상**

**★핵심 발견 (F-3)**: 이 결과는 `ch1_sec05_width`(폭 이중지위)와 `ch2_sec05_mixing` 파생 C 에 **이미 산문으로 존재**한다("0<Ω<2RT 에서 중심 기울기가 `(1-Ω/2RT)` 배 좁아진 유효 폭, 그 값도 평형이 결정"). 수식 검증(2줄 미분):

정규용액 등온선 `eq:Veq_BW`(`ch1_sec02a`): `RT ln[θ/(1-θ)] + Ω(1-2θ) = -F(V-U)` 를 음함수 미분하면
```
[RT/(θ(1-θ)) - 2Ω] dθ = -F dV
θ=1/2 에서:  dθ/dV|_{1/2} = -F/(4RT - 2Ω) = -F/[4RT(1-Ω/2RT)]
```
평형 peak 중심 기울기 `|dθ/dV|_{1/2}=1/(4 w_eff)` 로 읽으면
```
w_eff(Ω) = (RT/F)(1 - Ω/2RT)     (단상 Ω<2RT 한정)
```
`Ω→2RT⁻` 에서 `w_eff→0⁺`, peak 발산 — **상분리 임박(incipient phase separation)의 균질상 쪽 접근**이다. 이는 `dahn1991`·`bazant2013` 의 인력 상호작용 peak 첨예화와 정합.

판정: 수식은 정확하고 물리도 건전(균질 고용체가 spinodal 에 아래로부터 접근하며 peak 가 좁아짐). 사용자 피드백("통계역학으로 유도")에 부합. **그러나** 이 결과는 이미 문건에 산문으로 있고, 전면 유도 박스화의 한계효용이 낮은 반면 **정책 충돌 위험이 후보 중 최상**이다.

### 문건 내 배치 제안
- **최대 허용 범위(권장, 극경량)**: `ch1_sec05_width` §5.2(폭 이중지위)의 단상 문단에 **`w_eff(Ω)=(RT/F)(1-Ω/2RT)` 2줄 미분을 각주 또는 인라인 유도**로 승격 — **단상 한정 라벨 필수**, 독립 박스 금지. 이미 있는 산문("(1-Ω/2RT) 배")에 수식 근거를 붙이는 최소 개입.
- **비권장**: 독립 keybox/derivbox 로 `w_eff(Ω)` 식을 세우는 것. 두-상 warnbox(`ch2_sec05` `[C-87]`)가 금지한 `w_eff(Ω)=w(1-Ω/2RT)` 와 **글자까지 같은 식**이라, 독립 박스는 두-상 금지식과 시각적으로 구별 안 됨 — 재도입 오독을 부른다.

### D7 정통 선행 양식 적합 형태
정통 유도(정규용액 등온선 미분)이므로 원형 선행은 자연스러우나, **분량을 각주/인라인으로 억제**해야 정책 충돌을 피한다. 만약 유도한다면: [원형=이상 `Ω=0`, `w=RT/F` → 확장 동기="단상 인력이 중심 기울기를 비튼다" → `w_eff(Ω)` 2줄 → 회수 검산 `Ω→0` 이면 `RT/F`, `Ω→2RT⁻` 이면 `w_eff→0`(spinodal 임박)]. **회수 검산에 반드시 "두-상 `Ω>2RT` 에는 이 식을 쓰지 않는다(broadening 이 폭을 정함, §7)" 가드 포함**.

### 예상 분량
극경량안(각주+인라인): **총 약 4–6행.** 독립 박스화하면 10–14행이나 위험 대비 비권장.

### 필요 문헌
- **기존 원장 키로 충분**: `bazant2013`·`huggins2009`(정규용액 등온선·인력 첨예화). **신규 서지 불요.**

### 위험 (기존 확정 판정·자산과 충돌) — **최상(후보 중 유일 고위험)**
- **★C-2 확정 판정 충돌**: `ch2_sec05` 자산 `[C-2]`="v5 정정 구 `w_eff(Ω)` narrowing 식 완전 제거(재도입 금지)", `[C-87]`="warnbox narrowing 서술 실측반대"가 **두-상** `w_eff(Ω)` 재도입을 명시 금지. 단상 `w_eff(Ω)` 는 물리적으로 별개(단상=평형 예측, 두-상=현상학)이나, **식이 글자까지 동일**해 독자·후속 편집자가 혼동할 위험이 실재. task ★경계와 정확히 일치하는 위험.
- **★두-상 폭 판정 불변 훼손 금지**: 두-상 `w_j`=현상학 자유 피팅(§5·§7·파생 C 확정). 단상 유도가 이 판정을 흐리면 P3 검수 2·6항 위반.
- **필수 가드(집행 시)**: (a) "단상 `Ω<2RT` 한정" 라벨을 식 바로 옆에, (b) "두-상은 §7 broadening 이 폭 결정, `w_eff(Ω)` 미적용" 가드를 같은 자리에, (c) 독립 박스 금지(각주/인라인만). 세 가드 없이는 집행 불가.

### 권고 우선순위: **5순위(최하) — 조건부 또는 보류**
수식은 옳으나 이미 산문 존재 + 한계효용 낮음 + 정책 충돌 위험 최상. **권고: 극경량(각주 2줄)로만 하거나, C-2 혼동 위험을 감안해 현행 산문 유지(집행 보류)**. 사용자가 "단상 유도까지 명시"를 원하면 3가드 하 각주로만.
