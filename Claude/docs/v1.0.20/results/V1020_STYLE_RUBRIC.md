## A. 어조·문단 (register)
- A1. 완결 명제 문장만. 전보체(주술 없는 보충 명제) 금지 — 괄호 보충도 주술을 갖춘다.
- A2. [D1] 자기 개정 이력·버전 서술 금지(본문·각주·캡션). [D2] "X가 아니다" 단독 방어 금지 — 긍정 서술. 단 **오독 방지 가드**는 예외 관용: "공유하는 것은 ~뿐이고, 여기의 ~는 ~다"(FD 동형 가드 형식) 대조 구조 허용.
- A3. 절 도입 1문단 = [앞 절에서 받은 것 → 이 절이 놓는 것 → 어디로 전달]. 절 마무리 = 다음 절로 넘기는 다리 1–2문장. orphan(도입 없는 추가·사용 없는 도입) 0.
- A4. 문단 3–6문장, "---"(em-dash) 부연 관용, 나열은 (i)(ii)(iii) 또는 (a)(b)(c) enumerate.
- A5. 메타발언·고백조("정직하게 적으면" 류)·작업 라벨(P-번호·phase)·코드 함수명, 본문 노출 0 (코드 대응은 부록 B 전용).
- A6. 서술 시제·경어: 평서 현재 "~다" 종결. 독자 지칭 없음(명령형 "~하라" 금지; "~를 보라" 대신 "\S~가 닫는다/상술한다" 관용).

## B. 수식 제시 (헌법 ③ + D7)
- B1. 결과 박스 사슬 = **(a) 출발/전제 → (b) 적용 연산 → (c) 중간식 ≥1 → (d) \boxed 박스**. 단락 헤더 `\textbf{(a) 출발 --- ...}` 형식.
- B2. [D3] "대입하면 [박스]" 점프 0 — 부분적분·치환·전개·상쇄는 중간식으로 노출.
- B3. 박스 직후 verifybox: 극한 검산 ≥2(β→0/∞·θ→0/1·Ω→0 등) + 부호 읽기. 확장식이면 **확장 off ⇒ 원형 회수** 검산 필수.
- B4. **[D7 — 정통 유도 선행]** 표준 이론이 존재하는 유도 = [원형 유도·박스(문헌 병기)] → [확장 동기 1–2문장: 왜·무엇이 달라지나] → [확장 유도·박스] → [원형 회수 검산]. 원형과 확장의 박스는 분리한다.
- B5. 새 기호는 등장 식 직후 "---" 부연으로 정의하고 tab:notation 에 등재. 기호 충돌은 각주 가드(F vs Faraday·q(T) vs q 좌표·u_j vs θ_E 관용 유지).
- B6. 보편식 먼저, 극한·특수형은 코너 케이스로 뒤에.
- B7. 그림 좌표·수치는 "식 그대로의 수치 평가"(캡션에 명시) — 정성 개형이면 "개형" 명시.

## C. 용어 (한글 prose + 영어 학술 원어)
- C1. 첫 출현 병기 형식: `\emph{격자기체}(lattice gas)` — 이후 한글 또는 원어 일관 사용(절 내 혼용 금지).
- C2. 고정 대역표(위반 = P6 스윕 대상): 미시상태(microstate)·거시상태(macrostate)·앙상블(ensemble)·저장조(reservoir)·대정준(grand canonical)·평균장(mean-field)·정규용액(regular solution)·이중웰(double-well)·spinodal(원어)·준안정 가지(metastable branch)·과주행(overshoot)·두-상/단상(two-phase/single-phase)·점유율 θ·(탈리튬화) 진행률 ξ·**가역 발열**(reversible heat — "가역열" 금지, v1.0.17 결정)·전이 중심·분기 중심·꼬리(tail)·배치(configuration)·섞임 엔트로피(mixing entropy).
- C3. **페르미온(fermion)·보손(boson)** 표기(=ch2_sec01/03 현행. "보존" 표기 금지). Fermi--Dirac·Bose--Einstein 은 원어 하이픈 표기.
- C4. 두문자어 첫 출현 병기: MIT(insulator-to-metal transition)·MSMR(multiple-species multiple-reaction? — P1 원 논문 검증 후 정확 풀네임 확정)·GIC·SOC·OCV·DOS(밀도는 상태밀도 병기).
- C5. 특정 국적 표현 회피. 단위 [J/mol]·[V]·[mV/K] 대괄호 관용.
- C6. tier A/B/C 신뢰 등급 각주 체계 유지(sec11 각주가 정의 — 전 문건 공유).
