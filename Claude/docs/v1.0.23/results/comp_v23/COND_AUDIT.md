## 1. 참 문제 정식화 — 비선형 Volterra 자기일관 (재유도 항목 1)

### 1.1 운동방정식 → 지연 ODE (문건 재확인)

§5 평형 근방 선형 완화 `dξ_j/dt = k_j(ξ_eq,j − ξ_j)` 를 정전류 `dq/dt=|I|/Q_cell` 로 나눠 용량축으로 옮기고
(eq:Lqmid), 지연 `r_j ≡ ξ_eq,j − ξ_j` 로 정리하면(eq:Lq):

```
dr_j/dq = dξ_eq,j/dq − r_j / L_q,j ,      L_q,j = |I| / (Q_cell k_j)          (eq:Lq)
```

컷점 기울기로 V-축 환산하면(eq:LV) 방전 기준 하나의 진행축 위에서

```
dr/dV = σ(V) − r · κ(ξ) ,   κ(ξ) ≡ 1/L_V(ξ) ,   ξ = ξ_eq(V) − r ,   σ ≡ dξ_eq/dV.   (★참 문제)
```

### 1.2 왜 자기일관(비선형)인가 — `L_V=L_V(ξ)`

`k_j` 는 affinity `𝒜_j = sF(V−U_j) − Ω_j(1−2ξ_j)` 를 통해 상태의존이다(eq:kuniv·eq:dHeff). `L_q ∝ e^{−χ_d𝒜/RT}`
이므로 **`L_V` 는 `ξ` 의 함수**이고, `ξ = ξ_eq − r` 이므로 `r` 이 자기 자신의 감쇠율에 되먹임된다 → **비선형**.
상호작용 몫 `−Ω(1−2ξ)` 가 자기참조의 원천이다(이상 몫 `sF(V−U)` 는 V-의존일 뿐 ξ-되먹임 아님).

### 1.3 형식적 정확해 = 비선형 Volterra (적분인자)

변계수 적분인자 `μ(V)=exp[∫^V κ(ξ(V'))dV']` 로 닫으면

```
r(V) = ∫_{−∞}^{V} σ(u) · G[V,u;ξ] du ,    G[V,u;ξ] = exp[ − ∫_u^V κ(ξ(V')) dV' ].       (Eq-V)
```

`ξ(V') = ξ_eq(V') − r(V')` 이 핵 `G` 안에 있어 **미지량이 적분 안팎에 동시 등장** → **제2종 Volterra(인과·1측)**.

### 1.4 논문과의 구조 차이 (정직)

| 축 | 사용자 논문 (JCP 147 Eq.32) | 문건 lag (Eq-V) |
|---|---|---|
| 적분방정식 종 | **Fredholm 2종**(고정 구간 [σ,r], 양측) | **Volterra 2종**(가변 상한 V, 인과 1측) |
| 핵 구조 | 비퇴화(전역 결합) | **지수 = rank-1 분해핵 = Markov** |
| 등가 국소형 | 없음(전역) | **1계 비선형 ODE(★)** — `r` 은 같은 V 에서만 자기 결합 |
| 수치 비용 | 전역 역산 O(N²)/반복 | **전진 O(N) 1패스**(RK4) |
| ratio 기법 동기 | 전역 역산 회피(필수) | **부재** — 참 문제가 이미 값쌈 |

곧 **리터럴 Fredholm 이식은 불가**(문건에 Fredholm-2 없음). 이식되는 것은 Eq.34 의 *철학*(미지량→비→가해기준
치환)뿐이며, 그 철학이 Volterra 위에서 성립하는지는 §3 에서 재유도·검증한다.

## 3. 1차 ratio 보정 — 논문 Eq.34 철학 이식 (재유도 항목 3)

### 3.1 유도 (비 고립 → 가해기준 치환)

(Eq-V)의 핵을 동결핵 `G_0(V,u)=e^{−(V−u)/L_0}` 로 갈라 **보정비(memory ratio)** 를 고립:

```
G[V,u;ξ] = G_0(V,u) · R[V,u;ξ] ,   R[V,u;ξ] = exp[ − ∫_u^V δκ(ξ(V')) dV' ] ,  δκ ≡ κ(ξ)−κ_0 .
```

미지량 `ξ` 는 **오직 보정비 `R`(비 구조)** 를 통해서만 등장한다 — 이것이 논문 Eq.33 의 "미지량을 비로만 남긴다"의
Volterra 판이다. 논문 Eq.34(미지 비 → 가해기준 비)를 그대로 이식: **`R` 안의 미지 궤적 `ξ` 를 가해 기준 궤적
`ξ_0 = ξ_eq − r_0` 로 치환**한다:

```
r_1(V) = ∫_{−∞}^{V} σ(u) G_0(V,u) exp[ − ∫_u^V δκ(ξ_0(V')) dV' ] du .                (1차 ratio 닫힌형)
```

**등가 국소형(구현식)**: 위는 정확히 **가해기준 궤적 위에서 평가한 변계수 선형 ODE** 의 해다 —

```
dr_1/dV = σ(V) − r_1 · κ(ξ_0(V)) ,   ξ_0 = ξ_eq − r_0  (기준·기지).                  (1차 = 자기참조 해소)
```

즉 "자기참조 상태 `ξ=ξ_eq−r` 를 **가해 기준** `ξ_0=ξ_eq−r_0` 로 대체" → 선형화. 이것이 Eq.34 의 정본 이식이다.
(Volterra 라 각 스텝이 O(N) 전진해임을 명기 — §5.2 정직 함의.)

### 3.2 게이트 (a) — 동결극한 정확 회수 [PASS]

`κ(ξ)≡κ_0`(즉 상태의존 off, 물리적으로 `Ω→0` 또는 `g≡Ω/RT→0`)이면 `δκ≡0` → `R≡1` → **`r_1 = r_0` 정확**.
수치(Part B): `g=0` 에서 ‖r₁−r₀‖ = **1.06×10⁻⁸ · 2.90×10⁻⁸**(= RK4 이산화 바닥, Part A 앵커 2.9×10⁻⁸ 과 동급).
`g=3` 에서 ‖r₁−r₀‖ = **7.57×10⁻²**(보정 활성). → **닫힘 무결 확인**.

### 3.3 논문 통찰 확증 — "비는 함수보다 둔감" (차수 이득)

Part D2(기하 고정 `L_0/w=0.6`, 상태의존 `g` 만 축소):

```
err0(동결) ~ g^1.32   (기준의 오차는 상태의존에 ~선형)
err1(ratio) ~ g^2.59   (ratio 스텝이 선형 오차를 제거 → ~2차)
```

곧 ratio 치환은 **선행 차수 오차를 실제로 제거**한다 — 논문 "비가 함수보다 근사에 둔감"의 수치 확증. 동시성 축소
(Part D, `g·L_0` 동시)에서도 이득 `err1/err0 ~ s^{2.2}`.

## 4. 타당성 부등식 도출 + 논문 (i)(ii)(iii) 1:1 매핑 (재유도 항목 4) [게이트 b PASS]

### 4.1 부등식 도출

1차 치환의 오차원 = 보정비 지수 `∫_u^V δκ(ξ_0) dV'` 가 핵 지지폭(∼`L_V`) 위에서 작아야 함. `δκ·L_V ≈ δln L_V`
이고, 물리(§1.2)에서 `∂ln L_V/∂ξ = −2χ_d·(Ω/RT)`(상호작용 몫이 `−χ_d𝒜/RT` 지수를 채움) 이므로 핵 지지폭 위
`ξ` 변동 `Δξ_supp` 에 대해

```
┌─────────────────────────────────────────────────────────────────────────┐
│  ε ≡ |∂ln L_V/∂ξ| · Δξ_supp = 2 χ_d (Ω/RT) Δξ_supp ≪ 1 ,   Δξ_supp ≈ L_V/(4w)  │   (타당성)
│  ⟺  χ_d · [Ω(1−2ξ) 의 꼬리 위 변동] ≪ RT   (계획서 §3 제안형과 동일)              │
└─────────────────────────────────────────────────────────────────────────┘
```

**수치 확증(Part F — Picard 수축률)**: 1차 = 첫 Picard 스텝. 스텝당 오차감소율 = `e_{n+1}/e_n` 이 **ε 과 일치**:

| `L_0/w` · `g` | ε(예측) | `e1/e0`(실측 수축률) | 판정 |
|---|---|---|---|
| 0.3 · 1.0 | 0.073 | **0.101** | ε≪1 → 스텝1이 큰 이득(≈10×), 이후 기하수축(≈10×/스텝) |
| 0.6 · 2.0 | 0.276 | **0.432** | 중간 → 스텝1 ≈2.3× |
| 1.0 · 4.0 | 0.840 | **0.883** | ε∼1 → 수축 정지 근방(스텝1 ≈1.1×, 사실상 열화) |

수축률 ≈ ε(계수 ∼1.5 내). 곧 **타당성 부등식 = ratio/Picard 수축률 조건**이라는 정량 확증. `ε≳0.5` 에서 1차·반복
모두 무력 → 논문 "**반응대 넓어지면 근사 악화**"(line 380–388)의 문건판.

### 4.2 논문 Sec.III (i)(ii)(iii) → 문건 1:1 매핑표

| 논문 Sec.III 타당성 | 물리 의미 | **문건 대응(lag)** | 정량 |
|---|---|---|---|
| **(i)** 비등방 섭동 `K` 과대 아님 | 기준에서의 이탈이 소섭동 | 상태의존 핵률 `δκ`(=`Ω(1−2ξ)` 변조)가 동결률 `κ_0` 의 소섭동 | `2χ_d(Ω/RT)Δξ_supp ≪ 1` |
| **(ii)** 기준척도 `r_c`(Onsager) 지배 | 가해기준이 지배 물리 포착 | 동결 `L_0`(=이상 몫 컷 affinity `𝒜_cut` 설정)이 지배, `Ω` 변조는 하위 | `L_0` = 지배, `δκ` = 보정 |

## 5. 전달함수 기준해 + 1차의 주파수 해석 (재유도 항목 5)

### 5.1 동결 0차 = 1차 저역통과 (SURV-1 재현)

동결 `L_0` 는 LTI 라 0차 = 병진불변 합성곱 → 주파수영역 대각화:

```
H(ω) = 1/(1 + iω L_V) ,     dQ/dV_app(ω) = H(ω) · dQ/dV_eq(ω) .          (전달함수 기준해)
```

**수치 확증(Part A)**: FFT 대조 `‖A_app − H·A_eq‖` 상대오차 **5.47×10⁻⁹**(기계정밀), 차단 `ω_c=1/L_V=0.667`,
`|H(ω_c)|=0.7071`(반출력점). SURV-1 (A급) 재현. 크기 `|H|`=대칭 저역통과, 위상 `arg H=−arctan(ωL_V)`=인과 skew.

### 5.2 1차 보정의 주파수 해석 (정직)

상태의존 `κ(ξ_0(V))` 는 병진불변 파괴 → **단일 `H` 없음**. 1차는 극점 `1/L_V` 가 **V-따라 완만히 이동하는
공간변조(space-variant) 저역통과**(local `L_V(ξ_0(V))`)다. `ε≪1` 에서 극점 변조가 소섭동이라 "약하게 처프된
1차 필터"로 읽힌다 — 이것이 전달함수 관점의 타당성. `ε∼1` 에서 필터 개념 자체가 붕괴(강 비선형).

## 7. 유용성 정직 판정 (재유도 항목 6) — 강등 권고

### 7.1 실이득 regime 정량

1차 ratio 보정이 **실이득**을 주는 조건 = 아래 3중 교집합:

1. **가시성** `L_V/w ≳ 0.1` — 그래야 꼬리가 평형종에서 갈라짐(Part E: `L_V/w=1e-8` 기본상태는 **미해상 가드로
   평형종 직행 → lag·보정 전부 무의미**; `1e-2` 까지 이탈 <0.5%).
2. **상호작용 유효** `g=Ω/RT` 가 어느 정도 큼 — 아니면 동결 0차가 이미 우수(`g→0` 서 정확).
3. **타당성** `ε = 2χ_d g Δξ_supp ≲ 0.3` — 아니면 보정이 marginal(Part C: `ε≳0.5` 서 `err1/err0≳0.8`).

기본 흑연 `Ω>2RT`(`g≳2`, `χ_d=0.5`)에서 `ε ≈ L_V/(2w)`. 곧 실이득창 ≈ **`0.1 ≲ L_V/w ≲ 0.6`** (중간전류 대역):
여기서 1차가 오차를 **3–10× 감축**. 그 위(`L_V/w≳1`, 고전류)는 marginal, 그 아래(휴면)는 무의미.

### 7.2 강등 권고 (부분 — 셀링포인트 정정)

**철회 대상 = 계획서 §4-1 "가치 = 분석적 닫힘 + 계산 절감(중간 regime)" 중 "계산 절감/수치 인과기억 루프 대체".**
근거(정직): 문건 lag 자기일관은 **Markov(지수핵)라 참 문제 자체가 O(N) 전진 ODE**(§1.4). 코드 3경로 —
(a)현행 동결 0차, (b)1차 ratio(+1 O(N)패스), (c)참 비선형 Picard 수렴(O(N)/스텝×소수스텝) — **전부 O(N)**.
곧 ratio 는 정확해 (c) 대비 **계산을 아끼지 않는다**(논문 Fredholm-2 의 전역 역산 회피와 달리, 여기선 아낄 비용이 없음).

**유지 대상(실가치)**:
- **분석적 닫힌형** — 0차 합성곱 + 1차 보정의 해석적 표현(교육·통찰).
- **타당성 부등식(§4)** — 코드가 **이미 쓰는 동결 근사의 안전 증명서**. "언제 동결이 옳은가"를 `ε≪1` 로 정량 →
  실질 기여(P3-5 이행). 이것이 본 접목의 **진짜 상품**.
- **사용자 논문 자기 인용** — 교과서 정합·5-ter 부록 E 정체성.

**권고 프레이밍(부록 E)**: "**자기일관 lag 의 1차 분석적 닫힘 + 타당성 증명서 + 사용자 방법의 문건 내 시연**".
"수치 루프를 계산 절감으로 승격/대체"로 **팔지 말 것**. 이는 강등(rejection 아님) — 재유도는 건재하며 부록 E 저작
가치 있음. **코드(P3)**: ratio 옵션은 *선택적 고정밀 분석 경로*로 두되, "정확 경로"는 (c)참 비선형 전진해가 더
자연스럽다는 점을 코드 주석·부록 E.4 에 정직 명기(ratio 는 분석적 중간항).

## 8. 적용 불가 재확인 — 부록 E 서두 명시 문안 (재유도 항목 7) [게이트 d]

문건 자기일관 3종 중 이 기법 대상은 **II(동역학 lag)뿐**이다. 나머지는 부록 E 서두에 **"이 기법 대상 아님"** 못박음:

> **[부록 E 서두 권고 문안]** 본 부록의 ratio 닫힘·전달함수는 **동역학 지연(§8–9, Volterra 자기일관)** 에만
> 적용된다. 전하보존 `U_oc` 반전(eq:sm-mc-balance·eq:blend-balance, 초월 **대수** 근)과 배경 자기일관
> `Q_cell q=Q_bg(V_n)+ΣQ_jξ_j`(대수 순환)은 **적분핵이 없는 대수 근찾기**이며, Newton·이분으로 **이미 저렴·정확**하게
> 유일근이 구해진다(요동 양성 `∂⟨N⟩/∂μ>0` 이 유일성 보증, §6). 곧 ratio 기법의 대상이 아니다 — 이를 명기해
> 과장(대수 근에까지 적분방정식 기계를 얹는 오도)을 차단한다.

(검수 7항목 ②(전하보존 중심식 불가침)·④(순환의존 4분류) 정합: lag = "정의상 implicit → Volterra-2"; `U_oc`·배경 =
"수치해법 필요한 대수 근"으로 이미 분류됨.)
