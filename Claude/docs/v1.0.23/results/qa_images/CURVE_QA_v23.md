# 곡선 QA — 양 버전 물리 타당성 + 연속·매끄러움·미분가능성

> 사용자 요청: "예전 어떤 버전에서 연속·매끄러운·미분가능 곡선이 안 나왔다" → 양 버전 dQ/dV·DVA 샘플 렌더 + 정량 매끄러움 검증.
> 스크립트 `curve_qa.py`(재현). 그림 `qa_v1022.png`·`qa_v1023.png`.

## 판정: PASS — 전 곡선 연속·매끄러움(C2)·미분가능. 물리 값 정상. 양 버전 공유경로 bit-exact.

## 1. 매끄러움 정량 (격자세분 C2 테스트)
C2 매끄러움 ⟹ 이산 2차도함수가 유한 해석값으로 **수렴**(세분비 N→2N ≈ **1.0**). kink(1차도함수 점프) ⟹ 2차차분 **발산**(세분비 ≈ **2.0**).

**규약 캘리브레이션**: `x²`(smooth) 세분비 **1.000** · `|x|`(kink) 세분비 **2.001** — 지표 방향 검증됨.

| 패널 | 세분비 | 판정 |
|---|---|---|
| 흑연 dQ/dV 평형/동역학 dis·chg (v22·v23) | 1.000 | SMOOTH C2 ✓ |
| LCO dQ/dV (v22·v23) | 1.000 | SMOOTH C2 ✓ |
| 흑연 ratio-OFF(동결, L_V/w=0.3) | 1.023 | SMOOTH C2 ✓ |
| 흑연 ratio-ON(1차 보정) | 1.001 | SMOOTH C2 ✓ |

→ **전 패널 세분비 ≈1.0 = 발산(kink) 0**. 유한값(연속) 전건. v1.0.22 그림 6번 패널 `d²(dQ/dV)/dV²` 직접 플롯도 **스파이크 없는 매끈한 파형**(육안 확증).

## 2. 물리 값 범위
- 흑연 dQ/dV ∈ [+0.73, +7.30] — 전 구간 양수(peak>0), 정점 ~0.10 V. 정상.
- LCO dQ/dV ∈ [+0.67, +6.87] — 양수, 정점 ~3.92 V + staging shoulder ~4.0–4.05 V. 정상.
- DVA(dV/dQ) — 깊은 꼬리(Q→상한)서 상승(dQ/dV→0 의 역수), 물리적.
- 블렌드 스윕 0/10/20/30 wt% — Si 증가 시 고전위부 상승, 인접 wt% 간 **점프 없음**(R6-G2 Lipschitz 연속 육안 확인).
- ratio-ON ∈ [0, +9.9] — 1차 보정이 peak 를 고전위로 이동·sharpen(상태의존 L_V(ξ): 저-ξ 리딩에서 lag↑→지연, ξ→1서 동결 회수). 매끄러움 유지.

## 3. 양 버전 공유경로 동일성
```
흑연 discharge c=0.0: max|v22-v23| = 0.00e+00
흑연 discharge c=1.0: max|v22-v23| = 0.00e+00
흑연 charge   c=1.0: max|v22-v23| = 0.00e+00
```
→ 공유 dQ/dV 경로 **bit-exact**(v1.0.23 ratio 기본 off). 그림도 육안 동일.

## 4. 물리 관찰 (결함 아님)
- 기본 흑연 파라미터에서 c=1 동역학 곡선이 평형(c=0)과 **겹침** — L_V~7.5e-7 V 로 미해상 가드→평형종(부록 E.4 warnbox "기본 상태 휴면"·COND_AUDIT §7 일치). 동역학 꼬리(비대칭)는 해상 regime(L_V/w≳0.1, 그림 6번 ratio-OFF/ON 패널)에서 발현. 피팅 파라미터(낮은 ΔH_a·높은 dVdq_qa)면 가시창 진입.
