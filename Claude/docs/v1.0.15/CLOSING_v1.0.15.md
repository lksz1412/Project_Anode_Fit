## Part 1. ★★★ 최상위 헌법 3종 — 이 문건 작업의 정체성 (절대 우선)

이 교재(graphite dQ/dV·발열)는 **세 가지를 동시에** 만족해야 한다. 물리·화학 논리 무결이 이 셋의 전제이고, 우선순위는 **①교과서 register ②논문의 깊이 ③수식-주도**다. (사용자 verbatim: "이 문건의 제일 중요한 두가지는 교과서 수준의 문건, 논문의 깊이의 전문성이고, 세번째가 수식 주도의 문건이다. 수식만 쭉 따라가도 제대로 된 물리, 화학적 논리를 대부분 이해할 수 있는 수준의 문건." + "1.0.14 버전의 어투, 어조, 구성을 보면 교과서 수준·논문 깊이 문건임을 확인할 수 있다.")

### 헌법 ① — 교과서 수준 register (최우선)
- **어투**: 안정·객관·정중. 완성된 물리를 그냥 제시한다. 교재는 **자기 자신을 narrate 하지 않는다.**
- **[D1] 자기 diff·버전-이력 서술 금지**: "구판은/이전에는/폐지했다/이산화했었다/병리였다" 류 자기 개정 이력을 본문·각주·캡션·주석 어디에도 쓰지 않는다. 교정판 교과서는 diff 를 말하지 않고 결과만 제시한다.
- **[D2] 방어 어투 금지**: "이는 …일 뿐 …가 아니다" 류 "X가 아니다" 방어 서술 X. 무엇인지 **긍정 서술**로. ("가정이 아니다(전제)" 같은 자기모순도 금지.)
- **내부 라벨·고백조 누출 금지**: 작업단계 라벨(P5/P6/"최종 게이트"/"구현 대응"), 코드 함수명, "정직하게 적어두면" 류 고백조를 **렌더링 텍스트에** 넣지 않는다(그런 정보는 결과 문건·주석에).
- **연속성**: v1.0.14 본문의 목소리·어조·구성을 계승한다. 새 절이 나머지와 이질적으로 튀면 안 된다. **이전 문건의 문제점·개선점을 이어받아 연속성을 가진 채 문제만 고친다** — 백지 재작성 지시가 아니면 뼈대·목소리를 끊지 않는다.

### 헌법 ② — 논문의 깊이 전문성
- 간결보다 **면밀·논리적 납득** 우선. 유도는 각주 압축이 아니라 **완결 유도**로 (사용자: "비약 없는 정당한 수식전개를 전부 진행한 뒤 확정된 수식을 먼저 구해놓고").
- 코너·극한·전제·tier(신뢰 등급 A/B/C)를 정직 분리·명시. **실측 검증 vs 자기일관성 검증**을 구분(예: 완전식 수치 일치는 자기일관성 검증이지 두-상 폭 실측 검증이 아님).
- 서지(DOI) 병기, 1차 문헌 공백(Gn) 정직 노출. 허위 정밀 금지.

### 헌법 ③ — 수식-주도 문건
- **수식만 쭉 따라가도 물리·화학 논리를 대부분 이해**할 수 있어야 한다.
- 각 결과 박스 = **(a)출발식/전제 → (b)적용 연산 → (c)중간식 ≥1 → (d)박스**. **[D3] "대입하면 [박스]" 점프 0.** (부분적분·치환·지배수렴 등 중간식을 반드시 노출.)
- 보편식 먼저, 극한은 코너 케이스로. 부호·차원·극한 검산 병기.

### 헌법 3종 공통 하위 규율
- **완결 문장**: 전보체 괄호(주술 없는 보충 명제) 금지 — 주술 갖춘 완결 명제로. **절 도입/마무리 다리** 의무(추가는 앞에서 도입·뒤에서 사용, orphan 0).
- **한글 prose + 영어 학술 원어**(한글 번역 강요 X, 두문자어 첫 출현 병기), 특정 국적 표현 회피.
- **[D4~D6]** (이번 버전 특화) D4 격자 잔재(작업격자·Δ_grid·역보간·저역통과 등) 본문 잔존 0 / D5 대조 수식어(없어진 것과 대조하는 "연속" 등) 불요 / D6 스위치 잔재(수치 0-나눗셈 가드는 코드 소관·본문 밖).
- **분량·그림 = 콘텐츠의 자연 결과.** 억지 조사-급 보완 라운드 금지.

## Part 4. ★다음 버전 확정 방향 — w/T 피팅 (이번 세션 사용자 결정)

문건·코드 업데이트로 반영할 확정 방향(사용자 결정):

1. **폭은 w 맨값이 아니라 n으로 fit**(w=n·RT/F, RT/F 물리 앵커 유지). n = 무차원 비이상/분산 인자. 현재 n=1 고정 → **fit 대상 개방**. `'w'` 폴백값은 초기 추정으로 존치. (자기정합: 같은 n이 폭과 가역열 config 항을 동시에 정함 = 완전식.)
2. **T는 실측 환경온도 투입**(평가 데이터의 온도 정보 사용). 코드 `dqdv(V_app, T, …)` 는 스칼라(등온)/배열 T(V)(비등온) 모두 지원 — 각 점 폭이 그 점 실측 T를 반영.
3. **폭 T-의존은 가정이 아니라 데이터로 확정** — 4단 사다리:
   (i) 기본 **상수 n**(열적 w∝T) → (ii) 다온도 **per-T n 상수성 진단** → (iii) n이 T로 드리프트하면 그 전이를 **상수 w(T-동결)** 로 전환(대개 두-상 broadening) → (iv) 그래도 안 맞으면 **최소 n(T)**(선형 n₀+n₁(T−T_ref)) 1계수만.
4. **★n(T)로 가면 가역열 config 항 동반 변경**: ∂w/∂T = (R/F)·d(nT)/dT = (R/F)(n+T·n′). 현행 코드 config 항은 n 상수 전제이므로, n(T) 채택 시 폭·발열을 한 n(T)로 묶어 코드 반영 필요.
5. **참고 이력**: w 이중지위(단상 Ω≤2RT = 평형 예측 폭 / 두-상 Ω>2RT[LiC₁₂·LiC₆] = 현상학적 자유 피팅 폭, Ω가 가름). 두-상에 열적 서식 config 적용 = "물리 유도 아닌 모델 선택"(Ch2 srcbox 헤지). `use_w_eff`(Ω→폭)는 narrowing 버그로 제거됨 → Ω는 히스테리시스에만, n과 축퇴 없음.
6. **FITTING_GUIDE 명문화 대상**: 위 (1)~(3) + Part 1 헌법 3종을 다음 버전 계획서 상위에 배치.
